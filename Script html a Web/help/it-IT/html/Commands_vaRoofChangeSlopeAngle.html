Tetto: Cambia angolo di inclinazione

Tastiera:
vaTettoCambiaAngoloInclinazione
Icona:
clic sinistro
Barra degli strumenti Tetti:
Menu:
VisualARQ > Tetto > Cambia angolo di inclinazione

Opzione per cambiare l'angolo di inclinazione di un tetto.

Eseguire il comando vaTettoCambiaAngoloInclinazione e seguire i passi dalla linea di comando. Premere il tasto destro del mouse oppure INVIO per accettare e passare al passo successivo.

Passi:

  1. Selezionare il tetto per cui si desidera cambiare l'angolo di inclinazione.
  2. Introdurre un nuovo valore per l'angolo nella linea di comando.

L'angolo di inclinazione di un tetto si può editare anche nella sezione Proprietà di VisualARQ (nel pannello delle proprietà di Rhino ).
