Menu a discesa

La maggior parte dei comandi dell'applicazione si trova all'interno del menu a discesa di VisualARQ. Nel menu, vengono elencati tutti gli oggetti e strumenti di VisualARQ che si trovano nelle barre degli strumenti, con le loro opzioni corrispondenti.


Il menu a discesa di VisualARQ.

NOTA: Il menu a discesa dovrebbe apparire quando VisualARQ viene caricato nella sessione corrente di Rhino dopo aver installato VisualARQ. Se ciò non succede, è possibile che VisualARQ stia usando una versione precedente del file "visualarq.tb" (in Rhino 4.0) o del file "visualarq.rui" (in Rhino 5). In tal caso, occorre eliminare manualmente questo file, disinstallare VisualARQ ed installarlo di nuovo. I file "visualarq.tb" / "visualarq.rui" si trovano nelle seguenti directory:

  • "visualarq.tb": (Rhino 4.0): (C:\ProgramData\McNeel\Rhinoceros\4.0)
  • "visualarq.rui" (Rhino 5.0, 32 e 64 bit): (C:\Users\"Nome utente"\AppData\Roaming\McNeel\Rhinoceros\5.0\UI )
