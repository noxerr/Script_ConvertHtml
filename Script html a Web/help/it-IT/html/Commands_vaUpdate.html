Aggiorna oggetti selezionati

Tastiera:
vaAggiorna
Icona:
clic sinistro
Barra degli strumenti Strumenti:
Menu:
VisualARQ > Aggiorna

Quando si effettuano delle modifiche che non vengono rispecchiate automaticamente nel modello, basta selezionare l'oggetto o gli oggetti modificati e fare clic sull'icona Aggiorna.

NOTA: L'aggiornamento automatico tende a rallentare le prestazioni del programma. Per questo motivo, questa opzione è disponibile solo per determinati comandi.
