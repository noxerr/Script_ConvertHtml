Esporta tabella

Tastiera:
vaTabellaEsporta
Menu:
VisualARQ > Tabella di computo > Esporta
Menu di scelta rapida:
Tabella di computo > Esporta tabella

Una volta inserite nel modello, le tabelle si possono esportare in formato Excel (.xls) e .csv.

Passi:

  1. Selezionare la tabella di computo da esportare.
  2. Selezionare la cartella ed il formato file in cui si desidera salvare la tabella: .xls oppure .csv.

Nella cartella di destinazione selezionata, viene creato un nuovo file. Nel caso in cui vengano applicate delle modifiche alla tabella di Rhino-VisualARQ, questo file non si aggiorna automaticamente. Per avere l'ultima versione aggiornata, occorrerà ripetere i passi spiegati qui sopra e sovrascrivere il file precedente.
