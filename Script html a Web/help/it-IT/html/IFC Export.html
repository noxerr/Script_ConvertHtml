VisualARQ include un plug-in di esportazione IFC integrato che rende possibile l'esportazione dei file di VisualARQ e Rhino verso altri programmi AEC che fanno uso dei file IFC 2x3.

Cos'è l'IFC?

Il formato file "Industry Foundation Classes" (IFC) è il formato standard usato per lo scambio di dati nell'industria delle costruzioni, il quale consente di ridurre la perdita di informazioni nel passaggio dei file da un'applicazione ad un'altra.

Come esportare i documenti di Rhino-VisualARQ in formato IFC?

  • Andare su File > Salva con nome e salvare il modello nel formato file IFC.

    • Quali tipi di geometrie e dati vengono esportati da VisualARQ in formato IFC?

    Quando un documento viene salvato come file IFC, tutti gli oggetti architettonici di VisualARQ mantengono le loro informazioni e possono essere ulteriormente modificati nelle piattaforme software AEC in output. I solidi, le superfici e le mesh di Rhino vengono aperti come oggetti mesh indipendenti. Gli oggetti di annotazione (quali i testi, i riempimenti o le quote) non vengono esportati in formato IFC.
