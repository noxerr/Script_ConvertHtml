VisualARQ > Tabella > Tabella

L'oggetto tabella mostra le informazioni relative agli oggetti dello stesso tipo.

Quando si esegue il comando vaTabella, appare la finestra di dialogo di inserimento delle tabelle. La finestra mostra gli stili di tabella disponibili. Anche se nell'elenco appaiono gli stili predefiniti di tabelle per le aperture, i vani ed i muri, si possono creare nuovi stili di tabelle per il computo di qualsiasi altro oggetto di VisualARQ.

Passi:

  1. Selezionare lo stile di tabella che si desidera usare.
  2. Selezionare tutti gli oggetti che si desidera quantificare nella tabella. Il comando filtra la selezione automaticamente, rilevando solo il tipo di oggetto corrispondente allo stile di tabella selezionato.
  3. Specificare un punto di inserimento cliccando su un punto qualsiasi dello schermo per inserire la tabella.

Il comando termina con l'inserimento della tabella di computo degli oggetti selezionati.


Finestra di dialogo di inserimento dell'oggetto Tabella

Punti di controllo

Le tabelle hanno un punto di controllo sul vertice superiore sinistro.

Opzioni e parametri di inserimento

La finestra di dialogo di inserimento delle tabelle NON presenta nessun parametro di inserimento. L'unica opzione che si può scegliere in questa finestra è lo Stile della tabella da inserire.

I parametri che definiscono le caratteristiche di una tabella corrispondono allo stile di una tabella.

Esportazione delle tabelle in Excel

Una volta inserite nel modello, le tabelle si possono esportare in formato Excel e .csv con il comando vaTabellaEsporta.
