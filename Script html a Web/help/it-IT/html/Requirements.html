Requisiti del sistema

Software

Per poter essere eseguito, VisualARQ richiede Rhinoceros 4.0 SR7 o Rhinoceros 5 SR6 (32 o 64 bit).

Hardware

I requisiti hardware sono gli stessi di quelli richiesti per Rhinoceros 4.0 e Rhino 5.

Installazione

VisualARQ è dotato di una facile procedura di installazione guidata che consente di automatizzare il processo.

Per l'installazione e l'uso, si richiede il codice di licenza del prodotto. In rete, la licenza può essere usata da un utente per volta.

L'installazione avviene in modo predefinito in: C:\Programmi\AsuniCAD\VisualARQ "1.X".

Dopo aver installato il programma, cliccando sull'icona di collegamento di Rhino, appare una finestra di dialogo in cui si visualizzano i modelli template che l'utente può usare. Per lavorare con VisualARQ, è consigliabile scegliere uno dei quattro modelli che esso offre:

  • VisualARQ - Metri.3dm
  • VisualARQ - Millimetri.3dm
  • VisualARQ - Piedi.3dm
  • VisualARQ - Pollici.3dm


Elenco dei modelli di Rhino e VisualARQ.

In Rhino 5 (32 e 64 bit), i modelli template di VisualARQ si trovano nella seguente directory: C:\Utenti\"Nome utente"\AppData\Roaming\McNeel\Rhinoceros\5.0\Localization\en-US\Template Files. (Questa è la cartella dei modelli template in inglese. I modelli template nelle altre lingue sono contenuti nelle corrispondenti cartelle localizzate: "it-IT" anziché "en-US" per l'italiano).
