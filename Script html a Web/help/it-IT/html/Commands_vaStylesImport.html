Importa stili

Tastiera:
vaStiliImporta
Menu:
VisualARQ > Stili > Importa

Il comando vaStiliImporta consente l'importazione degli stili di VisualARQ da qualsiasi documento, in modo che siano disponibili all'interno della libreria degli stili del file .3dm in uso.
I file importati hanno estensione .val (libreria di VisualARQ) e, prima di essere importati, vanno esportati dal file originale.

Quando si esegue il comando vaStiliImporta, appare la finestra di dialogo di importazione degli stili.

Passi:

  1. Aprire il file .val che contiene gli stili di oggetto da importare nel file .3dm in uso. Se il percorso del file non viene visualizzato nel menu a discesa, premere il pulsante Sfoglia per cercare il file nel sistema. Premere il pulsante Avanti > per proseguire.
  2. Dall'elenco, selezionare gli stili di oggetto da importare, quindi premere il pulsante Importa per completare il processo.
