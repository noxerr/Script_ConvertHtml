Comandi da tastiera

Oltre che tramite le barre degli strumenti o i menu (a discesa o di scelta rapida), i comandi di VisualARQ sono eseguibili digitandone il nome dalla linea di comando.

I comandi di VisualARQ iniziano con il prefisso va seguito dal nome del comando. Basta digitare il prefisso (va) e, sulla linea di comando, si visualizzerà l'elenco di tutti i comandi di VisualARQ.

Per eseguire un comando dalla linea di comando, basta digitare il nome del comando. Grazie alla funzione di autocompletamento, man mano che si digita, si mostrano tutte le opzioni di comando che iniziano con le lettere appena scritte.
Per finire, fare clic per selezionare il comando desiderato dall'elenco dei comandi.

Così come per i comandi di Rhino, la linea di comando mostra anche i prompt e le opzioni dei comandi di VisualARQ tra parentesi:

  • I prompt sono messaggi che guidano l'utente all'interno di un comando, indicando che realizzi una particolare azione o che inserisca determinate informazioni.
  • Le opzioni di comando cambiano a seconda delle funzioni di un comando. Esse offrono varie alternative di uso dei comandi.

Per attivare un'opzione, cliccare su di essa oppure digitarne il nome o la lettera di collegamento (quella sottolineata) nella linea di comando.

Per completare un comando, premere INVIO.
