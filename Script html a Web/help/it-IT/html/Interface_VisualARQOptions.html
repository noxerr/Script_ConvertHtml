Opzioni di VisualARQ (nella finestra di dialogo "Opzioni di Rhino")

Per aprire la finestra di dialogo Opzioni di Rhino, andare al menu Strumenti e selezionare Opzioni....
Apparirà la finestra di dialogo Opzioni di Rhino, la quale sulla sinistra contiene una struttura ad albero con tutte le impostazioni disponibili per le Proprietà del documento e le Opzioni di Rhino.

Ognuna di queste due sezioni contiene una sotto-sezione con varie impostazioni disponibili per VisualARQ:

  • Opzioni di Rhino > Proprietà del documento > VisualARQ > Livelli


    Impostazioni per i livelli (layers), nelle opzioni di VisualARQ

  • Opzioni di Rhino > VisualARQ > Generali (Lingua, Messaggi e Visualizzazione), Icona scala ed Aggiornamenti


    Icona scala, nelle opzioni di VisualARQ
