Finestre di dialogo di inserimento

Le finestre di dialogo di inserimento mostrano le opzioni degli oggetti che si stanno inserendo. Esse sono visibili solo mentre si sta eseguendo un comando.

I pannelli delle opzioni si possono espandere (visualizzare) o comprimere (nascondere) cliccando sulle frecce in alto a destra.


Finestra di dialogo di inserimento dell'oggetto Muro

Le finestre di dialogo di inserimento si dividono in tre o più sezioni:

  • La parte superiore mostra l'elenco degli stili di oggetto disponibili.
  • La parte (o le parti) centrale mostra le principali opzioni di inserimento dell'oggetto.
  • La parte inferiore mostra un'anteprima (in vista piana e prospettica) dell'oggetto da inserire. Quando questa sezione è nascosta, l'anteprima non viene generata, per cui il programma non dovrà calcolarla.

NOTA: Gli oggetti Solaio e Tetto non usano le finestre di dialogo di inserimento.
