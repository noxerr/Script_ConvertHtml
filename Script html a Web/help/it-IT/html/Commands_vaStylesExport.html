Esporta stili

Tastiera:
vaStiliEsporta
Menu:
VisualARQ > Stili > Esporta

Il comando vaStiliEsporta consente di esportare gli stili di qualsiasi oggetto di VisualARQ, per importarli successivamente su altri file (.3dm).
L'estensione dei file esportati è .val (libreria di VisualARQ).

Quando si esegue il comando vaStiliEsporta, appare la finestra di dialogo di esportazione degli stili. Cliccare sul pulsante Avanti per andare al passo successivo.

Passi:

  1. Selezionare una delle due opzioni disponibili per l'esportazione del file:
    • File esistente: Opzione per salvare i nuovi stili creati su un file .val già esistente. Aprire il file .val esistente per aggiungervi gli stili di oggetto che si desiderano esportare. Se il percorso del file non viene visualizzato nel menu a discesa, premere il pulsante Sfoglia per cercare il file nel sistema. Il nuovo file sovrascriverà gli stili di oggetto aventi lo stesso nome ed aggiungerà i nuovi stili di oggetto all'elenco esistente.
    • Nuovo file: Opzione per creare un nuovo file .val con gli stili di oggetto che si desiderano esportare. Assegnare un nome al nuovo file .val e premere il pulsante Sfoglia per salvarlo nella cartella di output desiderata. Specificare le unità in cui si desidera salvare il file.
  2. Selezionare tutti gli stili di oggetto da esportare dall'elenco visualizzato e quindi premere il pulsante Esporta.
