Barre degli strumenti

Le barre degli strumenti di VisualARQ funzionano come quelle di Rhino:
  • Si possono trascinare e rilasciare su qualsiasi punto dello schermo oppure ancorare alla finestra principale.
  • Si possono personalizzare tramite l'aggiunta o l'eliminazione di pulsanti.
  • Alcuni pulsanti hanno un secondo comando, a cui si accede tramite clic destro del mouse, eseguendo un'opzione diversa relativa allo stesso comando.
  • Collocando il cursore sui pulsanti, vengono visualizzati i tooltip che indicano i nomi dei comandi che si possono eseguire con un clic destro o sinistro del mouse.
  • Alcuni pulsanti (indicati da un triangoletto sul vertice inferiore destro) sono vincolati ad altre barre degli strumenti con pulsanti a discesa.

Mantenere premuto il tasto sinistro del mouse sull'icona per visualizzare la barra degli strumenti vincolata ad un pulsante.

    • NOTA: Se le barre degli strumenti non vengono visualizzate automaticamente dopo aver installato VisualARQ, esse si possono caricare manualmente seguendo la stessa procedura di caricamento delle barre degli strumenti di Rhino: digitare BarraStrumenti nella linea di comando ed aprire il file VisualARQ.tb (in Rhino 4.0) oppure il file VisualARQ.rui (in Rhino 5), che si trovano in una delle seguenti directory, a seconda della versione di Rhino che si sta usando:

    • Rhino 4.0: (C:\ProgramData\McNeel\Rhinoceros\4.0)
    • Rhino 5.0 (32 e 64 bit): (C:\Users\"Nome utente"\AppData\Roaming\McNeel\Rhinoceros\5.0\UI)
