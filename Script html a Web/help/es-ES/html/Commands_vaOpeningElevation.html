VisualARQ > Alzado de carpintería > Alzado de carpintería

Este comando genera representaciones 2D de las elevaciones (o alzados) de las distintas carpinterías del modelo.

Pasos:

  1. Seleccione las puertas y ventanas de las que desea obtener un alzado. El comando filtra automáticamente la selección, para que sólo sean detectados los objetos de carpintería.
  2. Designe el inicio y el final de la inserción.

El comando termina al insertarse el dibujo 2D de cada una de las distintas carpinterías seleccionadas, con sus respectivas referencias. En la vista Planta se mostrarán estas mismas referencias, identificando así la ubicación de cada tipo de puerta y ventana dentro del modelo.

Alzados de carpintería acotados y con su referencia a cada carpintería del modelo

NOTA: Para todas las carpinterías (puertas o ventanas) que sean iguales en todas sus características – medidas, elevación y lado de apertura - se creará un único dibujo con una única referencia.

Los alzados de carpintería y sus referencias se crean por defecto en la capa Documentación.
Para ocultar las referencias, basta con desactivar la capa en la que han sido creadas.
Para eliminar las referencias, seleccione la carpintería que desee eliminar y entre en la sección Propiedades de VisualARQ (en el panel Propiedades de Rhino). Elimine el valor Texto de referencia en la sección Referencia.
