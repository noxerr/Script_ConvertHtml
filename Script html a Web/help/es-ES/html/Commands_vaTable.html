Tabla

Teclado:
vaTabla
Icono:
clic izquierdo [clic derecho]
Barra de herramientas Documentación:
Menú:
VisualARQ > Tabla > Tabla

El objeto tabla muestra información sobre objetos del mismo tipo.

Cuando ejecute el comando vaTabla, aparecerá el cuadro de diálogo de inserción de la tabla. Este cuadro muestra los estilos de tabla disponibles. Esta lista incluye dos estilos de tabla Huecos, Espacios y Muros predeterminados, pero se pueden crear nuevos estilos de tabla para cuantificar cualquier objeto de VisualARQ.

Pasos:

  1. Seleccione el estilo de tabla de que desee utilizar.
  2. Seleccione todos los objetos que desee cuantificar en la tabla. El comando filtra la selección automáticamente, detectando únicamente el tipo de objeto que corresponde, según el estilo de tabla elegido.
  3. Elija un punto de inserción haciendo clic sobre cualquier punto de la pantalla para insertar la tabla.

El comando finaliza cuando se inserta la tabla que cuantifica los objetos seleccionados.


Diálogo de inserción de tabla

Puntos de control

Las tablas tienen un punto de control en la esquina superior izquierda.

Opciones y parámetros de inserción

El cuadro de inserción de tablas NO tiene parámetros de inserción. La única opción elegible desde este cuadro, es el Estilo de tabla a insertar.

Los parámetros que definen las características de una tabla se corresponden con un estilo de tabla.

Exportar tabla a Excel

Una vez insertadas en el modelo, las tablas se pueden exportar al formato de Excel (.xls) y .csv con el comando vaTablaExportar.
