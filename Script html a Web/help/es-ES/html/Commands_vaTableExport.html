Exportar tabla

Teclado:
vaTablaExportar
Menú:
VisualARQ > Tabla de planificación > Exportar
Menú contextual:
Tabla de planificación > Exportar tabla

Una vez insertadas en el modelo, las tablas se pueden exportar al formato de Excel (.xls) y .csv con el comando vaTablaExportar.

Pasos:

  1. Seleccione la tabla de planificación que desee exportar.
  2. Seleccione la carpeta y el formato de archivo en el que desea guardar la tabla: .xls o .csv.

Se creará un nuevo archivo en la carpeta de destino seleccionada. Este archivo no se actualiza automáticamente cuando se aplican cambios a la tabla de Rhino-VisualARQ. Para tener la última versión actualizada, simplemente repita los pasos anteriores y sobrescriba el archivo antiguo.
