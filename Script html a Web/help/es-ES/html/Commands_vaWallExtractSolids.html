Muro: Extraer sólidos

Teclado:
vaMuroExtraerSólidos
Icono:
clic izquierdo
Barra de herramientas Muros:
Menú:
VisualARQ > Muro > Extraer sólidos

Opción para extraer uno o más sólidos previamente agregados a un muro.

Cuando ejecute el comando vaMuroExtraerSólidos, siga los pasos que se indican en la línea de comandos. Pulse el botón derecho del ratón o INTRO para aceptar y pasar al siguiente paso.

Pasos:

  1. Seleccione el muro desde el que desea extraer los objetos sólidos que añadidos previamente.
  2. Tras pulsar el botón derecho del ratón o INTRO para validar la selección del muro, los sólidos se separan del mismo, volviendo a quedar como elementos independientes.
