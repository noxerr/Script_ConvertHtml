Opciones de VisualARQ (dentro del cuadro de Opciones de Rhino)

Para abrir el cuadro de diálogo Opciones de Rhino, vaya al menú Herramientas y seleccione Opciones.
Aparecerá el cuadro de diálogo Opciones de Rhino que muestra en la sección de la izquierda un árbol con las configuraciones posibles para las Propiedades de documento y para las Opciones de Rhino.

En cada una de estas ramas, aparece un apartado con posibles configuraciones para VisualARQ:

  • Opciones de Rhino > Propiedades de documento > VisualARQ > Capas


    Configuración de capas, dentro de las Opciones de VisualARQ

  • Opciones de Rhino > VisualARQ > General (Idioma, Mensajes y Visualización), Icono de escala y Actualizaciones


    Icono de escala, dentro de las Opciones de VisualARQ
