Exportación de estilos

Teclado:
vaExportarEstilos
Menú:
VisualARQ > Estilos > Exportar

El comando vaExportarEstilos permite exportar estilos de cualquier objeto de VisualARQ, para importarlos posteriormente a otro/s fichero/s (.3dm).
Los archivos exportados tienen la extensión .val (librería de VisualARQ).

Cuando se ejecuta el comando vaExportarEstilos, aparece el cuadro de diálogo Exportar estilos de VisualARQ. Haga clic en el botón Siguiente para ir al paso siguiente.

Pasos:

  1. Active una de las dos opciones para exportar el archivo a:
    • Archivo existente: Opción para guardar los nuevos estilos creado en un archivo .val existente. Abra el archivo .val existente para agregar los estilos de objeto que desee exportar. Si la ruta del fichero no se encuentra dentro del menú desplegable, pulse el botón Examinar buscarlo dentro de su equipo. El nuevo archivo sobrescribirá los estilos de objeto que tienen el mismo nombre y agregará los nuevos estilos de objeto a la lista existente.
    • Nuevo archivo: Opción para crear un nuevo archivo .val con los estilos de objeto que desee exportar. Asigne un nombre al nuevo archivo .val y pulse el botón Examinar para guardarlo en la carpeta de salida deseada. Seleccione las unidades en las que desea guardar el archivo.
  2. Seleccione todos los estilos de objeto que desee exportar de la lista que aparezca y pulse finalmente el botón Exportar para finalizar.
