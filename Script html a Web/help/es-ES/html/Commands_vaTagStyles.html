Barra de herramientas Documentación:

VisualARQ > Etiqueta > Estilos de etiqueta
Acceso directo:
Seleccione un objeto de etiqueta > Alt, Mayús o Ctrl + doble clic

El estilo de etiqueta define las características de las etiquetas.


Diálogo de estilos de etiqueta

Edición general de estilos
Puede utilizar el comando vaEstilosDeEtiqueta o, alternativamente,
  • Utilice el comando vaEstiloPropiedades, seleccione una etiqueta y pulse Intro, o bien
  • Haga clic con el botón derecho en , seleccione una etiqueta y pulse Intro, o bien
  • Seleccione una etiqueta, abra el menú contextual pulsando el botón derecho del ratón durante un momento y seleccione Etiqueta > Propiedades de estilo, o bien
  • Pulse Alt, Mayús o Ctrl y haga doble clic en un objeto de etiqueta.
Los cambios aplicados a un estilo se aplican a todos los elementos con ese estilo.
Edición específica por objeto
  • Utilice el comando vaPropiedades, seleccione una etiqueta y pulse Intro, o bien
  • Haga clic en , seleccione una etiqueta y pulse Intro, o bien
  • Seleccione una etiqueta, abra el menú contextual pulsando el botón derecho del ratón durante un momento y seleccione Etiqueta > Propiedades, o bien,
  • Haga doble clic en un objeto de etiqueta.
Los cambios se aplican solo en el estilo del elemento seleccionado.

Nuevo estilo

Para crear un nuev estilo de etiqueta, pulse el botón Nuevo estilo. Para duplicar un estilo de etiqueta existente, selecciónelo en la lista del panel derecho y pulse el botón Duplicar.

Atributos

Los atributos definen las características de cada estilo de etiqueta para las vistas y la impresión. Los atributos válidos son:

  • Capa, Visibilidad, Estilo de texto, Estilo de cota, Color, Tipo de línea , Ancho de impresión de línea y Color de impresión de línea.
