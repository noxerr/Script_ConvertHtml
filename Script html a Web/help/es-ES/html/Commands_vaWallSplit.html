Muro: Partir

Teclado:
vaMuroPartir
Icono:
clic izquierdo
Barra de herramientas Muros:
Menú:
VisualARQ > Muro > Partir

Opción para dividir verticalmente un muro en dos muros diferentes en un punto específico.

Cuando ejecute el comando vaMuroPartir, siga los pasos que se indican en la línea de comandos. Pulse el botón derecho del ratón o INTRO para aceptar y pasar al siguiente paso.

Pasos:

  1. Seleccione el muro que desee partir.
  2. Elija el punto donde desea partir el muro. El punto siempre se sitúa en la línea base del muro.

El muro original ahora se divide en dos muros diferentes, que pueden editarse y modificarse por separado.
