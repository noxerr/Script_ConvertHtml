Cuadros de inserción

Los cuadros de inserción muestran las opciones de los objetos que se están insertando. Solo aparecen cuando se está ejecutando un comando.

Los paneles de opciones se pueden expandir (mostrar) o contraer (ocultar) haciendo clic en las flechitas de las esquinas.


Cuadro de inserción del objeto Muro.

Los cuadros de inserción se dividen en tres o más secciones:

  • La parte superior muestra una lista con los estilos de objeto disponibles.
  • La/s parte/s del medio muestra/n las opciones básicas de inserción del objeto.
  • La parte inferior muestra una previsualización -en planta y perspectiva- del objeto a insertar. Cuando esta sección está replegada no se genera la previsualización, por lo que el programa no debe calcularla.

NOTA: Los objetos Losa y Cubierta no utilizan cuadro de inserción.
