VisualARQ tiene un plug-in de exportación IFC integrado en el software, que permite exportar archivos de VisualARQ y Rhino a otros programas AEC utilizando archivos IFC 2x3.

¿Qué es el IFC?

El formato de archivo Industry Foundation Classes (IFC) es un modo estándar intercambio de objetos en la industria de la construcción para evitar la pérdida de información al transferir archivos entre diferentes aplicaciones.

¿Cómo se exportan los documentos de Rhino-VisualARQ a IFC?

  • Vaya a Archivo > Guardar como y guarde el modelo como IFC

    • ¿Qué tipo de geometría y datos exporta VisualARQ a IFC?

    Cuando un documento se guarda como archivo IFC, todos los objetos arquitectónicos de VisualARQ mantienen su información y pueden volver a editarse en las plataformas de software AEC. Los sólidos, superficies y mallas de Rhino se abren como objetos de malla independientes. Los objetos de anotación, como textos, sombreados o cotas no se exportan a IFC.
