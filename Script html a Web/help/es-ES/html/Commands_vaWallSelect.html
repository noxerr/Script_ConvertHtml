Muro: Seleccionar

Teclado:
vaMuroSeleccionar
Menú:
VisualARQ > Muro > Seleccionar
  • Muros conectados
  • Puertas
  • Ventanas

Comando para seleccionar muros y las carpinterías que contiene de distintos modos.

Cuando ejecute el comando vaMuroSeleccionar, aparecerá una lista de opciones en la línea de comandos.

  • Todos: Se seleccionan todos los muros del modelo.
  • Muros conectados: Seleccione un muro en el modelo y se seleccionarán todos los muros conectados.
  • Puertas: Seleccione un muro en el modelo y se seleccionarán todas las puertas que contenga el muro.
  • Ventanas: Seleccione un muro en el modelo y se seleccionarán todas las ventanas que contenga el muro.
