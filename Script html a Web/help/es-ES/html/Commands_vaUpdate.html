Actualizar objetos seleccionados

Teclado:
vaActualizar
Icono:
clic izquierdo
Barra de herramientas Herramientas:
Menú:
VisualARQ > Actualizar

Cuando se realizan cambios que no se ven reflejados en el modelo automáticamente, basta con seleccionar el/los objeto/s editado/s y pulsar el icono Actualizar.

NOTA: La actualización automática es un proceso que, de hacerse con frecuencia, ralentiza el funcionamiento del programa. Es por ello que únicamente ciertos comandos lo tienen.
