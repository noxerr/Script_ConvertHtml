Requisitos del sistema

Software

VisualARQ requiere Rhinoceros 4.0 SR7 o Rhinoceros 5 SR6 (32 o 64 bits) para funcionar.

Hardware

Los requisitos de hardware son los mismos que para Rhino 4.0 y Rhino 5.

Instalación

VisualARQ cuenta con un cómodo asistente de instalación, que automatiza el proceso.

Es necesario el código de licencia del producto para su instalación y uso. Una misma licencia solo se puede usar una vez simultáneamente en la red.

La instalación se realiza por defecto en: C:\Program Files\AsuniCAD\VisualARQ "1.X".

Una vez instalado el programa, al ejecutar el icono de acceso directo de Rhino, aparece un cuadro de diálogo donde se muestran las opciones de plantillas a utilizar. Para trabajar con VisualARQ, es aconsejable elegir una de las cuatro plantillas que ofrece:

  • VisualARQ - Metros.3dm
  • VisualARQ - Milímetros.3dm
  • VisualARQ - Pies.3dm
  • VisualARQ - Pulgadas.3dm


Lista de plantillas de Rhino y VisualARQ.

En Rhino 5 (32 y 64 bits), las plantillas de VisualARQ se almacenan en el siguiene directorio: C:\Users\"NombreDeUsuario"\AppData\Roaming\McNeel\Rhinoceros\5.0\Localization\en-US\Template Files. (Este directorio corresponde a la carpeta de plantillas en inglés. Cambie "en-US" por el idioma correspondiente para encontrar los directorios de las plantillas de VisualARQ en diferentes idiomas).
