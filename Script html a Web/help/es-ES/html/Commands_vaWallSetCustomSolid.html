Muro: Definir sólido personalizado

Icono:

Menú:
VisualARQ > Muro > Definir sólido personalizado

Opción para convertir un muro en otro muro desde un sólido modelado con Rhino.

Cuando ejecute el comando vaMuroDefinirSólidoPersonalizado, siga los pasos que se indican en la línea de comandos. Pulse el botón derecho del ratón o INTRO para aceptar y pasar al siguiente paso.

Pasos:

  1. Seleccione el muro que desee adaptar en el objeto de Rhino.
  2. Seleccione el objeto de Rhino que definirá el nuevo muro.

NOTA: Tenga en cuenta que el nuevo muro creado tomará como curva base (que define la alineación de los huecos dentro del muro) la curva del muro original.
