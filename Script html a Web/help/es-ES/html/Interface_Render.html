Renderizado

El administrador de materiales de VisualARQ admite materiales de todos los motores de renderizado compatibles con Rhino. La asignación de materiales de objeto en VisualARQ funciona igual que en Rhino.

La asignación de materiales de los diferentes componentes de un objeto de VisualARQ también es posible editando el Material de los atributos del componente desde el diálogo de Propiedades de estilo.

Lista de productos de animación y renderizado relacionados con Rhino:

  • Air Rendering Software: Renderizado fotorrealista, ilustración, IpR (plug-in)
  • Bongo: Animación para diseñadores (plug-in)
  • Brazil para Rhino: Herramientas de renderizado para profesionales (plug-in)
  • Flamingo nXt: Renderizado por trazado del rayos y radiosidad (plug-in)
  • Hayabusa 2.0 para Rhino: Real Time High quality renderer for Rhinoceros (complemento compatible con 3DM)
  • KeyShot: Realtime Raytracing and Global Illumination Application (complemento compatible con 3DM)
  • Maxwell Render, the light simulator: Renderizador de Maxwell para Rhino (plug-in)
  • NuGraf for large CAD rendering: Trazado de rayos multiproceso y conversión 3D para Rhino (plug-in)
  • Penguin: Renderizador conceptual no fotorrealístico (plug-in)
  • Renderosity: Foro en línea de Rhino (vínculo útil)
  • RhinoMan: Plug-in de renderizado compatible con RenderMan (plug-in)
  • V-Ray para Rhino: Renderizador de iluminación global avanzada (plug-in)
  • VSR Realtime Renderer: Renderizador en tiempo real totalmente integrado, disponible como modo de visualización de Rhino (plug-in)
