Etiqueta

Teclado:
vaEtiqueta
Icono:
clic izquierdo [clic derecho]
Barra de herramientas Documentación:
Menú:
VisualARQ > Etiqueta > Etiqueta

El comando vaEtiqueta se utiliza para mostrar las etiquetas con textos de referencia específicos en los objetos de VisualARQ que están en el modelo.

Los objetos a etiquetar deben tener el campo etiqueta rellenado antes de ejecutar este comando. El campo de etiqueta puede editarse en cada objeto de VisualARQ en la sección Propiedades de VisualARQ (en el panel Propiedades de Rhino ), en la sección General.

Cuando ejecute el comando, aparecerá el cuadro de diálogo de inserción de Etiqueta. Siga los pasos que se indican en la línea de comandos. Pulse el botón derecho del ratón o INTRO para aceptar y pasar al siguiente paso.


Diálogo de inserción de etiqueta

Pasos:

  1. Especifique un estilo de etiqueta desde el cuadro de diálogo de inserción.
  2. Seleccione los objetos a etiquetar.

La etiqueta se creará automáticamente junto al objeto seleccionado.

Puntos de control de etiquetas

Los objetos de etiqueta no tienen puntos de control. Se pueden mover independientemente de la posición de sus objetos de referencia. Cuando sus objetos de referencia se mueven, las etiquetas también se mueven automáticamente, manteniendo su posición relativa, a menos que la opción Posición fija está activada.

Eliminar una etiqueta

Solo tiene que seleccionar la etiqueta y eliminarla.

Parámetros de etiqueta

Los siguientes parámetros pueden definirse para cada objeto de etiqueta, una vez se han creado. Pueden editarse en la sección Propiedades de VisualARQ (en el panel Propiedades de Rhino ).
  • Estilo: el estilo de etiqueta define los atributos de las etiquetas.
  • Posición: coordenadas XYZ universales de la etiqueta.
  • Posición fija: opción para fijar la posición de la etiqueta, aunque se mueva su objeto de referencia.
  • Formato: formato del texto que se muestra en la etiqueta. Utilice la fórmula "%<elemento.etiqueta>%" para agregar el texto antes o después del texto de referencia de la etiqueta.
    • por ejemplo: para una puerta cuyo texto de referencia de etiqueta es "D3", puede escribir "Puerta %<elemento.etiqueta>%" para mostrar "Puerta D3" en la etiqueta.
