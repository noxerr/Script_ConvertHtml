Bienvenido a VisualARQ

¿Qué es VisualARQ?

VisualARQ es un plug-in dirigido a los usuarios de Rhino que trabajan en el sector arquitectónico (arquitectos, interioristas, promotores, etc). Ayuda a potenciar el flujo de trabajo y a mejorar la eficiencia del proceso de diseño, al permitir al usuario realizar un mayor número de tareas dentro de un único entorno de producción. De esta manera, reduce la pérdida de tiempo e información que conlleva el tener que recurrir a varios programas y formatos a lo largo de la elaboración del proyecto.

VisualARQ está totalmente integrado en la interfaz de Rhino y ofrece un potente editor de funciones que permite crear y modificar, con gran facilidad y rapidez, objetos arquitectónicos paramétricos. Asimismo, proporciona una serie de herramientas arquitectónicas que favorecen y facilitan el proceso de diseño y documentación.

A diferencia de otros productos del mercado, VisualARQ se integra de manera intuitiva en el flujo de trabajo natural de Rhino, con la menor curva de aprendizaje.

Características

  • Plug-in integrado totalmente en la interfaz de Rhino.
  • Dirigido a usuarios de Rhino que trabajan en el sector de la arquitectura.
  • Potencia el flujo de trabajo y mejora la eficiencia del proceso de diseño.
  • Permite realizar un mayor número de tareas, con mayor facilidad, dentro de un único entorno de producción.
  • Ofrece un potente editor de funciones para crear y editar fácil y rápidamente objetos arquitectónicos paramétricos.
  • Proporciona una serie de herramientas arquitectónicas.
