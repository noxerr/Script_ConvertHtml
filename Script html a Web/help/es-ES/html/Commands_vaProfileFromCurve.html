Perfil: Desde curva

Teclado:
vaPerfilDesdeCurva
Icono:
clic izquierdo
Barra de herramientas Herramientas:
Menú:
VisualARQ > Perfil > Desde curva

El comando vaPerfilDesdeCurva permite crear perfiles personalizados a partir de curvas planas y cerradas. Estos perfiles pueden utilizarse para crear estilos personalizados de pilares, vigas, puertas, ventanas y huecos, además de formas de componentes de barandas.

Pasos:

  1. Seleccione una curva plana y cerrada para definir la forma del perfil.
  2. Escriba el Nombre de perfil en la línea de comandos

Una vez creado el perfil personalizado, éste aparece al final de la lista de la librería de perfiles paramétricos predefinidos para pilares, vigas, puertas y ventanas.
El perfil resultante NO es paramétrico, por lo que podrá utilizarse únicamente con las medidas y proporciones con las que fue dibujado.
