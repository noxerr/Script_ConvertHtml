Importación de estilos

Teclado:
vaImportarEstilos
Menú:
VisualARQ > Estilos > Importar

El comando vaImportarEstilos permite importar estilos de cualquier documento de VisualARQ, para tenerlos disponibles dentro de la librería de estilos del fichero .3dm) que esté en uso.
Los archivos importados tienen la extensión .val (librería de VisualARQ) y deben haber sido exportados previamente del archivo original.

Cuando ejecute el comando vaImportarEstilos, aparecerá el cuadro de diálogo Importar estilos de VisualARQ.

Pasos:

  1. Abra el archivo .val que contiene los estilos de objeto que se desee importar al fichero .3dm en uso. Si la ruta del fichero no se encuentra dentro del menú desplegable, pulse el botón Examinar para buscarlo dentro de su equipo. Pulse el botón Siguiente para continuar.
  2. En la lista, seleccione los estilos de objeto que desee importar y pulse el botón Importar para finalizar el proceso.

Paso 1. Seleccione el archivo .val

Paso 2. Elegir los estilos de objeto a importar

Cada nuevo estilo de objeto importado se abrirá en un cuadro de diálogo Estilo.
