Comandos por teclado

Además de las barras de herramientas y de los menús (desplegables y contextuales), la forma de acceder a cualquier comando de VisualARQ es tecleando su nombre en la línea de comandos.

Los comandos que ofrece VisualARQ comienzan con el prefijo va, seguido por el nombre del comando. Con sólo teclear el prefijo (va), la línea de comandos despliega un listado donde se muestran todas las opciones correspondientes a VisualARQ.

Para ejecutar un comando desde la línea de comandos, escriba el nombre del comando. Gracias a la función de autocompletado, se mostrarán todas las opciones de comandos que comienzan con las letras escritas hasta ese momento.
Finalmente, seleccione con el botón izquierdo del ratón el comando deseado del listado de comandos.

Al igual que con los comandos de Rhino, la línea de comandos muestra también las solicitudes y las opciones de los comandos de VisualARQ entre paréntesis:

  • Las solicitudes son los mensajes que guían al usuario dentro de un comando, indicándole que realice una acción o introduzca información.
  • Las opciones de comandos cambian según las funciones de un comando. Ofrecen métodos alternativos de utilización de los comandos.

Para activar una opción, pulse el botón izquierdo del ratón sobre la opción deseada o bien escriba el nombre de la opción o la letra de acceso directo (que aparece subrayada) en la línea de comandos.

Para terminar un comando, pulse INTRO.
