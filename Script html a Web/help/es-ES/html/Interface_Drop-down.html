Menú desplegable

Dentro del menú desplegable de VisualARQ se organizan la mayoría de los comandos de la aplicación. Ahí aparecen listados todos los objetos y herramientas de VisualARQ que se encuentran en las barras de herramientas, con sus correspondientes opciones.


Menú desplegable VisualARQ.

NOTA: El menú desplegable de VisualARQ aparece cuando se carga VisualARQ en la sesión actual de Rhino después de instalarlo. Si el menú no aparece, es posible que VisualARQ esté utilizando un archivo antiguo de "visualarq.tb." (en Rhino 4.0) o "visualarq.rui" (en Rhino 5). En ese caso, debe eliminar este archivo manualmente, desinstalar VisualARQ y reinstalarlo. Los archivos "visualarq.tb." / "visualarq.rui" se encuentran en los siguiente directorios:

  • "visualarq.tb": (Rhino 4.0): (C:\ProgramData\McNeel\Rhinoceros\4.0)
  • "visualarq.rui" (Rhino 5.0 de 32 y 64 bits): (C:\Users\"NombreDeUsuario"\AppData\Roaming\McNeel\Rhinoceros\5.0\UI )
