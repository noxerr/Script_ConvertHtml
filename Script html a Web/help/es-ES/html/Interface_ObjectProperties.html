Teclado:

Icono:

VisualARQ > Propiedades > Propiedades de objeto...
Acceso directo:

Cuando se ejecuta el comando vaPropiedades, aparece en pantalla el cuadro de diálogo Propiedades de objeto . Este cuadro de diálogo permite editar algunas de las propiedades de los objetos seleccionados, independientemente de las propiedades del estilo al que pertenecen.

¿Cómo se organiza este cuadro de diálogo?

El cuadro de diálogo Propiedades de objeto se divide en tres partes:


Cuadro de diálogo Propiedades de objeto, propiedades del objeto Escalera.

  1. En el recuadro superior izquierdo se muestra la lista de objetos seleccionados en el momento de ejecutar el comando.
  2. En el recuadro inferior izquierdo se muestra una vista previa del objeto seleccionado. Puede definir las siguientes opciones para esta ventana de vista previa desde la barra de herramientas de la parte superior del diálogo:
    • Opción para mostrar u ocultar la ventana Vista previa.
    • Opción en mostrar el objeto en la vista en Planta o Perspectiva.
    Es posible modificar la vista de la previsualización utilizando el ratón, de una forma similar a como se manejan las vistas de Rhino:
    • Botón derecho del ratón: permite rotar el punto de vista en referencia al objetivo, para ver el modelo entero desde diferentes puntos situados alrededor de él.
    • CTRL + Botón derecho: permite acercar o alejar el punto de la cámara, para poder ampliar o reducir el campo de visión.
    • MAYÚS + Botón derecho: permite desplazar el punto de vista paralelamente al plano de visión: hacia arriba, hacia abajo, hacia la izquierda o hacia la derecha.
  3. Esta zona muestra las diferentes propiedades o parámetros que pueden editarse para cada objeto seleccionado. Esta zona comprende distintas pestañas, que varían dependiendo del tipo de objeto.

Editar las propiedades del objeto

Para editar un objeto, seleccione de la lista el objeto que quiera modificar y edite sus propiedades desde las pestañas ubicadas en el recuadro derecho.

Fichas

  • Estilo
    Todos los objetos que usan cuadros de diálogo disponen de esta pestaña. Se utiliza para asignar un estilo al objeto seleccionado.
  • Posición
    Todos los objetos que usan cuadros de diálogo disponen de esta pestaña. En ella se definen los parámetros de posición de cada objeto.

    Parámetros de posición del objeto Muro, Muro cortina, Baranda y Escalera:
    • Alineación (izquierda / centro / derecha)
    Parámetros de posición del objeto Pilar y Viga:
    • Alineación (punto de inserción en la geometría del perfil de sección)
    • Marca de inserción (solo para pilares)
    • Desfase (en x / y respecto al punto de inserción)
    Parámetros de posición del objeto Puerta, Ventana y Hueco:
    • Alineación (interior / centro / exterior, respecto al muro)
    • Elevación (respecto al muro)
  • Geometría
    Los muros, muros cortina, pilares y escaleras disponen de esta ficha. En ella se definen los valores geométricos (medidas) de cada objeto.
    • Anchura
    • Altura
  • Perfil
    Vigas, pilares, puertas, ventanas y huecos (los objetos que utilizan un perfil) disponen de esta ficha. En ella se definen las medidas del perfil que utiliza el objeto.
  • Intersecciones
    Solo los muros disponen de esta ficha. En ella se define el valor de Radio de unión y el valor de Extensión máxima del objeto Muro.
  • Hueco
    Solo las puertas y las ventanas disponen de esta ficha. En ella se define el valor de porcentaje de apertura.
  • Peldaños
    Sólo las escaleras disponen de esta pestaña. En ella se define el valor y número de Peldaños (Huella y Contrahuella).
  • Marca
    Solo las secciones tienen esta pestaña. Se utiliza para cambiar el texto de referencia de la marca de sección.
