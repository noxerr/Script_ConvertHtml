Contorno de hueco de escalera

Teclado:
vaEscaleraContornoDeHueco
Menú:
VisualARQ > Escalera > Contorno de hueco

Crea una curva de perfil que puede utilizarse para calcular el perfil a sustraer en una losa para mantener la altura de hueco de escalera mínima especificada.

Cuando ejecute el comando vaEscaleraContornoDeHueco, siga los pasos que se indican en la línea de comandos. Pulse el botón derecho del ratón o INTRO para aceptar y pasar al siguiente paso.

Pasos:

  1. Seleccione la escalera  desde la que desee crear el perfil.
  2. Seleccione la altura de hueco de escalera.
  3. Seleccione el grosor de la losa para la que desee mantener la altura de hueco de escalera mínima.

El perfil de curva se creará en la parte superior de la altura de escalera.
