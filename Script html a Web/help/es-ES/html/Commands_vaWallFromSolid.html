Icono:

VisualARQ > Muro > Desde sólido

Opción para crear un muro a partir de un sólido modelado con Rhino.

Cuando ejecute el comando vaMuroDesdeSólido, siga los pasos que se indican en la línea de comandos. Pulse el botón derecho del ratón o INTRO para aceptar y pasar al siguiente paso.

Pasos:

  1. Seleccione un objeto sólido. Puede ser cualquier tipo de forma siempre que sea una polisuperficie o una extrusión cerrada.
  2. Seleccione una curva base que determinará el punto donde se insertarán las puertas y ventanas dentro del muro. La curva debe ser plana y abierta.

NOTA: Asegúrese de que la curva base esté dibujada antes de ejecutar el comando.

Tras seleccionar la curva base, el sólido se convierte en muro, permitiendo la inserción de puertas y ventanas.
El muro resultante de esta operación puede ser deformado mediante sus flechas de control verticales y mediante los puntos de control horizontales de su curva base, pero NO es paramétrico. Mantiene las medidas y proporciones del sólido inicial.
