Cubierta: Cambiar ángulo de pendiente

Teclado:
vaCubiertaCambiarÁnguloDePendiente
Icono:
clic izquierdo
Barra de herramientas Cubiertas:
Menú:
VisualARQ > Cubierta > Cambiar ángulo de pendiente

Opción para cambiar la pendiente de una cubierta.

Cuando ejecute el comando vaCubiertaCambiarÁnguloDePendiente, siga los pasos que se indican en la línea de comandos. Pulse el botón derecho del ratón o INTRO para aceptar y pasar al siguiente paso.

Pasos:

  1. Seleccione la Cubierta a la que desee cambiar el ángulo de inclinación.
  2. Introduzca un nuevo valor de ángulo en la línea de comandos.

El valor de inclinación de cubierta también puede modificarse en la sección Propiedades de VisualARQ (en el panel Propiedades de Rhino).
