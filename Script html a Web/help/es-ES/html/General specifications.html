Especificaciones generales

Software

VisualARQ requiere Rhinoceros 4.0 SR7 o Rhinoceros 5 SR6 (32 o 64 bits) para funcionar.

Versiones y actualizaciones

VisualARQ comprueba automáticamente si hay actualizaciones. De haberlas, las muestra con un icono en el área de notificación de la barra de tareas de Windows.

Puede encontrar una opción para desactivar la comprobación de actualizaciones automáticas en la sección de VisualARQ dentro del cuadro de diálogo de Opciones de Rhino.

Idiomas

La interfaz, el instalador, las plantillas y las barras de herramientas de VisualARQ están disponibles en seis idiomas: inglés, español, francés, italiano, alemán y checo.

Dispone de una opción para escoger el idioma de VisualARQ en la sección de VisualARQ dentro del cuadro de diálogo de Opciones de Rhino.

Unidades

VisualARQ admite los sistemas de unidades métrico e imperial. Al cambiar las unidades del dibujo, VisualARQ muestra un mensaje preguntando si se deben escalar los estilos y definiciones de perfiles. Al importar y exportar estilos de objeto, estos se escalan de acuerdo con las unidades del documento.

  • VisualARQ - Metros.3dm
  • VisualARQ - Milímetros.3dm
  • VisualARQ - Pies.3dm
  • VisualARQ - Pulgadas.3dm

Capas

VisualARQ permite configurar la capa por defecto en comandos de creación. El diálogo de capas se encuentra en la sección de VisualARQ dentro del cuadro de diálogo de Opciones de Rhino.

Al iniciar los comandos de creación de objetos, se pone activa la capa donde irán. Al terminar el comando, se restaura la capa activa anterior, excepto si el usuario la cambia durante el comando.

Símbolos

VisualARQ agrega un símbolo de escala para indicar la escala de la vista usando las unidades del dibujo. Puede encontrar una opción para desactivar este icono de escala en la sección de VisualARQ dentro del cuadro de diálogo de Opciones de Rhino.

VisualARQ añade marcas de elevación en las vistas Frontal y Lateral para facilitar el trabajo con nivel modelos de varias plantas. Puede encontrar una opción para desactivar estas marcas de elevación en la sección de VisualARQ dentro del cuadro de diálogo de Opciones de Rhino.

Plantillas

Al instalar el programa, las plantillas de VisualARQ se copian a la carpeta de plantillas predeterminada de Rhino.

Las plantillas de VisualARQ contienen una librería predefinida de estilos de objeto, que se pueden modificar, importar y exportar para utilizarse en otros archivos.

Las plantillas de VisualARQ tienen una tolerancia de 1 mm.

Ayuda

VisualARQ incluye algunas opciones de ayuda que se encuentran en el menú principal de VisualARQ.
Por ejemplo, la Ayuda en línea en seis idiomas, enlaces a la sección de Soporte y el sitio web de VisualARQ, y el diálogo Acerca de con información sobre la licencia, el nombre de usuario, la empresa y el número de serie.
