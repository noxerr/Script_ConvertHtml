Matriz de carpintería

Teclado:
vaMatrizDeHueco
Icono:
clic izquierdo
Barra de herramientas Herramientas:
Menú:
VisualARQ > Hueco > Matriz de hueco

El comando vaMatrizDeHueco crea copias de puertas o ventanas dentro de un muro en un número especificado de filas y columnas.

Cuando ejecute el comando vaMatrizDeHueco, siga los pasos de la línea de comandos. Pulse el botón derecho del ratón o INTRO para aceptar y pasar al siguiente paso.

Pasos:

  1. Seleccione el muro donde desee realizar una matriz de huecos. Este muro debe contener todos los huecos que utilizará para realizar la matriz posteriormente.
  2. Seleccione todos los huecos que desee copiar. Pueden ser puertas, ventanas o ambas, pero deben estar insertadas en el mismo muro que eligió previamente.
  3. Introduzca el Número de copias en la dirección de trayectoria.
  4. Introduzca el Número de copias en la dirección de elevación.
  5. Introduzca el Punto base desde el que empezará a realizar las copias en la dirección de trayectoria.
  6. Introduzca la Distancia de trayectoria entre cada copia de objeto en de la dirección de trayectoria del muro.
  7. Introduzca el Punto base desde el que empezará a realizar las copias en la dirección de elevación.
  8. Introduzca la Distancia de trayectoria entre cada copia de objeto en de la dirección de elevación del muro.
