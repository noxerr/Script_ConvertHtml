Ouverture

Clavier :
vaOuverture
Menu :
VisualARQ > Ouverture > Ouverture

Le fonctionnement et le comportement d'une ouverture sont les mêmes que pour une fenêtre ou une porte.
Les ouvertures ne peuvent être insérées que dans un mur. Les deux éléments sont liés et l'ouverture est générée automatiquement dans le mur où elle est insérée.

Lorsque vous lancez la commande vaOuverture, la boîte de dialogue d'insertion d'une ouverture apparaît. Cette boîte de dialogue affiche une liste de tous les styles d'ouverture disponibles, ainsi que des options de base et un aperçu de la porte.

Les étapes nécessaires à l'insertion d'une ouverture dans un mur dépendront des options d'insertion sélectionnées.

Étapes :

  1. Sélectionnez le Style d'ouverture que vous voulez utiliser et définissez les options d'insertion de base.
  2. Il existe deux options pour permettre l'insertion d'une ouverture dans un mur :

    • Décalage : insérère l'ouverture à une distance spécifique (décalage) de l'extrémité la plus proche du mur où l'ouverture est insérée. Indiquez le côté d'ouverture en déplaçant le curseur ; la commande le définit automatiquement selon la position du curseur. Cliquez pour insérer l'ouverture.
    • Point d'insertion : insère librement l'ouverture dans le mur. Cliquez pour insérer l'ouverture au point désiré.
  3. Indiquez le côté d'ouverture en cliquant sur le côté désiré du mur. Cette étape n'est pas nécessaire lorsque vous insérez une ouverture avec l'option Décalage.

Vous pouvez définir autant d'ouvertures que vous voulez et vous pouvez annuler la commande en appuyant sur Entrée ou Échap.

Avec les deux options d'insertion, l'ouverture est générée automatiquement dans le mur.


Boîte de dialogue d'insertion d'un ouverture

Ouverture à partir d'une courbe

Les ouvertures peuvent être créées à partir de courbes, qui déterminent leur forme.

Étapes :

  1. Lancez la commande vaOuverture.
  2. Sélectionnez le Style d'ouverture que vous voulez utiliser et définissez les options d'insertion de base.
  3. Sélectionnez l'option ÀPartirDeCourbes dans la ligne de commandes.
  4. Sélectionnez une ou plusieurs courbes dans le modèle. Cette courbe doit être fermée et verticale.

L'ouverture est générée à l'intérieur du mur le plus proche de la courbe sélectionnée.

Remarque : Les courbes sélectionnées pour créer des ouvertures ne doivent pas obligatoirement être orientées parallèlement au mur où les ouvertures seront placées. Les ouvertures sont insérées et orientées automatiquement en fonction de la position du mur le plus proche.

Points de contrôle

Les ouvertures ont cinq points de contrôle :

  1. Point de contrôle pour le côté d'ouverture
  2. Point d'insertion gauche
  3. Point d'insertion du centre
  4. Point d'insertion droit
  5. Point de contrôle de la marque de référence

Pour déplacer l'ouverture (dans le mur ou sur un autre mur), sélectionnez-la à partir d'un des points et faites-la glisser où vous voulez.

Les ouvertures ne peuvent être sélectionnées que dans la vue de Dessus.

Options et paramètres d'insertion

Alignement

L'ouverture peut être alignée en fonction de l'épaisseur du mur selon 3 méthodes :

  • Intérieur
  • Centre
  • Extérieur

Pour modifier l'alignement d'une ouverture, modifiez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).

Alignement vertical

Deux alignements verticaux par rapport à la ligne de base du mur peuvent être utilisés lors de l'insertion des ouvertures :

  • Inférieur : distance de référence entre la ligne de base du mur et la surface inférieure de l'ouverture.
  • Supérieur : distance de référence entre la ligne de base du mur et la surface supérieure de l'ouverture.

L'alignement vertical n'est pas une propriété de style. La valeur est définie dans la boîte de dialogue d'insertion pour chaque objet inséré.


Pour modifier l'alignement vertical d'une ouverture, changez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).

Élévation

L'élévation d'une ouverture est toujours mesurée par rapport à la ligne de base du mur où elle est insérée, quel que soit son niveau. Lorsque l'ouverture a été insérée, la valeur d'élévation indique la position de l'ouverture dans les coordonnées Z du repère général, quels que soient le mur et le niveau d'insertion. L'élévation n'est pas une propriété de style. La valeur est définie dans la boîte de dialogue d'insertion pour chaque objet inséré.
La valeur d'élévation par défaut pour les ouvertures est de 0. Si vous modifiez cette valeur, la boîte de dialogue d'insertion montrera la dernière valeur utilisée.

Pour modifier l'élévation d'une ouverture, changez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).
