Mur : Diviser

Clavier :
vaMurDiviser
Icône :
clic
Barre d'outils Murs :
Menu :
VisualARQ > Mur > Diviser

Option permettant de diviser verticalement un mur en un point donné afin d'obtenir deux murs différents.

Lorsque vous lancez la commande vaMurDiviser, suivez les étapes de la ligne de commandes. Appuyez sur le bouton de droite ou Entrée pour accepter et passer à l'étape suivante.

Étapes :

  1. Sélectionnez le mur à diviser.
  2. Sélectionnez le point où vous voulez diviser le mur. Ce point est toujours situé le long de la ligne de base du mur.

Le mur original est maintenant divisé en deux murs différents qui peuvent être modifiés indépendamment l'un de l'autre.
