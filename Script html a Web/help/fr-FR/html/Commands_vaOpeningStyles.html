Styles d'ouverture

Clavier :
vaOuvertureStyles
Menu :
VisualARQ - Ouverture > Styles d'ouverture...
Menu contextuel :
Ouverture > Propriétés de style
Raccourci :
Sélectionner une ouverture, appuyer sur Alt, Maj ou Ctrl et double cliquer

Le style d'ouverture définit les caractéristiques de l'ouverture. Vous pouvez définir les paramètres suivants pour chaque style d'ouverture : Nom, Attributs, Forme (géométrie) et Blocs personnalisés.

Les styles d'ouverture sont définis par un composant de Dormant qui peut être ajouté ou supprimé à tout moment.


Différentes parties et composants d'une ouverture :
  1. 1 : Profil de l'ouverture
  2. 2 : Dormant d'ouverture

Pour créer un style d'ouverture, cliquez sur le bouton Nouveau style dans le gestionnaire de styles. Utilisez l'assistant pour définir les paramètres suivants.

  • Utilisez la commande vaPropriétés, puis sélectionnez une ouverture et appuyez sur Entrée, ou

Les modifications sont appliquées au style de l'élément sélectionné.

Attributs

Les attributs définissent les caractéristiques de chaque style d'ouverture pour l'affichage et l'impression. Attributs valides :

  • Général : Calque, Visibilité et Matériau
  • Projection : Couleur, Type de ligne, Largeur d'impression de ligne et Couleur d'impression de ligne
  • Section : Couleur, Type de ligne, Largeur d'impression de ligne, Motif, Échelle du motif, Angle du motif et Couleur du motif

Forme

La forme de l'ouverture est définie par un profil. VisualARQ possède une bibliothèque contenant des profils paramétriques prédéfinis : rectangulaire, circulaire, arc à 90º, gothique, etc.

Des profils de forme personnalisés peuvent aussi être utilisés pour définir les ouvertures.

Pour ajouter un profil personnalisé, dessinez une courbe fermée et plate et convertissez-la en profil avec la commande vaProfilÀPartirDeCourbe .
Lorsque le profil personnalisé a été créé, il apparaît à la fin de la liste dans la bibliothèque des profils prédéfinis.

Le profil obtenu n'est pas paramétrique, il ne peut donc être utilisé qu'avec les cotes et les proportions du dessin original.

Tailles

Les paramètres correspondant aux tailles des ouvertures varieront en fonction du type de profil sélectionné.
Plusieurs tailles peuvent être définies pour le même style d'ouverture.

Les tailles peuvent aussi être définies à tout moment, lors de l'insertion de l'ouverture, dans la boîte de dialogue d'insertion en choisissant l'option Tailles > Autre dans la section Profil.

Composants d'ouvertures

Dormant

Les styles d'ouverture peuvent avoir un dormant.
Pour ajouter un dormant à une ouverture, dans la liste des styles d'ouverture, cliquez avec le bouton de droite sur le nom de l' ouverture puis choisissez Nouveau > Dormant d'ouverture.
Pour supprimer le dormant d'une ouverture, dans la liste des styles d'ouverture, cliquez avec le bouton de droite sur le nom du dormant d'ouverture puis choisissez Supprimer.

Les valeurs définissant la largeur et la profondeur du dormant peuvent être modifiées. De plus, l'option Ajuster la profondeur à l'épaisseur du mur peut être activée.

Ouverture à partir de blocs personnalisés

Les styles d'ouverture peuvent être créés avec des blocs 2D et 3D qui définissent leur représentation dans les vues en plan et du modèle respectivement.
Pour créer un style d'ouverture personnalisé, choisissez un Profil (qui définit la forme de l'ouverture dans le mur), un Bloc 2D (qui définit la représentation dans le plan) et un Bloc 3D (qui définit la représentation du modèle).

Étapes :

  • Créer un profil personnalisé : dessinez une courbe fermée et plate et convertissez-la en profil avec la commande vaProfilÀPartirDeCourbe .
    Lorsque le profil personnalisé a été créé, il apparaît à la fin de la liste dans la bibliothèque des profils prédéfinis.
    Le profil obtenu n'est pas paramétrique, il ne peut donc être utilisé qu'avec les cotes et les proportions du dessin original.
  • Créer les blocs 2D et 3D : créez le modèle 3D et dessinez la représentation 2D à partir de celui-ci. Puis, créez deux blocs à partir du modèle et du dessin en utilisant la commande Bloc (Menu Édition  > Bloc > Créer une définition de bloc).
  • Créer le nouveau style d'ouverture : lorsque vous avez créé le profil et les blocs 2D et 3D, cliquez sur le bouton Nouveau style... dans la boîte de dialogue Gestionnaire de styles. Utilisez l'assistant pour attribuer un profil personnalisé au style.

    Passez les boîtes de dialogue où sont définis les paramètres du style d'ouverture en cliquant sur Suivant jusqu'à ce que la boîte de dialogue de définition d'un bloc personnalisé apparaisse.
    Dans la boîte de dialogue Bloc personnalisé, cochez la case qui active les options permettant d'utiliser des blocss personnalisés pour les vues du modèle et en plan. Sélectionnez ensuite des blocs 3D et 2D dans le menu déroulant afin de définir la représentation des ouvertures dans ces vues.

Avant de créer les blocs (2D/3D), vérifiez que les objets que vous voulez convertir en ouverture de VisualARQ sont alignés avec les axes X, Y et Z du repère général.
Une fois les objets alignés avec le repère général, choisissez le point inférieur central de votre géométrie comme point d'insertion lorsque vous créez le bloc.
L'orientation de l'ouverture sera ainsi correcte.



Assistant de création d'ouvertures, définition d'un bloc

Lorsque des blocs 3D et 2D sont utilisés pour définir la représentation d'un style personnalisé, les paramètres de taille et de dormant n'affectent pas l'apparence du style créé. Ces profils ne sont PAS paramétriques : leur apparence (taille, proportion, niveau de détail) dépend du bloc utilisé pour les définir.

VisualARQ peut également créer des styles d'ouverture hybrides, en remplaçant uniquement la représentation en plan (par un bloc 2D) et en utilisant la vue du modèle d'un style paramétrique ou vice-versa.
La représentation qui n'utilise pas un bloc doit être définie avec les paramètres du style (taille, et dormant), comme dans tous les styles d'ouverture standard.
