Lorsque vous lancez la commande vaPoutre, la boîte de dialogue d'insertion d'une poutre apparaît. Cette boîte de dialogue affiche une liste de tous les styles de poutres disponibles, ainsi que des options de base et un aperçu de la poutre.

Étapes :

  1. Sélectionnez le Style de poutre que vous voulez insérer et définissez les options d'insertion de base.
  2. Sélectionnez le Point de départ de la poutre en cliquant sur le point désiré.
  3. Sélectionnez la Fin de la poutre en cliquant sur le point désiré.

Vous pouvez définir autant de poutres que vous voulez et vous pouvez annuler la commande en appuyant sur Entrée ou Échap.

Poutre à partir d'une courbe

Les poutres peuvent être créées à partir d'une courbe du modèle :

Étapes :

  1. Utilisez la commande vaPoutre pour créer une nouvelle poutre. La boîte de dialogue d'insertion d'une poutre apparaît.
  2. Sélectionnez le Style de poutre que vous voulez insérer et définissez les options d'insertion de base.
  3. Sélectionnez ÀPartirDeCourbes dans la ligne de commandes puis sélectionnez une courbe dans le modèle.
  4. Appuyez sur ENTRÉE ou cliquez avec le bouton droit pour terminer la commande.

Poutre créée à partir d'une courbe, avec les points de contrôle activés.

Points de contrôle

Les poutres peuvent avoir différents points de contrôle en fonction de la méthode utilisée pour leur création. Ces points de contrôle peuvent être déplacés dans toutes les directions, afin de modifier la position et les dimensions de la poutre.

  • Poutre droite : la poutre possède deux points de contrôle, aux extrémités de l'objet.
  • Poutre à partir d'une courbe : la poutre possède autant de points de contrôle que la courbe à partir de laquelle elle a été créée.

Options et paramètres d'insertion

Alignement

L'alignement de la poutre détermine la position du point d'insertion de celle-ci en fonction de la forme de son profil.
Options d'alignement :

  • Supérieur gauche, Supérieur centre et Supérieur droit
  • Milieu gauche, Centre et Milieu droite
  • Inférieur gauche, Inférieur centre et Inférieur droit

Pour modifier l'alignement d'une poutre, modifiez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino

Décalage

Montre la distance (décalage) entre les points X et Y de la poutre par rapport au point d'insertion.

Pour modifier le décalage d'une poutre, changez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans la boîte de dialogue icône des propriétés de Rhino).

Élévation

L'élévation d'une poutre est toujours mesurée par rapport à la ligne de base du niveau où elle est insérée. L'élévation n'est pas une propriété de style. La valeur est définie dans la boîte de dialogue d'insertion pour chaque poutre insérée lors de sa création.
La valeur d'élévation par défaut pour les poutres est de 3 mètres. Si vous modifiez cette valeur, la boîte de dialogue de la Poutre montrera la dernière valeur utilisée.

Roulis

Cette valeur détermine l'angle de rotation de la poutre autour de son axe, en prenant la position de son alignement comme centre de rotation.

Le roulis de la poutre peut uniquement être modifié dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).

Symétrique

Lorsque cette option est activée, l'orientation de la poutre est transformée en son image symétrique.

Cette option peut uniquement être modifiée dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).
