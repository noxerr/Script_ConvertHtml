Exportation de styles

Clavier :
vaExporterStyles
Menu :
VisualARQ > Styles > Exporter

La commande vaExporterStyles permet d'exporter des styles d'objet de VisualARQ afin de pouvoir les réimporter dans d'autres fichiers (.3dm).
Les fichiers exportés ont l'extension .val (bibliothèque de VisualARQ).

Lorsque vous lancez la commande vaExporterStyles, la boîte de dialogue Expporter des styles apparaît. Cliquez sur le bouton Suivant pour passer à l'étape suivante.

Étapes :

  1. Choisissez une des deux options d'exportation du fichier :
    • Fichier existant : option pour enregistrer les nouveaux styles dans un fichier .val existant. Ouvrez le fichier .val pour ajouter les styles d'objets que vous voulez exporter. Si le chemin du fichier n'est pas affiché dans le menu déroulant, cliquez sur le bouton Parcourir pour le chercher dans votre système. Les styles d'objets portant le même nom seront remplacés et les nouveaux styles seront ajoutés.
    • Nouveau fichier : option pour créer un nouveau fichier .val contenant les styles d'objets que vous voulez exporter. Donnez un nom au nouveau fichier .val et cliquez sur le bouton Parcourir pour l'enregistrer dans le dossier désiré. Sélectionnez les unités dans lesquelles vous voulez enregistrer le fichier.
  2. Dans la liste affichée, sélectionnez tous les styles que vous voulez exporter et cliquez sur le bouton Exporter pour terminer l'opération.
