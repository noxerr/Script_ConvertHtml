Gestionnaire de styles

Le gestionnaire de styles permet de modifier les propriétés de style d'un objet et de créer de nouveaux styles pour cet objet.

Pour créer de nouveaux styles d'objet, cliquez sur le bouton Nouveau style.


Gestionnaire de styles pour l'objet Porte.

Un assistant vous guide par des étapes simples en vous demandant les paramètres du style d'objet défini.


Assistant de style de porte.

Un aperçu de l'objet est affiché en bas à gauche lorsque vous sélectionnez un style dans la liste.

Le composant sélectionné est marqué en turquoise lorsque l'objet possède plusieurs composants.

L'aperçu affiché peut être modifié avec la souris tout comme dans les vues de Rhino :

  • Bouton droit de la souris : permet de faire tourner la vue pour afficher le modèle selon différents points de vue.
  • Ctrl + bouton droit : permet de zoomer pour agrandir ou réduire le champ de vision.
  • Maj+ bouton droit : permet de déplacer le point de vue parallèlement au plan de vision, vers le haut, le bas, la droite ou la gauche.


Gestionnaire de styles, onglet Attributs.

Pour chaque style d'objet sélectionné, vous pouvez définir les propriétés affichées dans la case de droite. Cette zone comprend plusieurs onglets, qui peuvent être différents en fonction du type d'objet :

  • Attributs
    Cet onglet est affiché pour tous les objets qui utilisent une boîte de dialogue. Il permet de définir les fonctions (attributs) de chaque style d'objets et de chacun de ses composants. Attributs :
    • Calque
    • Visibilité
    • Couleur d'affichage
    • Type de ligne
    • Largeur d'impression
    • Couleur d'impression
    • Matériau
    • Hachures
    • Style de texte
    • Style de cote

    REMARQUE : À partir de l'attribut Matériau du style d'objet, vous pouvez assigner un matériau différent à chaque composant de l'objet. Pour assigner un seul matériau à tout l'objet, suivez la procédure habituelle de Rhino.
  • Géométrie
    Les murs, poteaux, portes et fenêtres possèdent cet onglet. Les valeurs géométriques pour chaque objet et chaque composant sont définies dans cet onglet.
    Valeurs géométriques pour l'objet Mur :
    • Hauteur
    • Largeur (ne peut être modifiée que pour chaque composant du mur)
    Valeurs géométriques pour l'objet Poteau :
    • Hauteur
    • Profil (définit la forme de l'objet)
    • Largeur du Revêtement
    Valeurs géométriques pour l'objet Porte :
    • Profil (définit la forme de l'objet)
    • Tailles (largeur et profondeur) du dormant et de la cale
    • Tailles (largeur et profondeur) et côté d'ouverture du battant
    Valeurs géométriques pour l'objet Fenêtre :
    • Profil (définit la forme de l'objet)
    • Tailles (largeur et profondeur) du dormant et de la cale
    • Tailles (largeur et profondeur) et côté d'ouverture du battant
    • Tailles (épaisseur, prolongement et profondeur) du rebord.
  • Tailles
    Cet onglet ne sert qu'aux objets qui utilisent des profils pour définir leur forme (poteaux, portes et fenêtres). Il permet de définir les tailles (largeur et auteur) du profil utilisé par l'objet. Cette option permet de créer différentes tailles pour le même profil.
  • Blocs
    Seules les portes et les fenêtres possèdent cet onglet. L'onglet définit les représentations 2D (plan) et 3D (modèle) de l'objet, en assignant un bloc déjà créé par l'utilisateur, avec des paramètres personnalisés pour chaque représentation. Cette option permet de créer des portes et des fenêtres personnalisées.
  • Jambages
    Seuls les murs présentent cet onglet. Il permet de définir les valeurs de chaque type de jambage du mur (Départ, Final, Angle et Ouverture).
  • Type
    Seuls les escaliers présentent cet onglet. Il permet de définir le type d'escalier utilisé, suivant si l'option Marches balancées est activée ou non.
  • Étapes
    Cet onglet n'apparaît que pour les escaliers, il permet de définir les valeurs géométriques des dalles de contremarche et de giron. Valeurs géométriques pour la Dalle de giron :
    • Épaisseur de la dalle
    • Nez
    Valeurs géométriques pour la Dalle de contremarche :
    • Épaisseur de la dalle
