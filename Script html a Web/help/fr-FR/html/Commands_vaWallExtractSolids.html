Mur : Extraire des solides

Clavier :
vaMurExtraireSolides
Icône :
clic
Barre d'outils Murs :
Menu :
VisualARQ > Mur > Extraire des solides

Option pour extraire un ou plusieurs solides qui avaient été ajoutés à un mur.

Lorsque vous lancez la commande vaMurExtraireSolides, suivez les étapes de la ligne de commandes. Appuyez sur le bouton de droite ou Entrée pour accepter et passer à l'étape suivante.

Étapes :

  1. Sélectionnez le mur dans lequel vous voulez extraire des solides qui ont été ajoutés auparavant.
  2. Cliquez avec le bouton de droite ou appuyez sur Entrée pour valider la sélection du mur ; les solides seront séparés du mur et redeviendront des éléments indépendants.
