Mur : Inverser

Clavier :
vaMurInverser
Icône :
clic
Barre d'outils Murs :
Menu :
VisualARQ > Mur > Inverser

Cette commande inverse la direction du chemin et des côtés d'un mur : le point de départ devient le point final, la partie intérieure devient la partie extérieure et vice-versa.

REMARQUE : N'oubliez pas que les portes, les fenêtres et les ouvertures comprises dans le mur sélectionné conservent leur position mais peuvent changer de côté d'ouverture lorsque le chemin est inversé.
