Panneau des Propriétés objet (dans le panneau Propriétés de Rhino)


Section des propriétés de VisualARQ pour un poteau
dans la boîte de dialogue des propriétés de Rhino.

Vous pouvez également modifier les propriétés d'un objet de VisualARQ dans la section Propriétés de VisualARQ. (disponible également dans le menu de Rhino Édition > Propriétés objet...
Cette section est intégrée dans le panneau des propriétés objet de Rhino mais elle n'est affichée que lorsque vous sélectionnez un objet de VisualARQ.

Les paramètres disponibles dans cette section ne sont pas les mêmes pour tous les objets de VisualARQ. Cet exemple montre les paramètres qui peuvent être modifiés pour l'objet Poteau.

Les paramètres de l'objet de VisualARQ sélectionné qui ne dépendent pas d'un style spécifique peuvent être modifiés dans la section Propriétés de VisualARQ.

Les paramètres disponibles dans cette section dépendent du type d'objet sélectionné et sont les mêmes que ceux de la boîte de dialogue Propriétés de l'objet.


Général
  • Nom : Nom de l'objet sélectionné.
  • Description : Description de l'objet sélectionné.
  • Style : Liste des styles d'objet disponibles dans les bibliothèques d'objets de VisualARQ ainsi que des nouveaux styles créés par l'utilisateur.
  • Numéro de série : Numéro de série de l'objet sélectionné.
Affichage
  • Densité isoparamétrique : Définit le nombre de courbes isoparamétriques dessinées par Rhino sur la surface.
  • Visibilité du plan : Masque ou affiche un objet dans une vue en plan à partir de sa position par rapport au plan de coupe à l'intérieur d'un niveau.
    • Caché : L'objet ne sera pas affiché, quelles que soient sa taille et sa position.
    • Dans le plan de coupe : L'objet sera affiché uniquement lorsqu'il se trouvera dans le plan de coupe.
    • En dessous du plan de coupe : L'objet sera affiché lorsqu'il se trouvera dans le plan de coupe ou en dessous.
    • Au dessus du plan de coupe : L'objet sera affiché lorsqu'il se trouvera dans le plan de coupe ou au dessus.
    • Dans le niveau : L'objet sera toujours affiché s'il se trouve à l'intérieur d'un niveau, quelles que soient sa taille et sa position.


    Schéma de visibilité du plan.


    Exemple des fenêtres qui seraient affichées dans l'image ci-dessus en fonction de l'option de visibilité dans le plan :

    • Caché : Aucun.
    • Dans le plan de coupe : Fenêtre 2.
    • En dessous du plan de coupe : Fenêtres 2 et 3.
    • Au dessus du plan de coupe : Fenêtres 1 et 2.
    • Dans le niveau : Fenêtres 1, 2 et 3.

Position
  • Point d'insertion : Coordonnées du point d'insertion d'un objet.
  • Marque d'insertion (uniquement pour les poteaux) : Active l'affichage de la marque d'insertion et de son type.
Géométrie
  • Hauteur : Définit la hauteur de l'objet. Cette valeur peut être modifiée quel que soit le style de l'objet. Dans les murs et les poteaux, cette hauteur peut aussi être définie par le style.
  • Profil : Description du profil de l'objet.
  • Largeur/Hauteur : Les dimensions de l'objet.
  • Profondeur de coupe (option uniquement disponible pour les portes, les fenêtres et les ouvertures) : Définit la profondeur maximale qu'un profil d'ouverture coupera dans le mur.
Ouverture

(Option disponible pour les portes et les fenêtres)

  • Ouverture (%) : Le pourcentage d'ouverture définit le degré d'ouverture des battants des portes et fenêtres dans les vues en plan et en perspective.
  • Côté d'ouverture : Direction dans laquelle le battant de porte ou de fenêtre s'ouvrira.
Profil

(Option disponible pour les poutres et les poteaux)

  • Roulis/Rotation : Rotation du profil de l'objet.
  • Alignement : Position d'insertion d'un objet en fonction de son profil.
  • Décalage de l'alignement : Distance à partir de la position de l'alignement.

Le bouton Autres...

En bas de la boîte de dialogue se trouve un bouton Autres... permettant d'accéder rapidement à certains raccourcis concernant les objets sélectionnés. Ces options sont similaires à celles trouvées dans le menu contextuel après avoir sélectionné un objet.

  • Supprimer
  • Dupliquer
  • Isoler les objets
  • Cacher des objets
  • Verrouiller des objets
  • Activer les points de contrôle
  • Inverser (uniquement pour les murs)
  • Sélectionner
  • Actualiser
