Attributs de Section de VisualARQ (dans le panneau Attributs de section de Rhino)


Attributs de section, panneau Propriétés de Rhino

Les Attributs de section déterminent les attributs (caractéristiques) des objets de VisualARQ et Rhino pour l'affichage et l'impression, lorsqu'ils sont affichés dans les vues en plan et lorsque le plan de coupe est activé dans le Gestionnaire de niveaux.

Les attributs de section se trouvent sous l'icône , dans le panneau Propriétés de Rhino , après avoir sélectionné un ou plusieurs objets.

Les paramètres disponibles dans cette section sont les mêmes pour tous les objets de VisualARQ et Rhino. Cependant, dans le cas des objets de VisualARQ, les attributs de section ne sont visibles que s'ils ne sont pas définis par style dans la boîte de dialogue du gestionnaire de styles correspondante.


Attributs de section :

  • Remplir avec une couleur
  • Type de ligne
  • Largeur d'impression de ligne
  • Couleur d'impression de ligne
  • Motif
  • Échelle du motif
  • Angle du motif
  • Couleur du motif
Remarque : Les attributs de Largeur d'impression de ligne et de Couleur d'impression de ligne ne sont visibles que lorsque l'Aperçu avant impression est activé.

Le bouton Autres...

En bas de la boîte de dialogue se trouve un bouton Autres... permettant d'accéder rapidement à certains raccourcis de commandes de dessin :

  • Aperçu avant impression
  • Échelle des hachures
  • Paramètres d'annotation
  • Importer des motifs de hachures
  • Importer des types de ligne

Attributs de section de VisualARQ (dans la boîte de dialogue du gestionnaire de styles)

Les attributs de section peuvent également être modifiés pour un style, à partir de la boîte de dialogue de gestion des styles d'objet de VisualARQ correspondante.

Attributs de section par style d'objet :

  • Couleur
  • Type de ligne
  • Largeur d'impression de ligne
  • Couleur d'impression de ligne
  • Motif
  • Échelle du motif
  • Angle du motif
  • Couleur du motif
