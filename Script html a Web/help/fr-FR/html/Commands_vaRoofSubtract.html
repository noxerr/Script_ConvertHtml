Toit : Soustraire une frontière

Clavier :
vaToitSoustraire
Icône :
clic
Barre d'outils Toits :
Menu :
VisualARQ > Toit > Soustraire une frontière

Option pour soustraire une frontière dans un toit.

Lorsque vous lancez la commande vaToitSoustraire, suivez les étapes de la ligne de commandes. Appuyez sur le bouton de droite ou Entrée pour accepter et passer à l'étape suivante.

Étapes :

  1. Sélectionnez le toit dans lequel vous voulez soustraire une frontière.
  2. Sélectionnez la frontière que vous voulez soustraire.

La silhouette de la frontière sélectionnée est supprimée dans la forme originale du toit.

REMARQUE : Les frontières ne doivent pas obligatoirement se trouver sur le plan du toit mais elles doivent être fermées et planes.
