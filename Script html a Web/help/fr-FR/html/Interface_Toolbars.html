Barres d'outils

Les barres d'outils de VisualARQ fonctionnent exactement comme celles de Rhino :
  • Elles peuvent être déplacées et laissées flottantes dans l'écran ou être ancrées sur le bord du modèle.
  • Elles peuvent aussi être modifiées en ajoutant ou en supprimant des boutons.
  • Certains boutons possèdent une deuxième commande, en relation avec la première, qui peut être lancée avec le bouton de droite.
  • Lorsque le curseur est placé sur un bouton, une légende affiche le nom des commandes qui peuvent être lancées avec le bouton de droite et le bouton de gauche.
  • Certains boutons (avec un triangle en bas à droite) possèdent une barre d'outils liée.

Cliquez sur l'icône et maintenez enfoncé le bouton de gauche de la souris pour afficher la barre d'outils liée à un bouton.

    • REMARQUE : Si les barres d'outils ne sont pas automatiquement affichées après l'installation de VisualARQ, elles peuvent être chargées manuellement en utilisant la même méthode que pour charger les barres d'outils de Rhino : Tapez BarreOutils dans la ligne de commandes et ouvrez le fichier de barres d'outils de VisualARQ (VisualARQ.tb  dans Rhino 4.0 ou VisualARQ.rui dans Rhino 5), qui se trouve dans un des dossiers suivants, en fonction de votre version de Rhino :

    • Rhino 4.0 : (C:\ProgramData\McNeel\Rhinoceros\4.0)
    • Rhino 5.0 (32 et 64 bits) : (C:\Users\"UserName"\AppData\Roaming\McNeel\Rhinoceros\5.0\UI)
