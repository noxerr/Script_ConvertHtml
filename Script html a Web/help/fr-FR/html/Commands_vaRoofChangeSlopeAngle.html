Toit : Changer l'angle de pente

Clavier :
vaToitChangerAnglePente
Icône :
clic
Barre d'outils Toits :
Menu :
VisualARQ > Toit > Changer l'angle de pente

Option pour changer l'angle de pente d'un toit.

Lorsque vous lancez la commande vaToitChangerAnglePente, suivez les étapes de la ligne de commandes. Appuyez sur le bouton de droite ou Entrée pour accepter et passer à l'étape suivante.

Étapes :

  1. Sélectionnez le toit dont vous voulez changer l'angle de pente.
  2. Entrer la nouvelle valeur d'angle désirée dans la ligne de commandes.

L'angle de pente d'un toit peut également être modifiée dans la section Propriétés de VisualARQ (dans la boîte de dialogue Propriétés de Rhino)
