Barre d'outils Documentation :
Menu :
VisualARQ > Élévation des ouvertures > Élévation des ouvertures

Cette commande génère des représentations 2D à partir des élévations des différentes ouvertures du modèle.

Étapes :

  1. Sélectionnez les portes et les fenêtres dont vous voulez obtenir l'élévation. La commande filtre automatiquement la sélection afin que ne soient affichées que les ouvertures.
  2. Sélectionnez le point de départ et final de l'insertion.

La commande se termine lorsque le dessin 2D de chaque ouverture sélectionnée est inséré, avec les références correspondantes. Les mêmes références seront affichées dans la vue en plan, pour indiquer la position de chaque type de portes et de fenêtres dans le modèle.

Références des ouvertures du modèle

Élévations des ouvertures cotées, avec une référence à chaque ouverture de modèle

REMARQUE : si plusieurs ouvertures (portes ou fenêtres) ont les mêmes caractéristiques - tailles, élévation et côté d'ouverture - un seul dessin est créé avec une seule référence.

Les élévations d'ouverture et leurs références sont créées par défaut sur le calque Documentation.
Pour masquer les références, désactivez le calque où elles ont été insérées.
Pour supprimer les références, sélectionnez l'ouverture que vous voulez supprimer puis ouvrez la section Propriétés de VisualARQ du panneau Propriétés de Rhino ). Supprimez la valeur du Texte de référence dans la section Référence.
