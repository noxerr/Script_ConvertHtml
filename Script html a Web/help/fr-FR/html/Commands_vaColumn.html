Des poteaux peuvent être insérés à l'intérieur ou à l'extérieur d'un mur.
Lorsqu'un poteau est inséré dans un mur, la limite du poteau coupe automatiquement la ligne du mur qu'il traverse afin de définir l'intersection entre les deux éléments (dans les vues en plan et en perspective).

Lorsque vous lancez la commande vaPoteau, la boîte de dialogue d'insertion d'un poteau apparaît. Cette boîte de dialogue affiche une liste de tous les styles de poteau disponibles, ainsi que des options de base et un aperçu du poteau.

Étapes :

  1. Sélectionnez le style de poteau que vous voulez insérer et définissez les options d'insertion de base.
  2. Sélectionnez le point d'insertion du poteau en cliquant sur le point désiré.
  3. Ensuite, indiquez l'orientation du poteau manuellement avec le curseur. Cliquez à nouveau quand le poteau est en position.

Vous pouvez définir autant de poteaux que vous voulez et vous pouvez annuler la commande en appuyant sur Entrée ou Échap.


Boîte de dialogue d'insertion d'un poteau

Points de contrôle

Les poteaux ont un seul point de contrôle, qui est toujours leur point d'insertion. Ce point correspond à l'alignement choisi pour le poteau lorsqu'il est inséré.


Points de contrôle sur les poteaux

Options et paramètres d'insertion

Alignement

L'alignement du poteau détermine la position du point d'insertion du poteau en fonction de la forme de son profil.

Options d'alignement :

  • Supérieur gauche, Supérieur centre et Supérieur droit
  • Milieu gauche, Centre et Milieu droite
  • Inférieur gauche, Inférieur centre et Inférieur droit

Pour modifier l'alignement d'un poteau, modifiez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino

Marque d'insertion

Cette marque est affichée sur le point d'alignement et indique la face fixe du poteau. Elle n'apparaît que lorsque l'option Marque d'insertion est activée. Vous pouvez choisir une des formes suivantes :

  • Circulaire
  • Triangulaire

Poteaux avec différents types de marques d'insertion

Les marques d'insertion sont uniquement affichées dans la vue en plan.
Pour modifier la marque d'insertion d'un poteau, modifiez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).

Décalage

Montre la distance (décalage) entre les points X et Y du poteau par rapport au point d'insertion.

Pour modifier le décalage d'un poteau, changez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).

Roulis

Cette valeur détermine l'angle de rotation du poteau autour de son axe, en prenant la position de son alignement comme centre de rotation.

Le roulis du poteau peut uniquement être modifié dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).

Symétrique

Lorsque cette option est activée, l'orientation du poteau est inversée symétriquement.

Cette option peut uniquement être modifiée dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).
