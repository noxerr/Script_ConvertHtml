Mur : À partir d'un solide

Clavier :
vaMurÀPartirDeSolide
Icône :
clic [ clic droit]
Barre d'outils Murs :
Menu :
VisualARQ > Mur > À partir d'un solide

Option pour créer un mur à partir d'un objet solide modélisé dans Rhino.

Lorsque vous lancez la commande vaMurÀPartirDeSolide, suivez les étapes de la ligne de commandes. Appuyez sur le bouton de droite ou Entrée pour accepter et passer à l'étape suivante.

Étapes :

  1. Sélectionnez un objet solide. Le type et la forme de ce solide ne sont pas importants mais il doit s'agir d'une extrusion ou d'une polysurface fermée.
  2. Sélectionnez une courbe de base qui déterminera le point où les portes et les fenêtres seront insérées dans le mur. La courbe doit être plane et ouverte.

REMARQUE : Vérifiez que la courbe de base est dessinée avant de lancer la commande.

Après avoir sélectionné la courbe, le solide est transformé en mur et vous pouvez commencer à insérer des portes et des fenêtres.
Le mur obtenu peut être déformé en utilisant les flèches de contrôle verticales et les points de contrôle horizontaux mais il n'est PAS paramétrique. Il conserve la taille et les proportions du solide initial.
