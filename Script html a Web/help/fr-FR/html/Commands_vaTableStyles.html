Le style de nomenclature définit les caractéristiques de la nomenclature. Vous pouvez définir les paramètres suivants pour chaque style de nomenclature : Nom, Attributs et Appliquer à.

Les styles de nomenclature sont définis par différents Champs de propriété qui peuvent être ajoutés ou supprimés à tout moment.

Nouveau style de nomenclature

Pour créer un style de nomenclature, cliquez sur le bouton Nouveau style dans le gestionnaire de styles. Suivez les instructions de l'assistant pour définir les paramètres du nouveau style de nomenclature :

  • Nom du style
  • Types d'objet : plusieurs types d'objets peuvent être sélectionnés
  • Champs de la nomenclature : indiquez le nom du champ et la propriété concernée.
  • Champ de quantité : option permettant d'indiquer le champ dans lequel les éléments de la nomenclature seront regroupés

Modifier un style de nomenclature

Lorsque le nouveau style de nomenclature est créé, il apparaît dans la liste à gauche. Sélectionnez le style de nomenclature que vous voulez définir et utilisez les onglets de droite pour définir les paramètres :

Ajouter un champ de propriétés : De nouveaux champs de propriété peuvent être ajoutés à un style de nomenclature. Sélectionnez le style nomenclature, cliquez avec le bouton de droite pour afficher le menu contextuel puis sélectionnez l'option Nouveau > Champ de propriété de la nomenclature.

Modifications sur le style en général
Utilisez la commande vaNomenclatureStyles ou,
  • Utilisez la commande vaStylePropriétés, puis sélectionnez une nomenclature et appuyez sur Entrée, ou
  • Cliquez avec le bouton de droite sur , puis sélectionnez une nomenclature et appuyez sur Entrée, ou
  • Sélectionnez une nomenclature et ouvrez le menu contextuel en appuyant sur le bouton de droite pendant un instant puis sélectionnez Nomenclature > Propriétés de style, ou
  • Appuyez sur Alt, Maj ou Ctrl puis double cliquez sur une nomenclature.
Les modifications réalisées sur un style sont appliquées à tous les éléments de ce style.
Modification d'un objet particulier
  • Utilisez la commande vaPropriétés, puis sélectionnez une nomenclature et appuyez sur Entrée, ou
  • Cliquez sur , puis sélectionnez une nomenclature et appuyez sur Entrée, ou
  • Sélectionnez une nomenclature et ouvrez le menu contextuel en appuyant sur le bouton de droite pendant un instant puis sélectionnez Nomenclature > Propriétés, ou,
  • Double-cliquez sur une nomenclature.
Les modifications sont appliquées au style de l'élément sélectionné.

Attributs

Les attributs définissent les caractéristiques de chaque style de nomenclature pour l'affichage et l'impression. Attributs valides :

  • Calque, Visibilité, Style de texte, Style de cote, Couleur, Type de ligne, Largeur d'impression de ligne, Couleur d'impression de ligne, Motif, Échelle du motif, Angle du motif et Couleur du motif.

Appliquer à

Cet onglet indique le type d'objets affichés dans la nomenclature. La commande filtre automatiquement la sélection afin que ne soient affichés que les objets dont le type est celui sélectionné.

Le gestionnaire de style des nomenclatures affiche deux onglets de plus lorsque vous sélectionnez une des colonnes de chaque style : Groupe et Propriété.

La quantité et le type de colonnes de chaque style de nomenclature peuvent être définis par l'utilisateur, en fonction de ce qui doit être quantifié. Pour les trois styles de nomenclature par défaut, les colonnes prédéfinies sont :

  • Style d'ouverture : Quantité, Référence, Largeur, Hauteur, Élévation et Côté d'ouverture.
  • Style d'espace : Nom, Périmètre et Surface.
  • Style de mur : Quantité, Style, Longueur, Surface et Volume.
Groupe

Cet onglet est utilisé pour définir les caractéristiques communes d'objets quantifiés afin qu'ils puissent être considérés comme membres d'un groupe.
Il n'apparaît que lorsque vous sélectionnez la colonne de la nomenclature correspondant à la quantité (utilisée dans les élévations d'ouverture).


Gestionnaire de styles pour les nomenclatures, onglet Groupe.

Propriété

Cet onglet est utilisé pour définir le type de propriété des objets quantifiés qui doit être affiché dans chaque colonne.
Il est affiché lorsque vous sélectionnez une des colonnes de la nomenclature, sauf dans le cas de la quantité.

Trier

Cet onglet est affiché lors de la sélection d'un style de nomenclature. Il est utilisé pour définir un critère de tri des différents champs de propriété dans le style de nomenclature et aussi pour trier les valeurs de chaque champ de propriété par ordre alphabétique ou numérique.

Cliquez sur le bouton Ajouter pour ajouter un nouveau champ de propriété qui établira le premier critère de tri des valeurs de ce style de nomenclature. Cliquez sur le nom de ce champs de propriété pour sélectionner un champ de propriété différent. Sélectionnez le type d'ordre (ascendant ou descendant) à côté du champ de propriété.

Vous pouvez ajouter des champs de propriété qui apporteront de nouveaux critères trier les valeurs dans les styles de nomenclature. Utilisez les flèches en bas à droite de la boîte de dialogue pour changer l'ordre des champs de propriété.
