VisualARQ dispose d'un module d'exportation vers le format IFC qui permet d'exporter des fichiers de VisualARQ et Rhino vers d'autres logiciels d'architecture utilisant le format IFC 2x3.

Qu'est-ce que IFC ?

Le format de fichier Industry Foundation Classes (IFC) est un standard d'échange d'objets dans l'industrie de la construction qui permet de réduire la perte d'informations lors de l'échange de fichiers entre différentes applications.

Comment exporter des documents de Rhino et VisualARQ vers le format IFC ?

  • Ouvrez le menu Fichier > Enregistrer sous et choisissez le format IFC

    • Quel type de géométrie et données sont exportées par VisualARQ vers IFC ?

    Lorsqu'un document est enregistré sous forme de fichier IFC, tous les objets architecturaux de VisualARQ conservent leurs informations et peuvent ensuite être modifiés dans des plateformes logicielles d'architecture et construction. Les solides, les surfaces et les maillages de Rhino sont ouverts sous forme d'objets maillés indépendants. Les objets d'annotation, tels que le texte, les hachures ou les cotes ne sont pas exportés vers le format IFC.
