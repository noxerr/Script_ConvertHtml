Le style de vue de section définit les attributs des vues de section.


Styles de vue de section

Modifications sur le style en général
Utilisez la commande vaVueSectionStyles ou,
  • Utilisez la commande vaStylePropriétés, puis sélectionnez une vue de section et appuyez sur Entrée, ou
  • Cliquez avec le bouton de droite sur , puis sélectionnez une vue de section et appuyez sur Entrée, ou
  • Sélectionnez une vue de section et ouvrez le menu contextuel en appuyant sur le bouton de droite pendant un instant puis sélectionnez Fenêtre > Propriétés de style, ou
  • Appuyez sur Alt, Maj ou Ctrl puis double cliquez sur une vue de section.
Les modifications réalisées sur un style sont appliquées à tous les éléments de ce style.
Modification d'un objet particulier
  • Utilisez la commande vaPropriétés, puis sélectionnez une vue de section et appuyez sur Entrée, ou
  • Cliquez sur , puis sélectionnez une vue de section et appuyez sur Entrée, ou
  • Sélectionnez une vue de section et ouvrez le menu contextuel en appuyant sur le bouton de droite pendant un instant puis sélectionnez Fenêtre > Propriétés, ou,
  • Double-cliquez sur une vue de section.
Les modifications sont appliquées au style de l'élément sélectionné.

Nouveau style

Pour créer un nouveau style de vue de section, cliquez sur le bouton Nouveau style. Pour dupliquer un style de vue de section existant, sélectionnez-le dans la liste de droite et cliquez sur le bouton Dupliquer.

Attributs

Les attributs définissent les caractéristiques de chaque style de vue de section pour l'affichage et l'impression. Attributs valides :

  • Vue de section : Calque et Visibilité
  • Texte de vue de section : Style de texte, Style de cote, Couleur, Type de ligne, Largeur d'impression de ligne et Couleur d'impression de ligne.
  • Objets sélectionnés dans une vue de section : Motif, Échelle du motif, Angle du motif et Couleur du motif
