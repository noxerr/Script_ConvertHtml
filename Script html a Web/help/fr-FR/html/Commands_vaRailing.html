Lorsque vous lancez la commande vaGardeCorps, la boîte de dialogue d'insertion d'un garde-corps apparaît. Cette boîte de dialogue affiche une liste de tous les styles de garde-corps disponibles, ainsi que des options de base et un aperçu du garde-corps.

Étapes :

  1. Sélectionnez le Style de garde-corps que vous voulez utiliser et définissez les options d'insertion de base.
  2. Sélectionner autant de points que vous voulez sur toute la longueur du garde-corps.
  3. Appuyez sur ENTRÉE ou cliquez avec le bouton droit pour terminer la commande de création de garde-corps.

Vous pouvez définir autant de garde-corps que vous voulez et vous pouvez annuler la commande en appuyant sur Entrée ou Échap.

Garde-corps créé à partir d'une courbe et ses points de contrôle.

Points de contrôle

Les garde-corps peuvent avoir différents points de contrôle en fonction de la méthode utilisée pour leur création: ces points de contrôle peuvent être déplacés dans toutes les directions, afin de modifier la position et les dimensions du garde-corps.

  • Garde-corps droit : le garde-corps possède deux points de contrôle, sur les extrémités de l'objet.
  • Garde-corps à partir d'une courbe : le garde-corps possède autant de points de contrôle que la courbe à partir de laquelle il a été créé.

Options et paramètres d'insertion

Alignement

Les garde-corps peuvent être alignés selon trois règles, en fonction de la direction de la courbe de base :

  • Centre
  • Droite
  • Gauche

Pour modifier l'alignement d'un garde-corps, modifiez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).

Décalage de l'alignement

Insère le garde-corps à une certaine distance de la courbe de base.

Pour modifier le décalage d'un garde-corps, changez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).

Élévation

L'élévation d'un garde-corps est toujours mesurée par rapport à la ligne de base du niveau où il est inséré. L'élévation n'est pas une propriété de style. La valeur est définie dans la boîte de dialogue d'insertion pour chaque garde-corps inséré lors de sa création.
La valeur d'élévation par défaut pour les garde-corps est de 0. Si vous modifiez cette valeur, la boîte de dialogue d'insertion montrera la dernière valeur utilisée.

Hauteur

Même si la hauteur est une propriété du style de garde-corps, vous pouvez assigner une valeur différente dans la boîte de dialogue d'insertion. Désactivez la case Hauteur par style et choisissez la valeur désirée en utilisant le contrôle de hauteur numérique.

Cette valeur peut aussi être modifiée dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).
