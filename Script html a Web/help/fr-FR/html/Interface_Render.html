Rendu

Le gestionnaire de matériaux de VisualARQ permet d'utiliser des matériaux de tous les Moteurs de rendu compatibles avec Rhino. Le placage de matériaux de VisualARQ fonctionne comme dans Rhino.

La définition de matériaux sur les différents composants d'un objet de VisualARQ est aussi possible en modifiant le Matériau du composant dans la boîte de dialogue Propriétés de style.

Liste de produits d'animation et rendu fonctionnant avec Rhino :

  • Air Rendering : Rendu photoréaliste, illustration et interactif (module)
  • Bongo : Animation pour les dessinateurs (module)
  • Brazil pour Rhino : Outils de rendu pour les professionnels (module)
  • Flamingo nXt : Rendu par lancer de rayons et radiosité (module)
  • Hayabusa 2.0 pour Rhino : Moteur de rendu haute qualité en temps réel (programme compatible avec le format 3DM)
  • KeyShot : Application de lancer de rayons et illumination globale en temps réel (Programme compatible avec le format 3DM)
  • Rendu Maxwell, simulateur de lumière : Module de rendu Maxwell pour Rhino (module)
  • NuGraf pour les rendus CAO de grande envergure : Lancer de rayons multi-tâches et conversion 3D pour Rhino (module)
  • Penguin : Moteur de rendu non photoréaliste (module)
  • Renderosity : Forum de discussion en ligne sur Rhino (lien utile)
  • RhinoMan: Module de rendu RenderMan (module)
  • V-ray pour Rhino : Moteur de rendu d'illumination globale (module)
  • VSR Realtime Renderer 3.0 : Moteur de rendu en temps réel entièrement intégré, disponible sous forme de mode d'affichage de Rhino (module)
