Mur > Propriétés de style
Raccourci :
Sélectionner un mur, appuyer sur Alt, Maj ou Ctrl et double cliquer

Le style de mur définit les caractéristiques du mur. Les paramètres suivants peuvent être définis pour chaque style : Nom, Attributs, Géométrie et Retour.

Les styles de mur sont définis par différents Composant qui peuvent être ajoutés ou supprimés à tout moment.

  3. Composant de mur C

Nouveau style de mur :

Vous disposez de deux techniques pour créer un nouveau style de mur :

  • À partir de l'assistant de création de style : cliquez sur le bouton Nouveau style dans le Gestionnaire de styles. Utilisez l'assistant pour entrer les paramètres suivants :
    1. Nom du style
    2. Nombre de composants
    3. Nom et épaisseur de chaque composant
    4. Hauteur
  • Dupliquer un style existant : sélectionnez un style dans la liste de droite et cliquez sur le bouton Dupliquer .

Les modifications sont appliquées au style de l'élément sélectionné.

Attributs

Les attributs définissent les caractéristiques de chaque style de mur pour l'affichage et l'impression. Attributs valides :

  • Général : Calque, Visibilité et Matériau
  • Projection : Couleur, Type de ligne, Largeur d'impression de ligne et Couleur d'impression de ligne
  • Section : Couleur, Type de ligne, Largeur d'impression de ligne, Motif, Échelle du motif, Angle du motif et Couleur du motif

Remarque: Les composants de différents murs possédant le même motif de hachures, sont fusionnés dans les intersections. Les composants non consécutifs d'un même mur, possédant le même motif de hachures, sont aussi fusionnés s'ils se rencontrent sur les retours.

Composants d'un mur

Composants

Les composant des murs sont uniquement affichés dans la vue en plan.
Vous pouvez définir les paramètres suivants pour chaque composant de mur : Nom, Attributs, Géométrie, Type, Fonction et Retour. Les paramètres de Type, Fonction et Retour peuvent être modifiés dans l'onglet Construction.


Onglet Attributs pour le Composant 1 du style de mur Standard, dans la boîte de dialogue des styles de mur.

Regardez l'exemple suivant d'un style de mur à 5 composants avec un retour extérieur au niveau des extrémités du mur :

  1. Composant intérieur 1 (Stuc)
  2. Composant 2 (Brique)
  3. Composant 3 (vide d'air)
  4. Composant 4 (isolation)
  5. Composant extérieur 5 (brique)
  6. Composant 4 (retour)
  7. Composant 5 (retour)

Ajouter de nouveaux composants : Vous pouvez ajouter de nouveaux composants à un style de mur en sélectionnant le style, en cliquant avec le bouton droit pour ouvrir le menu contextuel et en sélectionnant Nouveau > Composant de mur.

Supprimer des composants : sélectionnez le composant que vous voulez supprimer, cliquez avec le bouton droit pour ouvrir le menu contextuel et sélectionnez Supprimer. Vous pouvez aussi supprimer un style de mur ou un composant de mur avec l'icône dans la barre d'outils. Il n'est pas possible de supprimer un style de mur ou un composant utilisé par un mur existant du modèle.

Chaque composant reçoit un Nom et une Épaisseur spécifiques.

L'Épaisseur totale du mur correspond à la somme des épaisseurs de ses composants.
La hauteur du mur est définie comme une unité qui reste constante pour tous les composants.

Type

Il existe deux types de composants de mur qui définissent l'ordre de connexion entre eux :

  • Noyau : composants de priorité 1. Les composants de Noyau sont des éléments de structure du mur, ils seront normalement connectés les uns aux autres.
  • Normal : composants de priorité 2. Ils sont normalement liés à d'autres composants normaux une fois que les composants de noyau sont connectés. Ils peuvent aussi être placés autour des composants de noyau, suivant leurs paramèters de retour.

Fonction

Les composants de mur peuvent avoir 5 types de fonction. Ces fonctions ne servent que d'informations.

  • Aucun
  • Structure
  • Isolation
  • Finition
  • Décoration

Retour

Le retour du mur est défini par la façon dont les composants du mur se terminent aux extrémités fermées d'un mur et au niveau des ouvertures. Le retour concerne uniquement les murs composés d'au moins 3 composants et il est visible dans la fenêtre Dessus lorsque le mode de vue en plan est activé.

Il existe 3 types de retour pour les extrémités et pour les ouvertures :

  1. Aucun : Mur sans retour (au niveau des ouvertures et des extrémités fermées).
  2. Extérieur : Les composants extérieurs du mur forment un retour au niveau des extrémités fermées et/ou des ouvertures.
  3. Extérieur : Les composants intérieurs du mur forment un retour au niveau des extrémités fermées et/ou des ouvertures.


1. Retour d'un mur : Aucun.

2. Retour d'un mur : Extérieur.

3. Retour d'un mur : Intérieur.

Retour des composants d'un mur : Quatre options permettent de définir le retour des composants d'un mur :

  • Aucun : Aucun retour n'est créé.
  • Extrémités : le composant effectue un retour aux extrémités du mur.
  • Ouvertures : le composant effectue un retour au point de rencontre avec une porte, une fenêtre ou une ouverture.
  • Extrémités et ouvertures : le composant effectue un retour aux extrémités d'un mur et au point de rencontre avec une porte, une fenêtre ou une ouverture.

Remarque: Les composants de mur de type Noyau ne peuvent pas avoir de retour.

Joints de mur

Trois types de joints sont possibles entre les murs :

  • Aucun
  • Onglet
  • About

1. Joint de mur : Aucun.

2. Joint de mur : Onglet.

3. Joint de mur : About.

Édition des joints entre murs :

Les joints entre murs peuvent être modifiés dans les propriétés objet de VisualARQ, dans le panneau Propriétés de Rhino , après avoir sélectionné tous les murs impliqués dans l'intersection. Lorsque des murs se croisent, ils appliquent un joint automatique qui dépend du style du mur. Si les murs ont le même style, le joint par défaut utilisé est Onglet. Si les murs sont de styles différents, le joint par défaut utilisé est About.

Lorsque trois murs, ou plus, forment une intersection, le joint dépend de l'ordre dans lequel les murs ont été ajoutés au niveau du point d'intersection. L'option Ordre de tri de la section Géométrie permet de changer l'ordre jusqu'à ce que vous trouviez la solution qui convient le mieux.

Lorsque deux composants de mur se rejoignent, si leurs motifs de hachures sont différents, une ligne apparaît au niveau du joint. Si les deux composants ont le même motif de hachures, aucun ligne n'apparaît au niveau du joint.
