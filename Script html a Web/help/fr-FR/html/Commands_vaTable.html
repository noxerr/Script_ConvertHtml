Nomenclature

Clavier :
vaNomenclature
Icône :
clic [clic droit]
Barre d'outils Documentation :
Menu :
VisualARQ > Nomenclature > Nomenclature

La nomenclature est un tableau qui affiche des informations concernant des objets d'un même type.

Lorsque vous lancez la commande vaNomenclature, la boîte d'insertion apparaît. Elle affiche les styles de nomenclatures disponibles. Par défaut, la liste comprend les styles de nomenclature d'Ouvertures, d'Espaces et de Murs, mais de nouveaux styles de nomenclature peuvent être créés pour quantifier tous les objets de VisualARQ.

Étapes :

  1. Sélectionnez le style de nomenclature que vous voulez utiliser.
  2. Sélectionnez tous les objets que vous voulez quantifier dans la nomenclature. La commande filtre la sélection automatiquement et détecte le type d'objet correspondant au style de nomenclature sélectionné.
  3. Choisissez un point d'insertion en cliquant à l'écran pour insérer la nomenclature.

La commande se termine lorsque la nomenclature quantifiant les objets sélectionnés est insérée.


Boîte de dialogue d'insertion d'une nomenclature

Points de contrôle

Les nomenclatures ont un seul point de contrôle au sommet supérieur gauche.

Options et paramètres d'insertion

Les boîtes de dialogue d'insertion de nomenclatures n'ont pas de paramètres d'insertion. Elles permettent uniquement de sélectionner le style de tableau à insérer.

Les paramètres qui définissent les caractéristiques d'une nomenclature sont tirés du style de nomenclature.

Exporter une nomenclature vers Excel

Une fois insérées dans le modèle, les nomenclatures peuvent être exportées vers Excel et au format. CSV avec la commande vaExporterNomenclature.
