Sélectionner un espace, appuyer sur Alt, Maj ou Ctrl et double cliquer

Le style d'espace définit les caractéristiques des espaces/pièces. Vous pouvez définir les composants suivants pour chaque style d'espace : Attributs, Général, Nom, Surface et Périmètre.

Pour créer un style d'espace, cliquez sur le bouton Nouveau style... dans le gestionnaire de styles. Sélectionnez le style de zone que vous voulez définir et utilisez les onglets du panneau de droite pour choisir les paramètres suivants :


Boîte de dialoge du gestionnaire de styles d'espace, onglet Général.

Modifications sur le style en général
Utilisez la commande vaEspaceStyles ou,
  • Utilisez la commande vaStylePropriétés, puis sélectionnez un espace et appuyez sur Entrée, ou
  • Cliquez avec le bouton de droite sur , puis sélectionnez un espace et appuyez sur Entrée, ou
  • Sélectionnez un espace et ouvrez le menu contextuel en appuyant sur le bouton de droite pendant un instant puis sélectionnez Fenêtre > Propriétés de style, ou
  • Appuyez sur Alt, Maj ou Ctrl puis double cliquez sur un espace.
Les modifications réalisées sur un style sont appliquées à tous les éléments de ce style.
Modification d'un objet particulier
  • Utilisez la commande vaPropriétés, puis sélectionnez un espace et appuyez sur Entrée, ou
  • Cliquez sur , puis sélectionnez un espace et appuyez sur Entrée, ou
  • Sélectionnez un espace et ouvrez le menu contextuel en appuyant sur le bouton de droite pendant un instant puis sélectionnez Fenêtre > Propriétés, ou,
  • Double-cliquez sur un espace.
Les modifications sont appliquées au style de l'élément sélectionné.

Attributs

Les attributs définissent les caractéristiques de chaque style d'espace pour l'affichage et l'impression. Attributs valides :

  • Calque, Visibilité, Style de texte, Style de cote, Couleur, Type de ligne, Largeur d'impression de ligne, Couleur d'impression de ligne, Motif, Échelle du motif, Angle du motif et Couleur du motif.

Général

Options pour définir comment les informations sont affichées dans la légende de l'espace :

  • Montrer la bordure
  • Afficher le nom
  • Afficher la zone
  • Afficher le périmètre

Nom

Options pour définir comment le nom de la pièce est affiché dans la légende de l'espace :

  • Préfixe : texte affiché avant le nom
  • Suffixe : texte affiché après le nom

Aire

Options pour définir comment la surface de la pièce est affichée dans la légende de l'espace :

  • Préfixe : texte affiché avant la surface
  • Suffixe : texte affiché après la surface

Périmètre

Options pour définir comment le périmètre de la pièce est affiché dans la légende de l'espace :

  • Préfixe : texte affiché avant le périmètre
  • Suffixe : texte affiché après le périmètre
