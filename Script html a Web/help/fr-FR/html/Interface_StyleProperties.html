La boîte de dialogue des Propriétés de style (appelée également Gestionnaire de styles) permet de modifier les propriétés de style d'un objet et de créer de nouveaux styles pour cet objet. Les commandes suivantes ouvrent le gestionnaire de styles pour le type d'objet correspondant :

Objets architecturaux :

  • Mur : vaMurStyles
  • Mur-rideau : vaMurRideauStyles
  • Poutre : vaPoutreStyles
  • Poteau : vaPoteauStyles
  • Porte : vaPorteStyles
  • Fenêtre : vaFenêtreStyles
  • Ouverture : vaOuvertureStyles
  • Escalier : vaEscalierStyles
  • Garde-corps : vaGardeCorpsStyles
  • Dalle : (Les dalles n'ont pas de styles)
  • Toit : (Les toits n'ont pas de styles)

Objets de documentation :

  • Espace : vaEspaceStyles
  • Nomenclature : vaNomenclatureStyles
  • Étiquette: vaÉtiquetteStyles
  • Ligne de coupe de section : vaSectionStyles
  • Vue de section : vaVueSectionStyles
  • Vue en plan : vaVueEnPlanStyles

Lorsque vous lancez la commande vaStylePropriétés, la boîte de dialogue Propriétés du style apparaît.
Cette boîte de dialogue permet de modifier les propriétés de style des objets sélectionnés.

Comment s'organise cette boîte de dialogue ?

La boîte de dialogue des propriétés de style se divise en trois zones :


Boîte de dialogue Propriétés du style, propriétés de style d'une fenêtre.

  1. Le panneau supérieur gauche affiche la liste des styles pour les objets qui étaient sélectionnés lorsque la commande a été lancée.
  2. Le panneau inférieur gauche affiche l'aperçu du style d'objet que vous avez choisi dans la liste. Vous pouvez définir les options d'affichage suivantes pour cette fenêtre d'aperçu dans la barre d'outils située en haut de la boîte de dialogue :
    • Option pour afficher ou masquer la fenêtre d'aperçu.
    • Option pour afficher le style d'objet dans la vue en plan ou en perspective.
    L'aperçu affiché peut être modifié avec la souris tout comme dans les vues de Rhino :
    • Bouton droit de la souris : permet de faire tourner la vue pour afficher le modèle selon différents points de vue.
    • Ctrl + bouton droit : permet de zoomer pour agrandir ou réduire le champ de vision.
    • Maj+ bouton droit : permet de déplacer le point de vue parallèlement au plan de vision, vers le haut, le bas, la droite ou la gauche.
  3. Cette zone montre différentes informations en fonction de ce qui est sélectionné dans le panneau supérieur gauche :
    • Lorsque le dossier racine de styles d'objet est sélectionné, cette zone affiche la liste des styles existants. En bas de la boîte de dialogue apparaissent des options permettant de créer un Nouveau style, de Dupliquer ou de Supprimer un style dans la liste supérieure.
    • Lorsq'un style ou un composant spécifique est sélectionné, cette zone affiche les différents onglets et propriétés permettant de modifier chaque paramètre.

Créer un nouveau style d'objet

Pour créer un nouveau style, sélectionnez le dossier des styles dans le panneau supérieur gauche afin d'afficher la liste des styles dans le panneau de droite. Deux options sont disponibles pour créer un nouveau style :

  1. Utiliser l'assistant de création de style : appuyez sur le bouton Nouveau style.
  2. Dupliquer un style existant : sélectionnez dans la liste supérieure le style que vous souhaitez dupliquer et cliquez sur le bouton Dupliquer.


Gestionnaire de styles pour les garde-corps.

Dans l'option 1, un assistant vous guide par des étapes simples en vous demandant les paramètres du style d'objet défini. Les objets de VisualARQ n'utilisent pas tous l'assistant pour créer de nouveaux styles et certains paramètres ne peuvent être définis qu'à la fin de l'assistant (comme par exemples, les attributs de style, les petits-bois des vitres pour les portes et les fenêtres, l'épaisseur de la dalle des escaliers, etc.). Les types d'objet qui n'utilisent pas l'assistant ajoutent simplement un nouveau style à la fin de la liste après avoir cliqué sur le bouton Nouveau style.


Assistant de style de porte.

Modifier des styles d'objets

Pour modifier un style d'objet, sélectionnez le style ou le composant de style dans la liste et modifiez ses propriétés dans les onglets situés à droite. Ces onglets et propriétés peuvent être différents en fonction du type d'objet ou du composant.

Onglets

Attributs

Cet onglet est affiché pour tous les objets qui utilisent une boîte de dialogue. Il permet de définir les fonctions (attributs) de chaque style d'objets et de chacun de ses composants. Les attributs varient en fonction du style d'objet auquel ils sont appliqués.

  • Objet architectural : Murs, Murs-rideaux, Poutres, Poteaux, Portes, Fenêtres, Ouvertures, Escaliers et Garde-corps :
    • Attributs généraux :
      • Calque
      • Visibilité
      • Matériau
    • Attributs de section et de projection :
      • Couleur
      • Type de ligne
      • Largeur d'impression de ligne
      • Couleur d'impression de ligne
      • Motif
      • Échelle du motif
      • Angle du motif
      • Couleur du motif
  • Objet de documentation : Espaces, Nomenclatures, Étiquettes, Sections, Vues de section et Vues en plan :
    • Attributs généraux :
      • Calque
      • Visibilité
      • Style de texte
      • Style de cote
      • Couleur
      • Type de ligne
      • Largeur d'impression de ligne
      • Couleur d'impression de ligne
      • Motif
      • Échelle du motif
      • Angle du motif
      • Couleur du motif

REMARQUE : À partir de l'attribut Matériau du style d'objet, vous pouvez assigner un matériau différent à chaque composant de l'objet. Pour assigner un seul matériau à tout l'objet, suivez la procédure habituelle de Rhino.



Onglet Attributs dans la boîte de dialogue des styles de mur.


Géométrie

Tous les objets, à l'exception des escaliers, des dalles et des toits, possèdent cet onglet. Les valeurs géométriques pour chaque objet et chaque composant sont définies dans cet onglet.
Valeurs géométriques pour les murs :
  • Hauteur
  • Épaisseur (ne peut être modifiée que pour chaque composant du mur)
Valeurs géométriques pour les poteaux et les poutres :
  • Hauteur
  • Profil (définit la forme de l'objet)
  • Épaisseur du revêtement (uniquement pour les poteaux)
Valeurs géométriques pour les portes, les fenêtres et les ouvertures :
  • Profil (définit la forme de l'objet)
  • Tailles (largeur et profondeur) du dormant et de la cale
  • Tailles (largeur et profondeur) et côté d'ouverture du battant
  • Tailles (épaisseur, prolongement et profondeur) du rebord (uniquement pour les fenêtres et les ouvertures)
Valeurs géométriques pour les garde-corps :
  • Composant de lisse : marges finales et style. Joints plans et non plans.
  • Barreaux : marges supérieure et inférieure.

Tailles

Cet onglet ne sert qu'aux objets qui utilisent des profils pour définir leur forme (poteaux, portes, fenêtres et ouvertures). Il permet de définir les tailles (largeur et auteur) du profil utilisé par l'objet. Cette option permet de créer différentes tailles pour le même profil.

Blocs

Seules les portes, les fenêtres et les ouvertures possèdent cet onglet. L'onglet définit les représentations 2D (plan) et 3D (modèle) de l'objet, en assignant un bloc déjà créé par l'utilisateur, avec des paramètres personnalisés pour chaque représentation. Cette option permet de créer des portes et des fenêtres personnalisées.

Construction et Retour

Seuls les murs présentent cet onglet. Cet onglet est utilisé pour définir la façon dont les composants du mur se terminent aux extrémités fermées d'un mur et au niveau des ouvertures. L'onglet Construction est utilisé pour définir les types de composants du mur et leurs fonctions.

Type

Seuls les escaliers présentent cet onglet. Il permet de définir le type d'escalier utilisé, suivant si l'option Marches balancées est activée ou non.

Marches

Cet onglet n'apparaît que pour les escaliers, il permet de définir les valeurs géométriques des dalles de contremarche et de giron.

  • Valeurs géométriques pour la Dalle de giron :
    • Épaisseur de la dalle
    • Nez
  • Valeurs géométriques pour la Dalle de contremarche :
    • Épaisseur de la dalle

Division et Remplissage

Seuls les murs-rideaux présentent ces onglets. Ils sont utilisés pour définir les paramètres du composant de Cellule
