Importation de styles

Clavier :
vaImporterStyles
Menu :
VisualARQ > Styles > Importer

La commande vaImporterStyles permet d'importer des styles de VisualARQ créés dans un autre document, afin qu'ils soient disponibles dans la bibliothèque de styles du fichier .3dm actuel.
Les fichiers importés ont l'extension .val (bibliothèque de VisualARQ) et ils sont obtenus par exportation des styles d'un autre fichier.

Lorsque vous lancez la commande, vaImporterStyles, la boîte de dialogue Importer des styles apparaît.

Étapes :

  1. Ouvrez le fichier .val qui contient les styles que vous voulez importer dans le fichier .3dm ouvert. Si le chemin du fichier n'est pas affiché dans les menus déroulant, cliquez sur le bouton Parcourir pour le chercher dans votre système. Cliquez sur le bouton Suivant > pour continuer.
  2. Dans la liste, sélectionnez tous les styles que vous voulez importer et appuyez sur le bouton Importer pour terminer l'opération.

Étape 1. Sélectionnez le fichier .VAL

Étape 2. Sélectionnez les styles que vous voulez importer

Chaque style d'objet importé sera ouvert dans une boîte de dialogue Style.
