Lorsque des modifications ne sont pas automatiquement affichées dans le modèle, sélectionnez les objets concernés et cliquez sur l'icône Actualiser.

REMARQUE : Le processus d'actualisation automatique ralentit les performances du logiciel. C'est pourquoi uniquement certaines commandes offrent cette option.
