Options de VisualARQ (dans la boîte de dialogue Options de Rhino)

Pour ouvrir la boîte de dialogue Options de Rhino, cliquez sur le menu Outils et choisissez Options...
La boîte de dialogue Options de Rhino s'ouvrira et affichera un arbre dans la section de gauche avec les paramètres disponibles pour les Propriétés du document et les Options de Rhino.

Chaque branche affiche une section avec les paramètres de VisualARQ disponibles :

  • Options de Rhino > Propriétés du document > VisualARQ > Calques


    Paramètres des calques, dans les Options de VisualARQ

  • Options de Rhino > VisualARQ > Général (Langue, Messages et Affichage), Icône de l'échelle et Mises à jour


    Icône de l'échelle dans les Options de VisualARQ
