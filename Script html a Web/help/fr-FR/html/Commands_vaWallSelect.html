Mur : Sélectionner

Clavier :
vaMurSélectionner
Menu :
VisualARQ > Mur > Sélectionner.
  • Murs adjacents
  • Portes
  • Fenêtres

Cette commande permet de sélectionner les murs ainsi que les ouvertures qu'ils contiennent de différentes façons.

Lorsque vous lancez la commande vaMurSélectionner, suivez les étapes de la ligne de commandes.

  • Tous : Tous les murs du modèle sont sélectionnés
  • Murs adjacents : Choisissez un mur du modèle afin de sélectionner tous les murs adjacents.
  • Portes : Choisissez un mur du modèle afin de sélectionner toutes les portes contenues dans ce mur.
  • Fenêtres : Choisissez un mur du modèle afin de sélectionner toutes les fenêtres contenues dans ce mur.
