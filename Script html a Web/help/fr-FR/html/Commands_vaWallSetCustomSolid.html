Mur : Définir un solide personnalisé

Clavier :
vaMurDéfinirSolidePersonnalisé
Icône :
clic droit [clic]
Barre d'outils Murs :
Menu :
VisualARQ > Mur > Définir un solide personnalisé

Option pour transformer un mur à partir d'un objet solide modélisé dans Rhino.

Lorsque vous lancez la commande vaMurDéfinirSolidePersonnalisé, suivez les étapes de la ligne de commandes. Appuyez sur le bouton de droite ou Entrée pour accepter et passer à l'étape suivante.

Étapes :

  1. Sélectionnez le mur que vous voulez adapter à l'objet de Rhino.
  2. Sélectionnez l'objet de Rhino qui définit le nouveau mur.

REMARQUE : N'oubliez pas que la courbe de base du nouveau mur sera celle du mur original. Cette courbe définit l'alignement des ouvertures à l'intérieur du mur.
