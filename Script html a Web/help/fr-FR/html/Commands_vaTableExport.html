Exporter une nomenclature

Clavier :
vaExporterNomenclature
Menu :
VisualARQ > Nomenclature > Exporter
Menu contextuel :
Nomenclature > Exporter

Une fois insérées dans le modèle, les nomenclatures peuvent être exportées vers Excel (.xls) et au format. CSV.

Étapes :

  1. Sélectionnez la nomenclature que vous voulez exporter.
  2. Choisissez le dossier ainsi que le format de fichier dans lequel vous voulez enregistrer votre nomenclature ( .xls ou .csv).

Un nouveau fichier est créé dans le dossier de destination sélectionné. Ce fichier n'est pas actualisé automatiquement lorsque des modifications sont appliquées à la nomenclature de VisualARQ dans Rhino. Si vous voulez la dernière version actualisée, répétez les étapes précédentes et remplacez l'ancien fichier.
