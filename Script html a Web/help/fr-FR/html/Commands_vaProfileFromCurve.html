Profil : À partir d'une courbe

Clavier :
vaProfilÀPartirDeCourbe
Icône :
clic
Barre d'outils : Outils >
Menu :
VisualARQ > Profil > À partir d'une courbe

La commande vaProfilÀPartirDeCourbe peut être utilisée pour créer des profils personnalisés à partir de courbes planes et fermées. Ces profils peuvent être utilisés pour créer des styles de poteaux, de poutres, de portes, de fenêtres et d'ouvertures personnalisés, ainsi que des formes de composants de garde-corps.

Étapes :

  1. Sélectionnez une courbe plane et fermée pour définir la forme du profil.
  2. Tapez le nom du profil dans la ligne de commandes

Lorsque le profil personnalisé a été créé, il apparaît à la fin de la liste dans la bibliothèque des profils prédéfinis pour les poteaux, les poutres, les portes et les fenêtres.
Le profil obtenu n'est pas paramétrique, il ne peut donc être utilisé qu'avec les cotes et les proportions du dessin original.
