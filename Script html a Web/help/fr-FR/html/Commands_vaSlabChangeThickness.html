Dalle : Changer l'épaisseur

Clavier :
vaDalleChangerÉpaisseur
Icône :
clic
Barre d'outils Dalles :
Menu :
VisualARQ > Dalle > Changer l'épaisseur

Option pour changer l'épaisseur d'une dalle.

Lorsque vous lancez la commande vaDalleChangerÉpaisseur, suivez les étapes de la ligne de commandes. Appuyez sur le bouton de droite ou Entrée pour accepter et passer à l'étape suivante.

Étapes :

  1. Sélectionnez la dalle dont vous voulez changer l'épaisseur.
  2. Entrer la valeur d'épaisseur désirée dans la ligne de commandes.

L'épaisseur de la dalle sera modifiée tout en conservant la surface supérieure à la même élévation.
