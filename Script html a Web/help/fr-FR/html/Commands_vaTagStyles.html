Styles d'étiquette

Clavier :
vaÉtiquetteStyles
Icône :
clic droit [clic gauche]
Barre d'outils Documentation :
Menu :
VisualARQ > Étiquette > Styles d'étiquette
Raccourci :
Sélectionner une étiquette, appuyer sur Alt, Maj ou Ctrl et double cliquer

Le style définit les attributs des étiquettes.


Boîte de dialogue des Styles d'étiquette

Modifications sur le style en général
Utilisez la commande vaÉtiquetteStyles ou,
  • Utilisez la commande vaStylePropriétés, puis sélectionnez une étiquette et appuyez sur Entrée, ou
  • Cliquez avec le bouton de droite sur , puis sélectionnez une étiquette et appuyez sur Entrée, ou
  • Sélectionnez une étiquette et ouvrez le menu contextuel en appuyant sur le bouton de droite pendant un instant puis sélectionnez Fenêtre > Propriétés de style, ou
  • Appuyez sur Alt, Maj ou Ctrl puis double cliquez sur une étiquette .
Les modifications réalisées sur un style sont appliquées à tous les éléments de ce style.
Modification d'un objet particulier
  • Utilisez la commande vaPropriétés, puis sélectionnez une étiquette et appuyez sur Entrée, ou
  • Cliquez sur , puis sélectionnez une étiquette et appuyez sur Entrée, ou
  • Sélectionnez une étiquette et ouvrez le menu contextuel en appuyant sur le bouton de droite pendant un instant puis sélectionnez Fenêtre > Propriétés, ou,
  • Double-cliquez sur une étiquette.
Les modifications sont appliquées au style de l'élément sélectionné.

Nouveau style

Pour créer un nouveau style d'étiquette, cliquez sur le bouton Nouveau style. Pour dupliquer un style d'étiquette existant, sélectionnez-le dans la liste de droite et cliquez sur le bouton Dupliquer.

Attributs

Les attributs définissent les caractéristiques de chaque style d'étiquette pour l'affichage et l'impression. Attributs valides :

  • Calque, Visibilité, Style de texte, Style de cote, Couleur, Type de ligne, Largeur d'impression de ligne et Couleur d'impression de ligne.
