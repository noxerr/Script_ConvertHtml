Toit : Ajouter une frontière

Clavier :
vaToitAjouter
Icône :
clic
Barre d'outils Toits :
Menu :
VisualARQ > Toit > Ajouter une frontière

Option pour ajouter une frontière à un toit.

Lorsque vous lancez la commande vaToitAjouter, suivez les étapes de la ligne de commandes. Appuyez sur le bouton de droite ou Entrée pour accepter et passer à l'étape suivante.

Étapes :

  1. Sélectionnez le toit auquel vous voulez ajouter une frontière.
  2. Sélectionnez la courbe plane et fermée que vous voulez ajouter comme frontière.

La silhouette de la nouvelle frontière est ajoutée à la forme originale du toit.

REMARQUE : Les frontières ne doivent pas obligatoirement se trouver sur le plan du toit mais elles doivent être fermées et planes.
