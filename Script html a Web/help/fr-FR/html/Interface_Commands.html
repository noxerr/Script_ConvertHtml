Commandes au clavier

En plus des barres d'outils et des menus (déroulant et contextuels), les commandes de VisualARQ peuvent être lancées en tapant leur nom dans la ligne de commandes.

Toutes les commandes de VisualARQ commencent avec le préfixe va. Si vous tapez "va" dans la ligne de commandes, une liste de toutes les commandes de VisualARQ apparaît.

Pour exécuter une commande à partir de la ligne de commandes, écrivez le nom de la commande. Le nom sera complété automatiquement au fur et à mesure.
Sélectionnez finalement la commande désirée dans la liste.

Comme pour les commandes de Rhino, la ligne de commandes affiche aussi les invites et les options des commandes de VisualARQ entre parenthèses :

  • Les invites sont les messages qui guident les utilisateurs dans une commande, en indiquant l'action à réaliser ou les informations à fournir.
  • Les options des commandes changent selon les fonctions de celles-ci. Elles offrent des méthodes alternatives d'utilisation des commandes.

Pour activer une option, cliquez sur l'option ou bien tapez le nom de l'option ou la lettre de raccourci (soulignée) dans la ligne de commande.

Pour terminer la commande appuyez sur ENTRÉE.
