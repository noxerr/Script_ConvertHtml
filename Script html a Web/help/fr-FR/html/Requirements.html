Configuration requise

Logiciel

VisualARQ fonctionne avec Rhinoceros 4.0 SR7 et Rhinoceros 5 SR6 (32 ou 64 bits).

Matériel

La configuration minimale du système est la même que celle de Rhino 4.0 et Rhino 5.

Installation

VisualARQ possède un assistant d'installation qui permet d'automatiser le processus.

Une clé de licence est nécessaire pour installer et utiliser le logiciel. Plusieurs ordinateurs d'un réseau ne peuvent pas utiliser simultanément la même licence.

L'installation est réalisée par défaut dans : C:\Program Files\AsuniCAD\VisualARQ "1.X".

Une fois le programme installé, lorsque l'utilisateur double clique sur l'icône d'accès direct, une boîte de dialogue permet de définir les options des modèles à utiliser. Sélectionnez un des quatre modèles affichés avant de commencer à travailler avec VisualARQ :

  • VisualARQ - Mètres.3dm
  • VisualARQ - Milimètres.3dm
  • VisualARQ - Pieds.3dm
  • VisualARQ - Pouces.3dm


Liste de modèles de Rhino et VisualARQ.

Dans Rhino 5 (32 et 64 bits), les fichiers modèles de VisualARQ sont enregistrés dans les répertoires suivants : C:\Users\"Nom d'utilisateur"\AppData\Roaming\McNeel\Rhinoceros\5.0\Localization\fr-fr\Template Files. (Ce repertoire correspond au dossier des fichiers modèles en français. Remplacez "fr-fr" par un autre code de langue pour trouver les autres répertoires où sont stockés les fichiers modèles de VisualARQ dans d'autres langues).
