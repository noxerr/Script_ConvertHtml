Le fonctionnement et le comportement d'une fenêtre sont identiques à ceux d'une porte.
Les fenêtres ne peuvent être insérées que dans un mur. Les deux éléments sont liés et l'ouverture est générée automatiquement dans le mur où elle est insérée.

Lorsque vous lancez la commande vaFenêtre, la boîte de dialogue d'insertion d'une fenêtre apparaît. Cette boîte de dialogue affiche une liste de tous les styles de fenêtres disponibles, des options de base pour définir la fenêtre et un aperçu.

Les étapes nécessaires à l'insertion d'une fenêtre dans un mur dépendront des options d'insertion sélectionnées.

Étapes :

  1. Sélectionnez le Style de fenêtre que vous voulez utiliser et définissez les options d'insertion de base.
  2. Il existe deux options pour permettre l'insertion d'une fenêtre dans un mur :
    • Décalage : insère la fenêtre à une distance spécifique (décalage) de l'extrémité la plus proche du mur où la fenêtre est insérée. Indiquez le côté d'ouverture en déplaçant le curseur ; la commande le définit automatiquement selon la position du curseur. Cliquez pour insérer la fenêtre.
    • Point d'insertion : insère librement la fenêtre dans le mur. Cliquez pour insérer la fenêtre.
  3. Indiquez le côté d'ouverture en cliquant sur le côté désiré du mur. Cette étape n'est pas nécessaire lorsque vous insérez une fenêtre avec l'option décalage.

Vous pouvez définir autant de fenêtres que vous voulez et vous pouvez annuler la commande en appuyant sur Entrée ou Échap.

Avec les deux options d'insertion, l'ouverture correspondant à la fenêtre est générée automatiquement dans le mur.


Boîte de dialogue d'insertion d'une fenêtre

Fenêtre à partir d'une courbe

Les fenêtres peuvent être créées à partir de courbes, qui déterminent leur forme.

Étapes :

  1. Lancez la commande vaFenêtre.
  2. Sélectionnez le Style de fenêtre que vous voulez utiliser et définissez les options d'insertion de base
  3. Sélectionnez l'option ÀPartirDeCourbes dans la ligne de commandes
  4. Sélectionnez une ou plusieurs courbes dans le modèle. Cette courbe doit être fermée et verticale

La fenêtre est générée à l'intérieur du mur le plus proche de la courbe sélectionnée.

Remarque : Les courbes sélectionnées pour créer des fenêtres ne doivent pas obligatoirement être orientées parallèlement au mur où les fenêtres seront placées. Les fenêtres sont insérées et orientées automatiquement en fonction de la position du mur le plus proche.

Points de contrôle

Les fenêtres ont cinq points de contrôle :

  1. Point de contrôle pour le côté d'ouverture
  2. Point d'insertion gauche
  3. Point d'insertion central
  4. Point d'insertion droit
  5. Point de contrôle de la marque de référence

Pour déplacer la fenêtre (dans le mur ou sur un autre mur), sélectionnez-la à partir d'un des points et faites-la glisser où vous voulez.

Pour changer le côté d'ouverture de la fenêtre, sélectionnez le point de contrôle du côté d'ouverture (1) et déplacez-le de l'autre côté du mur.

Options et paramètres d'insertion

Alignement

La fenêtre peut être alignée avec l'épaisseur du mur selon 3 méthodes :

  • Intérieur
  • Centre
  • Extérieur

Pour modifier l'alignement d'une fenêtre, modifiez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino).

Pourcentage d'ouverture

Le pourcentage d'ouverture indique le degré d'ouverture des battants dans les vues en plan et en perspective. Le pourcentage d'ouverture est défini par deux options :

  • Ouverture maximale (%): cette valeur définit l'ouverture maximale que les battants de la fenêtre peuvent atteindre. L'ouverture maximale de la fenêtre est affichée uniquement dans la vue de dessus lorsque le mode de vue en plan de VisualARQ est activé, et elle est représentée par une simple ligne et un arc.
  • Ouverture (%) : cette valeur indique le degré d'ouverture des battants dans les vues en perspective et en plan.

Exemple d'ouverture d'une fenêtre: Ouverture maximale : 65 %, Ouverture : 0%

Le pourcentage d'ouverture n'est pas une propriété de style. La valeur est définie dans la boîte de dialogue d'insertion pour chaque fenêtre insérée.

La valeur d'ouverture par défaut pour les fenêtres est de 50 % mais vous pouvez utiliser des valeurs situées entre 0 et 100 %. Si vous modifiez cette valeur, la boîte de dialogue d'insertion montrera la dernière valeur utilisée.

Pour modifier le pourcentage d'ouverture d'une fenêtre, changez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino).


Alignement vertical

Deux alignements verticaux par rapport à la ligne de base du mur peuvent être utilisés lors de l'insertion des fenêtres :

  • Inférieur : distance de référence entre la ligne de base du mur et la surface inférieure de la fenêtre.
  • Supérieur : distance de référence entre la ligne de base du mur et la surface supérieure de la fenêtre.

L'alignement vertical n'est pas une propriété de style. La valeur est définie dans la boîte de dialogue d'insertion pour chaque objet inséré.


Pour modifier l'alignement vertical d'une fenêtre, changez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino).

Élévation

L'élévation d'une fenêtre dans la boîte de dialogue d'insertion est toujours mesurée par rapport à la ligne de base du mur où elle est insérée, quel que soit son niveau. Lorsque la fenêtre a été insérée, la valeur d'élévation indique la position de la fenêtre dans les coordonnées Z du repère général, quels que soient le mur et le niveau d'insertion. L'élévation n'est pas une propriété de style. La valeur est définie dans la boîte de dialogue d'insertion pour chaque objet inséré.
La valeur d'élévation par défaut pour les fenêtres est de 0. Si vous modifiez cette valeur, la boîte de dialogue d'insertion montrera la dernière valeur utilisée.

Pour modifier l'élévation d'une fenêtre, changez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino).
