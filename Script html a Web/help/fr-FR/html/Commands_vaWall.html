Les murs sont des éléments de base de VisualARQ.

Lorsque vous lancez la commande vaMur, la boîte de dialogue d'insertion d'un mur apparaît. Cette boîte de dialogue affiche une liste de tous les styles de murs disponibles, des options de base pour définir le mur et un aperçu.

Étapes :

  1. Sélectionnez le Style de mur que vous voulez utiliser et définissez les options d'insertion de base.
  2. Indiquez le point de départ du mur (point d'insertion) en cliquant. Après avoir défini le point de départ, une cote dynamique affichera en temps réel la longueur du mur inséré.
  3. Cliquez à nouveau pour définir le point final du mur.

La longueur et la direction du mur peuvent être définies :

  • en entrant les coordonnées (relatives et absolues) dans la ligne de commande.
  • en entrant une valeur pour définir la longueur ou l'angle dans la ligne de commande.
  • Manuellement, en cliquant dans le modèle.

Chaque nouveau mur est dessiné à côté du précédent jusqu'à ce que vous appuyiez sur la touche Entrée ou Échap ou que vous cliquiez avec le bouton de droite. La commande demande immédiatement le point de départ pour commencer à dessiner le mur suivant ; ce nouveau mur n'est pas lié au précédent mais est dessiné à partir du point sélectionné.

VisualARQ > Mur > À partir de courbes

Options pour créer un mur à partir de courbes déjà dessinées, qui définiront sa frontière.

Lorsque vous lancez la commande vaMur > ÀPartirCourbes, la boîte d'insertion apparaît. Suivez les invites dans la ligne de commande. Cliquez avec le bouton de droite en dehors de la boîte de dialogue d'insertion du mur ou appuyez sur Entrée pour terminer la commande.

Étapes :

  1. Sélectionnez une Courbe ou plusieurs Courbes qui seront utilisés pour générer des murs. Le type et la forme des courbes ne sont pas importants mais elles doivent être planes.
  2. Sélectionnez les options du mur dans la boîte de dialogue d'insertion afin de définir le mur créé à partir de la courbe. Cette étape peut être réalisée avant ou après avoir sélectionné les courbes permettant de générer les murs.

Les murs créés de cette façon se comportent comme tous les autres murs de VisualARQ. Ils peuvent être modifiés (par objet ou style) et vous pouvez insérer des portes et des fenêtres dedans.

Les murs créés avec des courbes sont paramétriques et le tracé peut aussi être modifié manuellement. Pour ce faire, VisualARQ dispose de deux options :

  • Déformer la trajectoire du mur en utilisant ses propres points de contrôle.
  • Déformer le mur avec la commande vaMurDéfinirChemin.


Points de contrôle

Les murs disposent de trois points de contrôle et de trois flèches de contrôle :

  • Les points de contrôle de départ et final peuvent être utilisés pour étirer ou changer la direction du mur, alors que le point de contrôle central est utilisé pour le courber dans le cas de murs courbés.
  • Les flèches de contrôle qui apparaissent dans la vue en plan sont utilisées pour étirer le mur horizontalement à partir de ses extrémités, alors que la flèche de contrôle qui apparaît dans toutes les autres vues (perspective, face et droite) est utilisée pour étirer le mur verticalement afin de pouvoir modifier sa hauteur manuellement.


Points de contrôle sur les murs

  1. Point de contrôle de départ (point d'insertion).
  2. Flèche de contrôle (vertical), (non modifiable dans la fenêtre Dessus).
  3. Point de contrôle final.
  4. Flèches de contrôle (horizontal).
  5. Points de contrôle d'un mur créé à partir d'une courbe.

Options et paramètres d'insertion

Hauteur

Même si la Hauteur est une propriété du style de mur, vous pouvez définir une valeur différente dans la boîte de dialogue d'insertion. Désactivez la case Hauteur par style et choisissez la valeur désirée en utilisant le contrôle de hauteur numérique.

Pour modifier la hauteur d'un mur, changez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).

Alignement

Trois options sont disponibles pour aligner un mur, en fonction de sa position définie à partir de la direction de sa courbe de base :

  • Centre
  • Droite
  • Gauche
Décalage de l'alignement

Insère le mur à une distance déterminée de la courbe de base

Pour modifier l'alignement d'un mur, changez sa valeur dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).

Intersections d'un mur :

Les intersections entre les murs (et leurs composants) sont calculées automatiquement. Par défaut, le mur est prolongé vers un mur proche quand il se trouve dans son rayon d'union.

  • Le rayon d'union définit la distance maximale entre la ligne de base d'un mur et celle d'un autre lorsque les deux murs sont joints.
  • L'extension maximale définit la distance maximale sur laquelle un mur sera prolongé à partir de ses extrémités jusqu'au point d'intersection avec un autre mur.

Pour modifier le rayon d'union et l'extension maximale d'un mur, changez leurs valeurs dans la boîte de dialogue Propriétés des objets ou dans la section Propriétés de VisualARQ (dans le panneau Propriétés de Rhino ).
