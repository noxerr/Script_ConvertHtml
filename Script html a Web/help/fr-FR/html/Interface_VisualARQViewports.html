Fenêtres de VisualARQ

Les fenêtres de VisualARQ ont le même comportement et sont contrôlées de la même façon que les fenêtres de Rhino. La principale différence se trouve dans la représentation des objets de VisualARQ dans la vue de Dessus (Vue en plan), appelée Étage 1 dans VisualARQ.


Vue en plan (Niveau 02)

Les objets de Rhino sont affichés en 3D « vus de dessus » mais les objets de VisualARQ sont dessinés en 2D dans ce type de vue. Le dessin 2D utilise l'affichage standard des éléments dans les plans d'architecture : les composants des murs sont affichés, les battements des portes sont montrés, les marches des escaliers sont numérotées, etc.

La Vue en plan de VisualArq est une section horizontale avec un plan de coupe situé à une élévation de 1,40 m par défaut par rapport à l'élévation de l'étage actuel. Le Gestionnaire de niveaux permet de modifier la hauteur des étages et l'élévation du plan de coupe pour chaque niveau.

Lorsque vous travaillez avec des fichiers modèles de VisualARQ, le titre de la vue en plan affiche par défaut Niveau 01 ou Étage 1. Lorsque vous créez des niveaux supplémentaires sur le dessin en utilisant le Gestionnaire de niveaux, VisualARQ vous permet de filtrer les informations affichées dans la vue sélectionnée et de montrer uniquement les éléments correspondant au niveau défini comme Niveau actuel, et la projection des objets visibles en-dessous. Le nom du niveau actuel est affiché dans la légende de la vue.

Vous pouvez lancer le Gestionnaire de niveaux de quatre façons différentes :

  • Dans le menu VisualARQ, cliquez sur Niveaux > Modifier les niveaux...
  • Cliquez avec le bouton de droite sur le titre de la vue et choisissez VisualARQ > Niveaux...
  • Dans la barre d'outils Outils de VisualARQ, cliquez sur l'icône Niveaux.
  • Lancez la commande vaNiveau dans la ligne de commandes.



Pour ouvrir la boîte de dialogue des niveaux cliquez avec le bouton sur le titre de la vue et choisissez VisualARQ > Niveaux...

Vous pouvez définir le niveau Actuel à partir de la boîte de dialogue Gestionnaire de niveaux.

Deux options permettent de passer du mode d'affichage standard d'une vue de dessus de Rhino à la représentation 2D de VisualARQ.

  • Dans la barre d'outils Gestionnaire de niveaux, cliquez sur l'icône Plan de coupe.
  • Lancez la commande vaDéfinirVue dans la ligne de commande. Cette commande change automatiquement la fenêtre active en vue de Dessus et active le plan de coupe du bâtiment ou du niveau sélectionné.

REMARQUE : Les options vaDéfinirVue > Niveau et vaDéfinirVue > Modèle peuvent être utilisées à tout moment.

Symbole d'échelle

Lorsque VisualARQ est chargé et que vous travaillez avec un de ses fichiers modèles, les fenêtres avec les vues en projection parallèle (soit toutes celles définies par défaut sauf la vue en perspective) affichent un symbole d'échelle en bas à droite.

Les options du symbole d'échelle se trouvent dans la section VisualARQ de la boîte de dialogue Options de Rhino.


Références d'élévation

Les vues de face et latérales affichent les références d'élévation ; leur nombre, élévation et nom dépendent des niveaux créés avec le gestionnaire de niveaux. La référence de niveau apparaît en surbrillance et indique le niveau dans lequel vous travaillez

Les références d'élévation peuvent être modifiées dans la section VisualARQ de la boîte de dialogue Options de Rhino.
