Caractéristiques générales

Logiciel

VisualARQ fonctionne avec Rhinoceros 4.0 SR7 et Rhinoceros 5 SR6 (32 ou 64 bits).

Versions et mises à jour

VisualARQ recherche automatiquement les mises à jour. Si une mise à jour est disponible, une icône apparaît dans la zone de notification de la barre de tâches de Windows

Une option pour désactiver la vérification automatique des mises à jour se trouve dans la section VisualARQ de la boîte de dialogue Options de Rhino .

Langues

L'interface, le programme d'installation, les fichiers modèles et les barres d'outils de VisualARQ sont disponibles en 6 langues : français, anglais, espagnol, italien, allemand et tchèque.

Une boîte de dialogue pour choisir la langue de VisualARQ se trouve dans la section VisualARQ de la boîte de dialogue Options de Rhino .

Unités

Les systèmes d'unités métrique et impérial peuvent être utilisés dans VisualARQ. Lorsque les unités du dessin sont modifiées, un message apparaît pour vous demander si vous voulez changer l'échelle des styles et des définitions de profil. Lorsque des styles d'objet sont importés et exportés, ils sont mis à l'échelle en fonction des unités du document.

  • VisualARQ - Mètres.3dm
  • VisualARQ - Millimètres.3dm
  • VisualARQ - Pieds.3dm
  • VisualARQ - Pouces.3dm

Calques

Vous pouvez configurer les calques utilisés par les commandes de création. Le panneau Calques se trouve dans la section VisualARQ de la boîte de dialogue Options de Rhino.

Lorsqu'une commande de création d'objets est lancée, le calque défini dans les options est utilisé comme calque actuel. Lorsque la commande se termine, le calque actif auparavant redevient le calque actuel sauf si l'utilisateur l'a changé pendant la commande.

Symboles

VisualARQ affiche un symbole qui indique l'échelle de la vue dans les unités du dessin. Une option pour désactiver l'icône de l'échelle se trouve dans la section VisualArq de la boîte de dialogue Options de Rhino .

VisualARQ ajoute des marques d'élévation dans les vues de face et de côté pour un travail plus facile dans les modèles à plusieurs niveaux. Une option pour désactiver les marques d'élévation se trouve dans la section VisualARQ de la boîte de dialogue Options de Rhino.

Fichiers modèles

Lors de l'installation, les fichiers modèles de VisualARQ sont copiés dans le dossier de fichiers modèles par défaut de Rhino.

Les fichiers modèles de VisualARQ contiennent une bibliothèque prédéfinie de styles d'objet qui peuvent être modifiés, importés et exportés afin de les utiliser dans d'autres fichiers.

Les fichiers modèles de VisualARQ ont une tolérance d'1 mm.

Aide

VisualARQ dispose d'options d'aide, qui sont répertoriées dans le menu principal de VisualARQ.
Parmi celles-ci vous trouverez un fichier d'aide en 6 langues, des liens vers la section d'assistance et le site Internet de VisualARQ ainsi qu'une boîte de dialogue contenant des informations sur la licence, le nom de l'utilisateur, l'entreprise et le numéro de série.
