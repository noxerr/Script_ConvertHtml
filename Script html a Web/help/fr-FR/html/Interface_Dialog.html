Boîtes de dialogue d'insertion

Les boîtes de dialogue d'insertion affichent les options de l'objet qui est inséré. Elles ne sont visibles que lorsqu'une commande est en cours.

Les panneaux des options peuvent être développés (affichés) ou réduits (cachés) en cliquant sur les petites flèches dans les coins.


Boîte de dialogue d'insertion d'un Mur.

Ces boîtes de dialogue sont divisées en plusieurs sections :

  • La section supérieure affiche une liste avec les styles d'objet disponibles.
  • La section du milieu affiche les options de base pour insérer l'objet.
  • La section inférieure affiche un aperçu - dans une vue en plan et dans une vue en perspective - de l'objet à insérer. Aucun aperçu n'est affiché lorsque cette section est réduite, le programme n'a donc pas besoin de le calculer.

REMARQUE : les objets Dalle et Toit n'utilisent pas la boîte de dialogue d'insertion.
