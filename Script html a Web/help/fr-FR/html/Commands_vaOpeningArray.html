Matrice d'ouvertures

Clavier :
vaMatriceOuverture
Icône :
clic
Barre d'outils : Outils >
Menu :
VisualARQ > Ouverture > Matrice d'ouvertures

La commande vaMatriceOuverture dépose des copies de portes ou de fenêtres dans un mur sur un certain nombre de lignes et de colonnes.

Lorsque vous lancez la commande vaMatriceOuverture, suivez les étapes de la ligne de commandes. Appuyez sur le bouton de droite ou Entrée pour accepter et passer à l'étape suivante.

Étapes :

  1. Sélectionnez un Mur où vous voulez insérer une matrice d'ouvertures. Ce mur doit contenir toutes les ouvertures que vous utiliserez pour la matrice ensuite.
  2. Sélectionnez toutes les ouvertures que vous voulez copier. Ce peut être des portes, des fenêtres ou les deux mais elles doivent être insérées dans le même mur que celui choisi avant.
  3. Tapez le Nombre de copies dans la direction du chemin.
  4. Tapez le Nombre de copies dans la direction de l'élévation.
  5. Indiquez le Point de référence à partir duquel vous commencerez les copies le long de la direction du chemin.
  6. Indiquez la Distance du chemin entre chaque copie de l'objet le long de la direction du chemin.
  7. Indiquez le Point de référence à partir duquel vous commencerez les copies le long de la direction de l'élévation.
  8. Indiquez la Distance du chemin entre chaque copie de l'objet le long de la direction de l'élévation.
