Description générale de l'interface

La plupart des commandes de VisualARQ sont groupées dans différentes barres d'outils.

Les commandes sont exécutées avec les boutons des barres d'outils, à partir du menu déroulant VisualARQ ou en écrivant le nom dans la ligne de commandes.

La plupart des commandes possèdent des cadres d'insertion, des boîtes de dialogue et des assistants qui rendent leur utilisation plus facile en vous guidant tout au long du processus.

Les éléments suivants de VisualARQ sont affichés à l'écran lorsque vous ouvrez un nouveau dessin de Rhino avec un fichier modèle de VisualARQ.


Interface de Rhino, avec un modèle de VisualARQ

  1. Menu de VisualARQ
  2. Ligne de commande : un message (surligné en turquoise) confirme que VisualARQ a été chargé. Le message montre que Tibidabo a été chargé et que VisualARQ a été installé. Il indique également la date de la version actuelle.
  3. Barres d'outils de VisualARQ : barres d'outils de base.
  4. Symbole d'échelle : il se trouve dans les fenêtres de projection parallèle. Les options du symbole d'échelle se trouvent dans la section VisualARQ de la boîte de dialogue Options de Rhino.
  5. Références d'élévation de l'Étage 1 (niveau 0) dans les vues de Face/Derrière et Gauche/Droite. Les références d'élévation peuvent être modifiées dans la section VisualARQ de la boîte de dialogue Options de Rhino.
  6. Boîte de dialogue propriétés d’un objet
  7. Boîte de dialogue du gestionnaire de calques
  8. Boîte de dialogue du gestionnaire de niveaux
