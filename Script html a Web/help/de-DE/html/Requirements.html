Systemanforderungen

Software

VisualARQ benötigt Rhinoceros 4.0 SR7 oder Rhinoceros 5 SR6 (32-Bit oder 64-Bit).

Hardware

Die Systemvoraussetzungen sind dieselben wie für Rhinoceros 4.0 und Rhino 5.

Installation

VisualARQ verfügt über einen benutzerfreundlichen Installationsassistenten, der den Prozess automatisiert.

Zur Installation und Verwendung des Programms ist ein CD-Code erforderlich. In einem Netzwerk kann eine Lizenz nur von einem Benutzer gleichzeitig verwendet werden.

Standardmäßige Installation unter: C:\Programme\AsuniCAD\VisualARQ "1.X".

Starten Sie nach der Installation des Programms Rhino wie gewohnt und wählen Sie im Begrüßungsbildschirm eine VisualARQ-Vorlage aus. Es stehen Ihnen standardmäßig die folgenden vier Vorlagen zur Verfügung:

  • VisualARQ - Meter.3dm
  • VisualARQ - Millimeter.3dm
  • VisualARQ - Fuß.3dm
  • VisualARQ - Zoll.3dm


Rhino- und VisualARQ-Vorlagen

Unter Rhino 5 (32-Bit und 64-Bit) sind die VisualARQ-Vorlagen in folgendem Ordner gespeichert: C:\Benutzer\[USERNAME]\AppData\Roaming\McNeel\Rhinoceros\5.0\Localization\<Language code>\Template Files. "Language code" bezieht sich auf die Sprachversion von VisualARQ. Wenn Sie eine deutsche Version besitzen, steht hier "de-de". Anderssprachige Versionen haben dementsprechend abweichende Codes.
