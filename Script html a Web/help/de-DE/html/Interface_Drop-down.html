VisualARQ-Menü

Die meisten Befehle des Programms sind aus dem VisualARQ-Menü heraus ausführbar. Alle VisualARQ-Objekte und -Werkzeuge, die sich in den Werkzeugleisten befinden, werden mit den entsprechenden Optionen angezeigt.


VisualARQ-Menü.

HINWEIS: Das VisualARQ-Menü wird in der Rhino-Oberfläche nach der Installation von VisualARQ angezeigt. Wenn dies nicht der Fall ist, nutzt VisualARQ eventuell eine veraltete visualarq.tb-Datei (in Rhino 4.0) oder visualarq.rui-Datei (in Rhino 5). In diesem Fall muss diese Datei manuell gelöscht und anschließend VisualARQ deinstalliert und neu installiert werden. Die Dateien visualarq.tb / visualarq.rui befinden sich in folgendem Verzeichnis:

  • visualarq.tb: (Rhino 4.0): (C:\ProgramData\McNeel\Rhinoceros\4.0)
  • visualarq.rui (Rhino 5.0 32-Bit und 64-Bit): (C:\Benutzer\"Benutzername"\AppData\Roaming\McNeel\Rhinoceros\5.0\UI )
