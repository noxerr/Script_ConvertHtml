Tabelle

Tastatur:
vaTabelle
Symbol:
Linksklick [Rechtsklick]
Werkzeugleiste Dokumentation:
Menü:
VisualARQ > Tabelle > Tabelle

In einer Tabelle werden Informationen über Objekte gleichen Typs anzeigt.

Durch Ausführen des Befehls vaTabelle wird das Dialogfenster für das Einfügen von Tabellen geöffnet. Dieses zeigt eine Liste der verfügbaren Tabellenstile an. Standardmäßig enthält die Liste die Tabellenstile für Öffnungen, Flächen und Wände, wobei neue Tabellenstile für alle anderen VisualARQ-Objekte hinzugefügt werden können.

Schritte:

  1. Wählen Sie den gewünschten Tabellenstil aus.
  2. Wählen Sie alle Objekte aus, die Sie in die Tabelle aufnehmen möchten. Ihre Auswahl wird dabei automatisch gefiltert und auf Objekte des ausgewählten Typs reduziert.
  3. Wählen Sie durch Klicken Sie mit der linken Maustaste auf einen Bereich des Bildschirms einen Einfügungspunkt für die Tabelle.

Der Vorgang wird mit dem Einfügen der Tabelle für die ausgewählten Objekte abgeschlossen.


Einfügedialog Tabelle

Kontrollpunkte

Tabellen haben einen Kontrollpunkt in der oberen linken Ecke.

Einfügungsoptionen und -parameter

Das Einfügedialogfenster für Tabellen verfügt über keine Einfügungsparameter. Sie können hier lediglich den Stil der einzufügenden Tabelle auswählen.

Die Eigenschaften einer Tabelle werden über einen Tabellenstil definiert.

Export der Tabelle nach Excel

Tabellen können mithilfe des Befehls vaTabellenExport im XLS- oder CSV-Format nach Excel exportiert werden.
