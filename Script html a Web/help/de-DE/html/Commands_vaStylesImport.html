Import von Stilen

Tastatur:
vaStilImport
Menü:
VisualARQ > Stile > Importieren

Der Befehl vaStilImport kann verwendet werden, um die Stile von beliebigen Objekten in VisualARQ zu importieren, so dass diese in der Stilbibliothek der verwendeten 3DM-Datei verfügbar sind.
Zu importierende Dateien müssen die Dateiendung .val (VisualARQ Library) haben und vorher aus der Originaldatei exportiert worden sein.

Durch Ausführen des Befehls vaStilImport wird das Dialogfenster für den Import von Stilen geöffnet.

Schritte:

  1. Öffnen Sie die VAL-Datei mit den Objektstilen, die Sie in Ihre 3DM-Datei importieren möchten. Wenn der Dateipfad nicht im Dropdown-Menü angezeigt wird, klicken Sie auf die Schaltfläche Durchsuchen, um in Ihrem System danach zu suchen. Klicken Sie auf die Schaltfläche Weiter, um fortzufahren.
  2. Wählen Sie aus der Liste alle Objektstile aus, die Sie importieren möchten und klicken Sie auf die Schaltfläche Importieren, um den Vorgang abzuschließen.

Schritt 1. Wählen Sie eine VAL-Datei aus

Schritt 2. Wählen Sie die zu importierenden Objektstile aus

Importierte Objektstile werden in einem Stildialogfenster geöffnet.
