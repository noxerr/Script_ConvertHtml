Dach: Neigungswinkel ändern

Tastatur:
vaDachNeigungswinkelÄndern
Symbol:
Linksklick
WerkzeugleisteDächer:
Menü:
VisualARQ > Dach > Neigungswinkel ändern

Option zum Ändern des Neigungswinkels eines Dachs.

Folgen Sie nach Ausführen des Befehls vaDachNeigungswinkelÄndern den Anweisungen in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.

Schritte:

  1. Wählen Sie das Dach aus, dessen Neigungswinkel Sie ändern möchten.
  2. Geben Sie einen neuen Neigungswinkel in der Befehlszeile ein.

Der Neigungswinkel kann auch im Abschnitt VisualARQ in den Rhino-Optionen geändert werden.
