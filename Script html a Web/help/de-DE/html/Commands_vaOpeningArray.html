Öffnungsanordnung

Tastatur:
vaÖffnungsAnordnung
Symbol:
Linksklick
Werkzeugleiste Werkzeuge:
Menü:
VisualARQ > Öffnung > Öffnung Anordnung

Mit dem Befehl vaÖffnungsAnordnung können Sie eine Anordnung von mehreren Türen oder Fenstern (in einer definierten Anzahl von Reihen und Spalten) in einer Wand anlegen.

Folgen Sie dazu nach Ausführung des Befehls vaÖffnungsAnordnung den Schritten in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.

Schritte:

  1. Wählen Sie eine Wand aus, in der Sie eine Anordnung von Öffnungen einfügen möchten. Diese Wand muss alle Öffnungen enthalten, die der Anordnung zugrunde liegen sollen.
  2. Wählen Sie alle Öffnungen aus, die Sie kopieren möchten. Dies können Türen, Fenster oder beides sein, aber sie müssen in dieselbe Wand eingesetzt werden, die Sie vorher ausgewählt haben.
  3. Geben Sie die Anzahl der Kopien in der Pfadrichtung an.
  4. Geben Sie die Anzahl der Kopien in Höhenrichtung an.
  5. Geben Sie den Basispunkt ein, von dem aus Sie die Kopien entlang der Pfadrichtung erstellen möchten.
  6. Geben Sie den Pfadabstand zwischen den Objektkopien entlang der Pfadrichtung der Wand ein.
  7. Geben Sie den Basispunkt ein, von dem aus Sie die Kopien entlang der Höhenrichtung erstellen möchten.
  8. Geben Sie den Pfadabstand zwischen den Objektkopien entlang der Höhenrichtung der Wand ein.
