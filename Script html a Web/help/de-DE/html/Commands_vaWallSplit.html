Wand: Teilen

Tastatur:
vaWandTeilen
Symbol:
Linksklick
Werkzeugleiste Wände:
Menü:
VisualARQ > Wand > Teilen

Option zur vertikalen Teilung einer Wand in zwei einzelne Wände.

Folgen Sie nach Ausführen des Befehls vaWandTeilen den Anweisungen in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.

Schritte:

  1. Wählen Sie die Wand aus, die Sie teilen möchten.
  2. Wählen Sie den Punkt, an dem Sie die Wand teilen möchten. Dieser Punkt befindet sich immer auf der Grundkurve der Wand.

Anschließend wird die Wand in zwei einzelne Wände geteilt, die Sie unabhängig voneinander bearbeiten können.
