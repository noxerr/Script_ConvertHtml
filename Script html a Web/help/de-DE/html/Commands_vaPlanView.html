Planansicht

Tastatur:
vaPlanansicht
Symbol:
Linksklick [Rechtsklick]
Werkzeugleiste Dokumentation:
Menü:
VisualARQ > Planansicht > Planansicht

Der Befehl vaPlanansicht stellt die 3D-Objekte dar und projiziert diese auf eine horizontale Ebene, wobei verborgene Seiten nicht angezeigt werden.

Die Planansicht wird gemäß einer bestehenden Ebene des Modells erstellt, so dass ein Gebäude und eine Ebene vorhanden sein müssen, bevor dieser Befehl ausgeführt werden kann.

Nach Ausführung des Befehls vaPlanansicht öffnet sich ein Dialog zum Einfügen, in dem eine Liste von Planansichtsstilen zur Verfügung steht. Auf diesen Dialog folgt ein weiterer, in dem die Ebene zur Anzeige der Planansicht angegeben werden muss. Folgen Sie nach dem Schließen dieser Dialoge den Schritten in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.


Einfügedialog 1: Wählen Sie den Planansichtsstil


Einfügedialog 2: Wählen Sie die Ebene der Planansicht aus.

Schritte:

  1. Wählen Sie den gewünschten Planansichtsstil aus.
  2. Wählen Sie im folgenden Dialogfenster die Ebene aus, die Sie in der zweidimensionalen Planansicht anzeigen lassen möchten. (Wenn noch keine Ebenen erzeugt wurden, beenden Sie den Befehl und führen Sie zunächst vaEbenen aus, um ein neues Gebäude und eine Ebene zu erstellen.)
  3. Bestimmen Sie die Begrenzungsfläche, innerhalb deren die Objekte in die Planansicht mit einbezogen werden. Es sind zwei Optionen verfügbar:
    1. Rechteckige Begrenzungsfläche: geben Sie die zwei gegenüberliegenden Eckpunkte der rechteckigen Fläche an.
    2. Benutzerdefinierte Begrenzungsfläche: Klicken Sie dazu in der Befehlszeile auf die Option AusKurve und wählen Sie eine bereits vorhandene geschlossene und planare Kurve, die alle Objekte umschließt, die Sie in der Planansicht anzeigen lassen möchten.
  4. Wählen Sie einen Einfügungspunkt. Klicken Sie mit der linken Maustaste auf den Bildschirm, um die Planansicht einzufügen.

Der Befehl wird beendet, wenn die 2D-Zeichnung der Planansicht am definierten Einfügungspunkt eingefügt wurde. Die Zeichnung wird auf der aktuellen Ebene erstellt.

Kontrollpunkte

Die Zeichnung der Planansicht hat dieselben Kontrollpunkte wie die Begrenzungsfläche, die die Objekte der Planansicht umschließt.

Parameter für die Planansicht

Folgende Parameter können für Planansichten definiert werden, nachdem sie erzeugt wurden. Diese Option kann nur in den VisualARQ-Eigenschaften (im Panel der Rhino-Eigenschaften ) verändert werden).
  • Stil: Der Stil bestimmt die Attribute der Planansichten.
  • Ebene: Option zur Änderung der in der Planansicht angezeigten Ebene.
  • Titel: als Titel der Planansicht wird standardmäßig der Name der Ebene verwendet. Es kann aber auch jeder andere Text verwendet werden.
  • Skala: Skalierungsfaktor, in dem die Planansicht angezeigt wird.
  • Automatische Aktualisierung: Option zur Aktivierung der automatischen Aktualisierung der Planansicht bei Änderung der Ebeneneigenschaften (Name, Höhe vom Boden oder Schnittebene).
  • Begrenzung anzeigen: Option zum Anzeigen oder Verbergen der Begrenzungsfläche der Planansicht.
  • Bemaßungen anzeigen: Mit dieser Option werden die Bemaßungslinien beim Erstellen einer Planansicht mit den Außenwänden, Dächern und Öffnungen eines Projekts gezeichnet.
  • Toleranzfaktor: dieser Wert gibt den Detailgrad an, mit dem die Planansicht erzeugt wird. Je größer dieser Wert, desto weniger Details hat die Planansicht.
