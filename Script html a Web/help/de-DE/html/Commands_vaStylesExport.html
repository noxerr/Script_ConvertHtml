Export von Stilen

Tastatur:
vaStilExport
Menü:
VisualARQ > Stile > Export

Der Befehl vaStilExport kann verwendet werden, um die Stile beliebiger VisualARQ-Objekte zu exportieren, die dann in anderen 3DM-Dateien importiert werden können.
Exportierte Dateien werden mit der Dateiendung .val (VisualARQ Library) abgespeichert.

Durch Ausführen des Befehls vaStilExport wird das Dialogfenster für den Export von Stilen geöffnet. Klicken Sie auf die Schaltfläche Weiter, um zum nächsten Schritt zu gelangen.

Schritte:

  1. Wählen Sie eine der folgenden Optionen, um die Datei zu exportieren:
    • Vorhandene Datei: Zum Speichern des neuen Stils in einer bestehenden VAL-Datei. Öffnen Sie die bestehende VAL -Datei, der Sie die zu exportierenden Objektstile hinzufügen möchten. Wenn der Dateipfad nicht im Dropdown-Menü angezeigt wird, klicken Sie auf die Schaltfläche Durchsuchen, um in Ihrem System danach zu suchen. Die neue Datei überschreibt Objektstile gleichen Namens und fügt neue Objektstile der Liste bereits vorhandener Stile hinzu.
    • Neue Datei: Zum Erzeugen einer neuen VAL-Datei mit den zu exportierenden Objektstilen. Wählen Sie eine neue VAL-Datei und klicken Sie auf Durchsuchen, um die Datei im gewünschten Ordner abzuspeichern. Wählen Sie den Einheitentyp, in dem die Stilinformationen gespeichert werden sollen.
  2. Wählen Sie die Objektstile aus, die Sie exportieren möchten und klicken Sie auf die Schaltfläche Exportieren, um den Vorgang abzuschließen.
