Wenn vorgenommene Änderungen nicht automatisch im Modell angezeigt werden, wählen Sie die bearbeiteten Objekte aus und klicken Sie auf das Symbol Aktualisieren.

HINWEIS: Der automatische Aktualisierungsprozess verlangsamt normalerweise die Leistung des Programms. Deshalb bieten nur manche Befehle diese Option an.
