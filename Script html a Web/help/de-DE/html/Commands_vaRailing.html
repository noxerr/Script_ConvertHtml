Geländer

Tastatur:
vaGeländer
Symbol:
Linksklick [Rechtsklick]
Werkzeugleiste Objekte:
Menü:
VisualARQ > Geländer > Geländer

Durch Ausführen des Befehls vaGeländer wird das Einfüge-Dialogfenster für Geländer geöffnet. Das Dialogfenster zeigt eine Liste aller verfügbaren Geländerstile sowie die grundlegenden Optionen des Geländerobjekts und die entsprechende Vorschau an.

Schritte:

  1. Wählen Sie den Geländerstil und die grundlegenden Einfügungsoptionen aus.
  2. Wählen Sie eine beliebige Anzahl von Punkten entlang des Geländerobjekts aus.
  3. Schließen Sie die Ausführung des Befehls durch Drücken von ENTER oder mit einem Rechtsklick ab.

Sie können so viele Geländer einfügen, wie Sie möchten und den Befehl durch Drücken von ENTER oder ESC beenden.


Einfügedialog Geländer

Geländer aus Kurve

Geländer können auch aus einer Kurve im Modell erzeugt werden:

Schritte:

  1. Wählen Sie den Geländerstil und die grundlegenden Einfügungsoptionen aus.
  2. Wählen Sie in der Befehlszeile AusKurven und wählen Sie dann eine Kurve oder mehrere Kurven aus dem Modell. Die Kurven können jeden beliebigen Typs oder Form sein, so lange sie offen sind.
  3. Schließen Sie die Ausführung des Befehls durch Drücken von ENTER oder mit Klick auf die rechte Maustaste ab.

Geländer samt Kontrollpunkten aus einer Kurve.

Kontrollpunkte

Geländer können je nachdem, wie sie erzeugt wurden, unterschiedliche Kontrollpunkte haben: Diese Kontrollpunkte können in alle Richtungen bewegt werden, wodurch sich die Position und die Bemaßungen des Geländers ändern.

  • Gerades Geländer: das Geländer verfügt über je einen Kontrollpunkt an jedem Ende des Objekts.
  • Wand aus Kurve: das Geländer hat genau so viele Kontrollpunkte wie die Kurve, aus der es erzeugt wurde.

Einfügungsoptionen und -parameter

Ausrichtung

Geländer können in Abhängigkeit von der Richtung der Grundkurve drei Regeln entsprechend ausgerichtet werden:

  • Mitte
  • Rechts
  • Links

Um die Ausrichtung eines Geländers zu verändern, bearbeiten Sie die Werte im Dialogfenster Objekteigenschaften oder im Abschnitt VisualARQ in den Rhino-Optionen .

Ausrichtungsversatz

Fügt das Geländer in einem bestimmten Abstand von der Richtung seiner Grundkurve ein.

Zum Ändern des Versatzes eines Geländers bearbeiten Sie seinen Wert in den Objekteigenschaften oder im Abschnitt VisualARQ in den Rhino-Optionen .

Höhe vom Boden

Die Höhe vom Boden eines Geländers wird immer in Bezug auf die Grundlinie der Ebene, in der sie eingefügt wird, gemessen. Die Höhe vom Boden ist keine Stileigenschaft. Der Wert wird für jedes Geländerobjekt beim Erzeugen desselben im Einfügedialog definiert.
Der standardmäßige Höhenwert für Geländer ist 0. Wenn Sie diesen Wert ändern, zeigt der Einfügedialog den zuletzt verwendeten Wert an.

Höhe

Obwohl die Höhe eine Eigenschaft des Geländerstils ist, können Sie einen vom Standardwert für die Höhe im Einfügedialog abweichenden Wert eingeben. Deaktivieren Sie dazu das Kontrollkästchen Höhe nach Stil und wählen Sie den gewünschten Wert anhand der numerischen Höhensteuerung aus.

Dieser Wert kann auch in den Objekteigenschaften oder im Abschnitt VisualARQ in den Rhino-Optionen geändert werden.
