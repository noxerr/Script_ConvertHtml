Dach: Begrenzung abziehen

Tastatur:
vaDachAbziehen
Symbol:
Linksklick
Werkzeugleiste Dächer:
Menü:
VisualARQ > Dach > Begrenzung abziehen

Option zum Abziehen einer Begrenzung von einem Dach.

Folgen Sie nach Ausführen des Befehls vaDachAbziehen den Anweisungen in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.

Schritte:

  1. Wählen Sie das Dach aus, von dem Sie eine Begrenzung abziehen möchten.
  2. Wählen Sie die Begrenzung aus, die Sie abziehen möchten.

Daraufhin wird dem Dach die Silhouette der Begrenzung abgezogen.

HINWEIS: Begrenzungen müssen sich nicht in der Dachebene befinden, es muss sich aber um geschlossene und planare Kurven handeln.
