Wand: Umkehrung

Tastatur:
vaWandUmkehren
Symbol:
Linksklick
Werkzeugleiste Wände:
Menü:
VisualARQ > Wand > Pfad umkehren

Damit wird die Richtung der Pfade und Seiten einer Wand umgekehrt: der Startpunkt wird zum Endpunkt, der interne Teil wird zum externen Teil, und umgekehrt.

HINWEIS: Die Türen, Fenster und Öffnungen in der gewählten Wand bleiben in ihrer Position, können aber ihre Öffnungsseite ändern, wenn Sie den Pfad der Wand umkehren.
