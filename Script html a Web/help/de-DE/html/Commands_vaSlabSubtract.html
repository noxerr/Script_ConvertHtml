Decke: Begrenzung abziehen

Tastatur:
vaDeckeAbziehen
Symbol:
Linksklick
Werkzeugleiste Decken:
Menü:
VisualARQ > Decke > Begrenzung abziehen

Option zum Abziehen einer Begrenzung von einem Decke.

Folgen Sie nach Ausführen des Befehls vaDeckeAbziehen den Anweisungen in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.

Schritte:

  1. Wählen Sie die Decke aus, von der Sie eine Begrenzung abziehen möchten.
  2. Wählen Sie die Begrenzung aus, die Sie abziehen möchten.

Daraufhin wird der Decke die Silhouette der Begrenzung abgezogen.

HINWEIS: Begrenzungen müssen sich nicht in der Deckenebene befinden, es muss sich dabei aber um planare und geschlossene Kurven handeln.
