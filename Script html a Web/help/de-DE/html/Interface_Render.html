Rendern

Der VisualARQ-Materialmanager unterstützt Materialien aus allen Renderern, die mit Rhino kompatibel sind. Die Materialzuweisung an VisualARQ-Objekte funktioniert wie in Rhino.

Eine Materialzuweisung an die verschiedenen Komponenten eines VisualARQ-Objekts ist ebenfalls möglich, indem die Material-Komponente im Dialogfenster der Stileigenschaften bearbeitet wird.

Mit Rhino kompatible Software für Animation und Rendering:

  • Air Rendering Software: Photoreal + illustration rendering + IpR (Plug-in)
  • Bongo: Animation für Designer(Plug-in)
  • Brazil für Rhino: Fortgeschrittene Rendering-Werkzeuge für Profis (Plug-in)
  • Flamingo nXt: Raytrace und Radiosity Rendering (Plug-in)
  • Hayabusa 2.0 für Rhino: Echtzeit-HQ-Renderer für Rhinoceros (mit 3DM-Unterstützung)
  • KeyShot: Programm zum Echtzeit-Raytracing und für Globale Beleuchtung (mit 3DM-Unterstützung)
  • Maxwell Render, der Lichtsimulator: Maxwell Render Plug-in für Rhino (Plug-in)
  • NuGraf für großes CAD-Rendering: Multithread-Raytracing und 3D-Konversion für Rhino (Plug-in)
  • Penguin: Konzeptueller, nicht-photorealistischer Renderer (Plug-in)
  • Renderosity: Rhino-Diskussionsforum (Link)
  • RhinoMan: RenderMan-kompatibles Rendering-Plug-in (Plug-in)
  • V-Ray for Rhino: Erweiterter Globale-Beleuchtungs-Renderer (Plug-in)
  • VSR Realtime Renderer: Voll integrierter Echtzeit-Renderer, verfügbar als Rhino-Anzeigemodus (Plug-in)
