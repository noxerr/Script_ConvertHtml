Wand: Volumenkörper abziehen

Tastatur:
vaWandVolumenkörperAbziehen
Symbol:
Linksklick
Werkzeugleiste Wände:
Menü:
VisualARQ > Wand > Volumenkörper abziehen

Option zum Abziehen eines oder mehrerer Volumenkörper von einer Wand.

Folgen Sie nach Ausführen des Befehls vaWandVolumenkörperAbziehen den Anweisungen in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.

Schritte:

  1. Wählen Sie die Wand aus, von der Sie einen oder mehrere Volumenkörper abziehen möchten.
  2. Wählen Sie die Volumenkörper aus, die Sie abziehen möchten. Diese Volumenkörper müssen geschlossene Extrusionen oder Flächenverbände sein und die Wand, von der sie abgezogen werden, an einem oder mehreren Punkten schneiden.

Das Volumen des Volumenkörpers wird aus dem Wandkörper entfernt. Das Resultat kann in einer beliebigen Ansicht angezeigt werden.

Die Wandeigenschaften werden durch diese Operation nicht beeinflusst, so dass Sie der geänderten Wand Türen und Fenster hinzufügen können.
