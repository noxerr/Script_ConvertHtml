VisualARQ-Ansichtsfenster

VisualARQ-Ansichtsfenster weisen das gleiche Verhaltens- und Steuerungsmuster wie Rhino-Ansichtsfenster auf. Der Hauptunterschied liegt in der Darstellung der VisualARQ-Objekte in der Draufsicht (Planansicht), die in VisualARQ "Geschoss 1" heißt.


Planansicht (Ebene 02).

Rhino-Objekte werden von oben in 3D angezeigt, während VisualARQ-Objekte in diesen Ansichten in 2D gezeichnet werden. Die 2D-Darstellung nutzt die Standardanzeige von Geschosselementen in Architekturzeichnungen: Wände zeigen ihre Komponenten an, Türen ihre Öffnungsrichtung, Stufen von Treppen werden nummeriert, etc.

Die VisualARQ-Planansicht ist ein horizontaler Schnitt mit einer Schnittebene auf einer standardmäßigen Höhe von 1,40 m über dem aktuellen Geschoss. Die können die Höhe der Geschosse und die Höhe der Schnittebene jedes Geschosses im Dialogfenster Ebenenmanager ändern.

Bei Verwendung von VisualARQ-Vorlagen wird als Titel der Planansicht standardmäßig als Ebene 01 oder Erdgeschoss angegeben. Wenn Sie im Dialogfenster Ebenenmanager weitere Ebenen erstellen, können Sie die in der ausgewählten Ansicht dargestellten Informationen filtern und sich nur die der als Aktuelle Ebene definierten Ebene entsprechenden Elemente und eine Projektion der sichtbaren Elemente darunter anzeigen lassen. Die aktuelle Ebene wird im Tooltip der Ansicht angezeigt.

Der Ebenenmanager kann auf vier verschiedene Arten geöffnet werden:

  • Wählen Sie im Menü VisualARQ den Eintrag Ebenen > Ebenen bearbeiten ...
  • Klicken Sie mit der rechten Maustaste auf den Titel der Ansicht und wählen Sie VisualARQ > Ebenen...
  • Klicken Sie in der VisualARQ-Werkzeugleiste auf die Schaltfläche Ebenen.
  • Führen Sie in der Befehlszeile den Befehl vaEbenen aus.



Zum Öffnen des Ebenen-Dialogfensters klicken Sie mit der rechten Maustaste auf den Titel der Ansicht und wählen Sie VisualARQ > Ebenen ...

Sie können jede beliebige Ebene im Ebenenmanager als Aktuell setzen.

Es gibt zwei Möglichkeiten, zwischen der üblichen Rhino-Draufsicht und der VisualARQ-2D-Zeichnung zu wechseln.

  • Klicken Sie im Ebenenmanager auf die Schaltfläche Schnittebene.
  • Geben Sie in der Befehlszeile den Befehl vaAnsichtDefinieren ein. Dadurch wird im aktuellen Ansichtsfenster die Draufsicht eingestellt und die Schnittebene eines gewählten Gebäudes oder einer Ebene aktiviert.

HINWEIS: Sie können die Optionen vaAnsichtDefinieren > Ebene und vaAnsichtDefinieren > Modell in jeder Ansicht verwenden.

Skalasymbol

Wenn VisualARQ geladen ist und Sie mit einer VisualARQ-Vorlagen arbeiten, zeigen die Ansichtsfenster mit paralleler Projektion (d.h. alle außer der perspektivischen Ansicht) ein der unteren rechten Ecke eine Skala an.

Die Optionen des Skalasymbols können in den VisualARQ-Optionen im Dialogfenster der Rhino-Optionen bearbeitet werden.


Höhenreferenzen

In der Front- und Seitenansicht werden die Höhenreferenzen inklusive Nummer, Name und Höhe vom Boden angezeigt, die im Ebenenmanager eingestellt wurden. Die Ebene, die Sie gerade bearbeiten, wird dabei farblich hervorgehoben dargestellt.

Die Höhenreferenzen können in den VisualARQ-Optionen im Dialogfenster der Rhino-Optionen eingestellt werden.
