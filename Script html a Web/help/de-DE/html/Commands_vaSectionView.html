Schnittansicht

Tastatur:
vaSchnittAnsicht
Symbol:
Linksklick [ Rechtsklick]
Werkzeugleiste Dokumentation:
Menü:
VisualARQ > Schnitt > Schnittansicht

Durch Ausführen des Befehls vaSchnittAnsicht wird eine 2D-Zeichnung erzeugt, die die 3D-Objekte in einer Ebene darstellt. Die Schnittansicht wird gemäß der Position der Schnittmarkierung im Modell erstellt. Nach Ausführung des Befehls wird das Einfüge-Dialogfenster für die Schnittansicht geöffnet. Folgen Sie den Schritten in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.


Einfügedialog Schnittansicht

Schritte:

  1. Wählen Sie im Einfügedialog einen Schnittansichtsstil aus.
  2. Wählen Sie die Schnittmarkierung aus.
  3. Wählen Sie einen Einfügungspunkt. Klicken Sie mit der linken Maustaste auf den Bildschirm, um die Schnittzeichnung einzufügen.

Diese Schnittzeichnung verbleibt an die Schnittmarkierung angeheftet, wenn Sie sie nicht zerlegen. Wenn die Schnittmarkierung gelöscht wird, verschwindet auch die Schnittansicht. Bei Verschieben der Position der Schnittmarkierung wird die Schnittzeichnung automatisch aktualisiert. Die Schnittzeichnung behält zudem die Ebeneneigenschaften der Objekte bei, die sie repräsentiert.

Parameter einer Schnittansicht

Folgende Parameter können für Schnittansichtsobjekte definiert werden, nachdem sie erzeugt wurden. Diese Option kann nur in den VisualARQ-Eigenschaften (im Panel der Rhino-Eigenschaften ) verändert werden).
  • Stil: Der Stil definiert die Attribute der Schnittansichten
  • Titel: Der Titel der Schnittansicht ist standardmäßig der Referenztext der entsprechenden Schnittmarkierung, dem derselbe Text noch einmal mit einem nachgeschobenen " ' " hinzugefügt wird. (z.B. Schnittreferenztext = A, Titel der Schnittansicht = A-A'). Der Titel der Schnittansicht kann aber auch durch einen anderen Text ersetzt werden:
  • Schnitt: Option zur Änderung der Schnittansicht durch Auswahl einer anderen Schnittmarkierung.
  • Automatische Aktualisierung: Option zur Aktivierung eines automatischen Updates der Schnittansicht, wenn das entsprechende Schnittmarkierungsobjekt bewegt oder bearbeitet wird.
  • Toleranzfaktor: dieser Wert gibt den Detailgrad an, mit dem die Schnittansicht erzeugt wird. Je größer dieser Wert, desto weniger Details hat die Schnittansicht.

Modell in Dynamischer Schnittansicht, Schnittmarkierung und Schnittansicht

Anmerkung 1: Versteckte Seiten werden in der Schnittansicht nicht angezeigt.

Anmerkung 2: Die Schnittansicht und die entsprechende Schnittmarkierung werden in der Ebene erzeugt, die bei Ausführung des Befehls aktiviert ist. Wenn eine gelöscht wird, wird die andere ebenfalls gelöscht.

Anmerkung 3: In der Schnittansicht werden nur die im Schnittbereich durch ihre Einfügungspunkte und Schnittlängenpunkte beschränkten Objekte angezeigt.
