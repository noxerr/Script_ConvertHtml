Übersicht über die Benutzeroberfläche

Die meisten VisualARQ-Befehle sind in verschiedenen Werkzeugleisten gruppiert.

Sie können Befehle in VisualARQ über die Schaltflächen der Werkzeugleisten, aus dem VisualARQ-Menü heraus oder durch Eingabe in die Befehlszeile ausführen.

Die meisten Befehle verfügen über Einfügedialoge, Dialogfenster und/oder Assistenten, die Sie durch die Ausführung begleiten.

Die folgenden VisualARQ-Elemente werden auf dem Bildschirm angezeigt, wenn Sie eine neue Rhino-Zeichnung mit einer VisualARQ-Vorlage öffnen.


Rhino-Benutzeroberfläche mit einer geöffneten VisualARQ-Vorlage.

  1. VisualARQ-Menü
  2. Befehlszeile: Eine türkis hervorgehobene Nachricht in der Befehlszeile zeigt an, dass VisualARQ geladen wurde. Die Nachricht weist darauf hin, dass Tibidabo geladen und VisualARQ installiert wurde. Zudem wird das Datum des Builds angegeben.
  3. VisualARQ-Werkzeugleisten: Drei Basis-Werkzeugleisten.
  4. Skalasymbol: Dieses befindet sich in den Ansichtsfenstern mit paralleler Projektionsansicht. Die Optionen des Skalasymbols können in den VisualARQ-Optionen im Dialogfenster der Rhino-Optionen bearbeitet werden.
  5. Die Höhenangaben von Geschoss 1 (Höhe 0.00) in den Ansichten Front/Rück und Links/Rechts. Die Höhenreferenzen können in den VisualARQ-Optionen im Dialogfenster der Rhino-Optionen eingestellt werden.
  6. Objekteigenschaftendialog
  7. Ebenenmanager für die Objektschichten eines Modells
  8. Ebenenmanager für die Geschossebenen eines Modells
