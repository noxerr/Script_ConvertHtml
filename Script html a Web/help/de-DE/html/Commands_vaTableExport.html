Tabellenexport

Tastatur:
vaTabellenExport
Menü:
VisualARQ > Bauteiltabelle > Exportieren...
Kontextmenü:
Bauteiltabelle > Tabelle exportieren...

Tabellen können im XLS- oder CSV-Format nach Excel exportiert werden.

Schritte:

  1. Wählen Sie die zu exportierende Bauteiltabelle aus.
  2. Wählen Sie den Ordner und das Dateiformat aus, in dem Sie Ihre Tabelle speichern möchten: .xls oder .csv

In dem gewählten Zielordner wird nun eine neue Datei erzeugt. Diese Datei wird aber nicht automatisch aktualisiert, wenn an der entsprechenden VisualARQ-Tabelle Änderungen vorgenommen werden. Um die exportierte Datei auf den neuesten Stand zu bringe, wiederholen Sie einfach die obigen Schritte und überschreiben Sie die vorhandene Version.
