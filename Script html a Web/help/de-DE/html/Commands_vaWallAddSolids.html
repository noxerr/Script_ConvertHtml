Wand: Volumenkörper hinzufügen

Tastatur:
vaWandVolumenkörperHinzufügen
Symbol:
Linksklick
Werkzeugleiste Wände:
Menü:
VisualARQ > Wand > Volumenkörper hinzufügen

Option zum Hinzufügen eines oder mehrerer Volumenkörper an eine Wand.

Folgen Sie nach Ausführen des Befehls vaWandVolumenkörperHinzufügen den Anweisungen in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.

Schritte:

  1. Wählen Sie die Wand aus, der Sie einen oder mehrere Volumenkörper hinzufügen möchten.
  2. Wählen Sie die Volumenkörper aus, die Sie hinzufügen möchten. Diese Volumenkörper müssen geschlossene Extrusionen oder Flächenverbände sein und die Wand, zu der sie hinzugefügt werden, an einem oder mehreren Punkten schneiden.

Das Volumen eines Volumenkörpers wird dem Wandkörper hinzugefügt und wird Teil der Wand. Das Resultat kann in einer beliebigen Ansicht angezeigt werden.

Die Wandeigenschaften werden durch diese Operation nicht beeinflusst, so dass Sie der geänderten Wand Türen und Fenster hinzufügen können.
