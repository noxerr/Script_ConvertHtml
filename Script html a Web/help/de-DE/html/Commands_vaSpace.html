Fläche

Tastatur:
vaFläche
Symbol:
Linksklick [Rechtsklick]
Werkzeugleiste Dokumentation:
Menü:
VisualARQ > Fläche >
  • Aus Punkt
  • Aus Kurve
  • Aus planarer Fläche

    • Das Flächenobjekt stellt das Flächenelement dar, das die Informationen (Name, Flächeninhalt, Umfang, etc.) einer Fläche oder eines Raums mit einem Schraffierungsmuster und einer Markierung anzeigt.

    Ein Raum wird definiert als eine beliebige geschlossene Fläche, die durch Wände und/oder Stützen auf allen vier Seiten begrenzt wird. Wände und Stützen können sich auf verschiedenen Höhe befinden, wenn sie Teil des gleichen Geschosses sind.

    Schritte:

    1. Es gibt drei Möglichkeiten, eine Fläche zu erzeugen. Diese Optionen erscheinen in der Befehlszeile, wenn Sie den Befehl vaFläche ausführen:
      • Fläche aus Wänden und/oder Stützen (Raum): Klicken Sie mit der linken Maustaste in den Raum, für den Sie die Fläche berechnen möchten.
      • Fläche aus einer Kurve: Klicken Sie auf die Befehlszeilenoption AusKurve. Wählen Sie eine geschlossene und planare Kurve aus.
      • Fläche aus einer Fläche: Klicken Sie auf die Befehlszeilenoption AusPlanarFläche. Wählen Sie eine planare Fläche.
    2. Daraufhin öffnet sich das Dialogfenster Neue Fläche. Geben Sie folgende Informationen für die ausgewählte Fläche ein:
      • Name
      • Flächentyp: Nützlich oder Bebaut
      • Stil

    Einfügedialog Fläche

    Daraufhin wird ein Schraffierungsmuster zur Kennzeichnung der benutzten Fläche eingefügt und auf einem Etikett der Name der Fläche und ihre geometrischen Eigenschaften angezeigt.

    Sie können beliebig viele Flächen erzeugen und die Ausführung des Befehls durch Drücken von ENTER oder ESC beenden.

    Bearbeitung einer Fläche

    Tastatur:
    vaFlächeKonturDefinieren

    Eine Fläche kann mithilfe des Befehls vaFlächeKonturDefinieren auf drei verschiedene Arten verändert werden:

    1. Wählen Sie die Fläche aus, die Sie ändern möchten.
    2. Wählen Sie einen neuen Punkt (in einem Raum) zum Einfügen der Fläche oder wählen Sie eine der Optionen in der Befehlszeile: AusKurve oder AusPlanarFläche.

    Beim Bearbeiten einer Fläche bleiben ihre Eigenschaften erhalten.

    ANMERKUNG: Flächen aus Wänden und/oder Stützen können manuell mit der Maus in andere Räume verschoben werden. Sie können außerdem mit dem Befehl vaAktualisieren aktualisiert werden, wenn die ursprünglichen Wände bzw. Stützen verändert wurden.

    Kontrollpunkte

    Flächen verfügen über zwei Kontrollpunkte:

    1. Einen Einfügungspunkt für das Schraffierungsmuster, das die Fläche markiert.
    2. Einen Einfügungspunkt für das Etikett.

    HINWEIS: Alle Kontrollpunkte der Fläche werden verwendet, um diese zu verschieben (sie skalieren die Fläche nicht).

    Um eine Fläche in einen neuen Raum zu verschieben, wählen Sie die Fläche aus (Schraffierungsmuster und Etikett) und ziehen Sie sie in den neuen Raum.

    Um die Flächenbeschriftung zu verschieben, aktivieren Sie die Flächenkontrollpunkte und verschieben Sie den Etikettenkontrollpunkt an den gewünschten Punkt.

    Einfügungsoptionen und -parameter

    Der Einfügedialog für die Fläche bietet folgende Einstellungsmöglichkeiten: Name, Flächentyp und Stil.

    Die Parameter, die die Eigenschaften einer Fläche definieren, entsprechen einem Flächenstil, obgleich sie auch unabhängig von ihrem Stil im Dialogfenster der Objekteigenschaften oder im Abschnitt VisualARQ in den Rhino-Eigenschaften geändert werden können.

    Name

    Der Name einer Fläche wird zur Unterscheidung und zur Angabe ihrer Funktionalität verwendet.

    Um den Namen einer Fläche zu ändern, gehen Sie in das Dialogfenster der Objekteigenschaften und klicken Sie mit der linken Maustaste auf den Namen der Fläche.

    Flächentyp

    Es sind zwei Typen von Flächen erhältlich:

    • Nützlich: eine durch einen Raum oder ein Gebäude begrenzte Fläche (ausschließlich der Wandstärke).
    • Bebaut: eine durch die äußeren Begrenzungen der Konstruktionsebene begrenzte Fläche.

    Um den Flächentyp einer Fläche zu ändern, löschen Sie die Fläche und erzeugen Sie sie erneut unter Auswahl des gewünschten Flächentyps.

    Höhe

    Das Flächenobjekt hat einen Höhenwert, aus dem sich der Rauminhalt berechnet. Dieser Wert kann im Abschnitt VisualARQ in den Rhino-Eigenschaften geändert werden).
