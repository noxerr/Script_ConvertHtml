Wand: Personalisierten Volumenkörper definieren

Tastatur:
vaWandBenutzerdefiniertenVolumenkörperDefinieren
Symbol:
Rechtsklick [Linksklick]
Werkzeugleiste Wände:
Menü:
VisualARQ > Wand > Benutzerdefinierten Volumenkörper definieren

Option zur Verwandlung einer Wand in eine andere auf einem in Rhino modellierten Volumenkörper basierende Wand.

Folgen Sie nach Ausführen des Befehls vaWandBenutzerdefiniertenVolumenkörperDefinieren den Anweisungen in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.

Schritte:

  1. Wählen Sie die Wand aus, die Sie an das Rhino-Objekt anpassen möchten.
  2. Wählen Sie den Rhino-Volumenkörper aus, auf dem die neue Wand basieren soll.

HINWEIS: Beachten Sie, dass die neu erzeugte Wand als Grundkurve (zur Bestimmung der Ausrichtung der Öffnungen) diejenige der ursprünglichen Wand übernimmt.
