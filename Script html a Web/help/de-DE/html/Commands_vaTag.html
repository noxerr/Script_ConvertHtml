Mit dem Befehl Beschriftung können Sie den VisualARQ-Objekten Etiketten mit speziellen Referenztexten zuweisen.

Vor Ausführung des Befehls muss das Beschriftungsfeld des Objekts muss ausgefüllt werden. Sie können das Beschriftungsfeld eines VisualARQ-Objekts in den VisualARQ-Eigenschaften (in den Rhino-Eigenschaften ) im Abschnitt Allgemein ausfüllen.

Durch Ausführen des Befehls wird das Einfüge-Dialogfenster für Beschriftungen geöffnet. Folgen Sie den Schritten in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.


Einfügedialog Beschriftung

Schritte:

  1. Wählen Sie im Einfügedialog einen Beschriftungsstil aus.
  2. Wählen Sie die zu beschriftenden Objekte aus.

Die Beschriftung wird automatisch neben dem ausgewählten Objekt erzeugt.

Beschriftungskontrollpunkte

Beschriftungen haben keine Kontrollpunkte. Sie können unabhängig von der Position Ihrer Referenzobjekte verschoben werden. Beim Verschieben der Referenzobjekte werden auch die Beschriftungen automatisch verschoben und behalten ihre relative Position, falls diese nicht als fest eingestellt sind.

Löschen einer Beschriftung

Zum Löschen einer Beschriftung wählen Sie diese einfach aus und löschen Sie sie.

Beschriftungsparameter

Folgende Parameter können für Beschriftungen definiert werden, nachdem sie erzeugt wurden. Diese Option kann nur in den VisualARQ-Eigenschaften (im Panel der Rhino-Eigenschaften ) verändert werden).
  • Stil: Der Stil bestimmt die Attribute der Beschriftungen.
  • Position: Die Weltkoordinaten der Beschriftung.
  • Feste Position: Mit dieser Option können Sie die Position der Beschriftung an der gewünschten Stelle fixieren, die sich auch beim Verschieben des Referenzobjekts nicht ändert.
  • Format: Das Textformat der Beschriftung. Mit der Formel "%<element.tag>%" können Sie Text vor und/oder nach dem Referenztext der Beschriftung hinzufügen.
    • Geben Sie beispielsweise für eine Tür mit dem Referenztext "D3" die Formel "Tür %<element.tag>%" ein, um schließlich als Beschriftung "Tür D3" zu erhalten.
