Kontur der Treppendurchgangsöffnung

Tastatur:
vaTreppeKonturDurchgangsöffnung
Menü:
VisualARQ > Treppe > Kontur Durchgangsöffnung

Damit wird eine Konturkurve erzeugt, die dazu verwendet werden kann, das Profil einer Treppendurchgangsöffnung zu berechnen.

Folgen Sie nach Ausführen des Befehls vaTreppeKonturDurchgangsöffnung den Anweisungen in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.

Schritte:

  1. Wählen Sie die Treppe aus, von der Sie eine Profilform erstellen möchten.
  2. Geben Sie die Höhe der Treppendurchgangsöffnung an.
  3. Geben Sie die Stärke der Decke an, auf der Sie die Konturkurve anlegen möchten.

Die Kurve wird oberhalb der Treppenhöhe erstellt.


h: Höhe Treppendurchgangsöffnung
