Decke: Begrenzung hinzufügen

Tastatur:
vaDeckeHinzufügen
Symbol:
Linksklick
Werkzeugleiste Decken:
Menü:
VisualARQ > Decke > Begrenzung hinzufügen

Option zum Hinzufügen einer Begrenzung an eine Decke.

Folgen Sie nach Ausführen des Befehls vaDeckeHinzufügen den Anweisungen in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.

Schritte:

  1. Wählen Sie die Decke, zu der Sie eine Begrenzung hinzufügen möchten.
  2. Wählen Sie die Begrenzung aus, die Sie hinzufügen möchten.

Daraufhin wird der Originalform der Decke die Silhouette der neuen Begrenzung hinzugefügt.

HINWEIS: Begrenzungen müssen sich nicht in der Deckenebene befinden, es muss sich dabei aber um planare und geschlossene Kurven handeln.
