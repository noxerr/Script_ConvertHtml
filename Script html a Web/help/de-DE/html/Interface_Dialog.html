Einfügedialoge

In den Einfügedialogen können Sie die Optionen des einzufügenden Objekts festlegen. Die Dialogfenster werden nur während der Ausführung eines Befehls angezeigt.

Die einzelnen Abschnitte darin können Sie durch einen Klick auf den kleinen Pfeil in der rechten oberen Ecke aus- bzw. wieder einblenden.


Einfügedialog für das Wandobjekt.

Die Einfügedialoge werden in drei oder mehr Abschnitte unterteilt:

  • Der obere Teil zeigt eine Liste der verfügbaren Objektstile an.
  • Die mittleren Abschnitte zeigen die grundlegende Einfügungsoption des Objekts an.
  • Der untere Abschnitt zeigt eine Vorschau - in Drauf- und perspektivischer Ansicht - des einzufügenden Objekts an. Wenn dieser Abschnitt ausgeblendet ist, wird keine Vorschau angezeigt und das Programm muss sie dementsprechend auch nicht berechnen.

HINWEIS: Decken- und Dachobjekte verwenden keine Einfügedialoge.
