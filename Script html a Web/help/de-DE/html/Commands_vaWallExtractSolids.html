Wand: Volumenkörper extrahieren

Tastatur:
vaWandVolumenkörperExtrahieren
Symbol:
Linksklick
Werkzeugleiste Wände:
Menü:
VisualARQ > Wand > Volumenkörper Extrahieren

Option zum Extrahieren eines oder mehrerer Volumenkörper, die vorher einer Wand hinzugefügt wurden.

Folgen Sie nach Ausführen des Befehls vaWandVolumenkörperExtrahieren den Anweisungen in der Befehlszeile. Bestätigen Sie mit ENTER oder der rechten Maustaste und gehen Sie zum nächsten Schritt.

Schritte:

  1. Wählen Sie die Wand, von der Sie die vorher hinzugefügten Volumenkörper extrahieren möchten.
  2. Drücken Sie die rechte Maustaste oder die Eingabetaste, um die Wandauswahl zu bestätigen. Die Volumenkörper werden dadurch von der Wand abgetrennt und erneut zu unabhängigen Elementen.
