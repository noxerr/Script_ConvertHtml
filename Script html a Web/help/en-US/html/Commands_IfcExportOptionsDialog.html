IFC Export Options Dialog

Keyboard:
vaIfcExportOptions
Icon:
right click [left click]
Documentation toolbar:
Menu:
VisualARQ > Ifc > Layer Properties

The Ifc Export Options Dialog command opens a floating dialog box, from which it is possible to assign an IFC class to the existing layers in the document.

The IFC class define the categories of the objects when they are exported to IFC.

The Ifc Export Options Dialog can also be launched from the Save Dialog (Menu: File > Save as..), selecting the IFC file format, and clicking on the Options... button.

Steps:

  1. Click on the IFC class name, next to each layer, to assign an IFC type from the list displayed.

The checkbox next to each Layer's name determines whether the geometry of that layer will be exported to IFC or not.
