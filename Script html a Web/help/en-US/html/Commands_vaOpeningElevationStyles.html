Opening elevation Styles

Keyboard:
vaOpeningElevationStyles
Icon:
right click [left click]
Documentation toolbar:
Menu:
VisualARQ > opening elevation > opening elevation Styles
Shortcut key:
Select a opening elevation object > (Alt, Shift or Ctrl) + double left click

The Opening elevation style defines the attributes of Opening elevations.

New Opening elevation style:

You can create new styles and duplicate existing styles from the Opening elevation Style Manager dialog box.

Delete an opening elevation style:

You can delete opening elevation styles from the Opening elevation Style Manager dialog box. You can only delete those styles that are not being used by any existing opening elevation in the model.

Edit an opening elevation style:

Select the opening elevation style at the top left panel, and edit their different parameters from the tabs that appear in the right panel.


Opening Elevation Styles dialog

General style editing
You can use the vaOpeningElevationStyles command or, alternatively,
  • Use the vaStyleProperties command, then select a opening elevation and press Enter, or
  • Right click on , then select an opening elevation and press Enter, or
  • Select an opening elevation and open the context menu by pressing the right mouse button for a while then select Opening elevation > Style Properties, or
  • Press Alt, Shift or Ctrl then double click on an opening elevation object.
The changes applied to a style are applied to all elements with that style.
Object-specific editing
  • Use the vaProperties command, then select an opening elevation and press Enter, or
  • Click on , then select an opening elevation and press Enter, or
  • Select an opening elevation and open the context menu by pressing the right mouse button for a while then select Opening elevation > Properties, or,
  • Double click on an opening elevation object.
The changes are applied only to the style of the selected element.

Attributes

Attributes define the features of each opening elevation style for display and print purposes. The valid attributes are:

  • Layer, Visibility, Text style, Dimension style, Color, Linetype, Line print width and Line print color.
