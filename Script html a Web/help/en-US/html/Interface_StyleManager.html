Style Manager

The Style Manager dialog box can be used to edit the style properties of the selected object and to create new styles for that object.

To create new object styles, press the New style button.


Style Manager dialog box for the Door object.

A wizard will guide you throughout the process, requesting the parameters of the object style defined in simple steps.


Door Style wizard.

A preview of the object is displayed on the lower left box when you select a style from the object style list.

The selected component is marked in turquoise when the object has various components.

The preview can be modified with the mouse, similar to the method used in Rhino views:

  • Right mouse button: you can rotate the viewpoint to display the model from different points.
  • Ctrl + Right mouse button: you can zoom in or out to expand or reduce the field of view.
  • Shift + Right mouse button: you can move the viewpoint parallel to the plane of vision; upwards, downwards, right or left.


Style Manager dialog box, Attributes tab.

For each seleted object style you can define the properties shown on the right box. This area includes several tabs, which can be different, depending on the object type:

  • Attributes
    This tab is displayed for all objects that use a dialog box. The tab defines the features (attributes) of each object style and each of its components. The attributes are:
    • Layer
    • Visibility
    • Display color
    • Linetype
    • Print width
    • Print color
    • Material
    • Hatch
    • Text style
    • Dimension style

    NOTE: From the Material attribute of the object style you can assign a different material to each object component. To assign a single material to the whole object, follow the usual procedure in Rhino.
  • Geometry
    Walls, columns, doors and windows have this tab. The geometric values of each object and object component are defined in this tab.
    Geometric values for the Wall object:
    • Height
    • Width (can only be edited for each wall component)
    Geometric values for the Column object:
    • Height
    • Profile (defines the object shape)
    • Coating width
    Geometric values for the Door object:
    • Profile (defines the object shape)
    • Sizes (width and depth) for the frame and doorstop
    • Sizes (width and depth) and opening side for the leaf
    Geometric values for the Window object:
    • Profile (defines the object shape)
    • Sizes (width and depth) for the frame and doorstop
    • Sizes (width and depth) and opening side for the leaf
    • Sizes (thickness, extension and depth) for the sill.
  • Sizes
    This tab is only used for the objects that use profiles to define their shape (columns, doors and windows). This tab is used to define the sizes (width and height) of the profile used by the object. This option can be used to create different sizes for the same profile.
  • Blocks
    Only doors and windows have this tab. The tab defines the 2D (Plan) and 3D (Model) representations of the object, assigning a block that has been previously created by the user with custom features to each one. This option can be used to create custom doors and windows.
  • Reinforcements
    Only walls have this tab. This tab is used to define the value of each type of wall reinforcement (Start, End, Corner and Opening).
  • Type
    Only stairs have this tab. It is used to define the type of stair used, depending on whether the Balanced stair option is activated or not.
  • Steps
    Only stairs have this tab, where the geometric values of the tread and riser slabs can be specified. Geometric values for the Tread slab:
    • Slab thickness
    • Nosing
    Geometric values for the Riser slab:
    • Slab thickness
