Section View Styles

Keyboard:
vaSectionViewStyles
Icon:
right click [left click]
Documentation toolbar:
Menu:
VisualARQ > Section > Section View Styles
Shortcut key:
Select a section view object > (Alt, Shift or Ctrl) + double left click

The section view style defines the attributes of section views.

New Section view style:

You can create new styles and duplicate existing styles from the Section View Style Manager dialog box.

Delete a Section view style:

You can delete section view styles from the Section view Style Manager dialog box. You can only delete those styles that are not being used by any existing Section view in the model.

Edit a Section view style:

Select the section view style at the top left panel, and edit their different parameters from the tabs that appear in the right panel.


Section view styles dialog

General style editing
You can use the vaSectionViewStyles command or, alternatively,
  • Use the vaStyleProperties command, then select a section view and press Enter, or
  • Right click on , then select a section view and press Enter, or
  • Select a section view and open the context menu by pressing the right mouse button for a while then select Section View> Style Properties, or
  • Press Alt, Shift or Ctrl then double click on a section view object.
The changes applied to a style are applied to all elements with that style.
Object-specific editing
  • Use the vaProperties command, then select a section view and press Enter, or
  • Click on , then select a section view and press Enter, or
  • Select a section view and open the context menu by pressing the right mouse button for a while then select Section View > Properties, or,
  • Double click on a section view object.
The changes are applied only to the style of the selected element.

Attributes

Attributes define the features of each section view style for display and print purposes. The valid attributes are:

  • Section View: Layer and Visibility
  • Section View text: Text style, Dimension style, Color, Linetype, Line print width and Line print color.
  • Sectioned objects in a Section View: Pattern, Pattern scale, Pattern angle and Pattern color
