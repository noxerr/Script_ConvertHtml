System Requirements

Software

VisualARQ requires Rhinoceros 4.0 SR7 or Rhinoceros 5 SR6 (32 or 64 bit) to run.

Hardware

Hardware requirements are the same as for Rhinoceros 4.0 and Rhino 5.

Installation

VisualARQ has an easy to use installation wizard that automates the process.

The product license key is required for installation and use. A license can only be used once simultaneously on the network.

It installs by default in: C:\Program Files\AsuniCAD\VisualARQ "1.X".

After you have installed the program, click on the Rhino shortcut icon to display the dialog box that shows the templates you can use. Please select one of the four templates shown before starting to work with VisualARQ:

  • VisualARQ - Meters.3dm
  • VisualARQ - Milimeters.3dm
  • VisualARQ - Feet.3dm
  • VisualARQ - Inches.3dm


Rhino and VisualARQ template list.

In Rhino 5 (32 and 64-bit), the VisualARQ templates are stored in the following directory: C:\Users\"UserName"\AppData\Roaming\McNeel\Rhinoceros\5.0\Localization\en-US\Template Files. (This directory corresponds to the en-US templates folder. Change "en-US" to the corresponding language folder to find the other directories where the VisualARQ templates are stored in different languages).
