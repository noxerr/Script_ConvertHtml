Edit Levels

Keyboard:
vaLevels
Icon:
left click
Tools toolbar:
Menu:
VisualARQ > Levels > Edit levels

The vaLevels command opens a dockable Level manager dialog box from which it is possible to create, delete and edit new buildings and levels.

From the Level manager dialog box you can control each level CPlane (construction plane), visibility and its Elevation and Cut Plane values.

The Level manager dialog box is divided in two parts:

  • A toolbar line with the following options: New Building, New Level, Delete and Cut Plane visualization.
  • The list of buildings and levels and their properties: CPlane (construction plane), On/Off visibility, Plan View visibility, Elevation value, Cut Plane Height value, Top Offset and Bottom Offset values.


Level Manager dialog box.



Parameters for Level "1":

  • L0: Elevation of Level "0" and position of Cplane of Level 0
  • L1: Elevation of Level "1" and position of Cplane of Level 1
  • L2: Elevation of Level "2" and position of Cplane of Level 2 (and top limits of Level 1)
  • C.P: Level 1 Cut Plane height
  • T.O.: Top offset distance of Level 1
  • B.O.: Bottom offset distance of Level 1

New building

Creates new buildings from which you can add levels. Each building has its own elevation value and the elevation values of the contained levels are relative to that building.

New level

Adds a new level to a selected building. The new level will be automatically created above the upper existing level, and will have by default the same elevation value than the level underneath. A building element must be created to create levels.

Delete

Deletes a specific level or building. This does not affect all Rhino and VisualARQ objects which are contained within the deleted level or building. They remain in the same place.

Cut Plane

The Cut Plane represents an invisible horizontal section plane that each level has, situated at a specific height from each level elevation. The Cut Plane button turns On or Off the Cut Plane of a selected level, in the active viewport. A building and a level must be created to enable the Cut Plane status.

When the Cut Plane is turned On for a specific level in the active viewport, everything above the cutting plane of that level will NOT be displayed.

When the Cut Plane is turned On in Top viewports, the VisualARQ objects, such as doors, windows, stairs, etc... will show their 2D plan view graphic representation.


Cut Plane On: the viewport shows the level sectioned according to its Cut Plane position.
Cut Plane Off: the viewport displays the entire level (according to its visibility On/Off status).
Cut Plane disabled: you need to select a level to enable its Cut Plane status.

CPlane

The CPlane (construction plane) determines the plane elevation from where you are working on. Click on any level CPlane icon to move the CPlane into the elevation of that level in the active viewport.

On/Off

The level On/Off visibility status shows or hides the objects or parts of the objects situated between the current level Elevation, and the upper level Elevation (taking into account the Top and Bottom offset values).

The level is visible and you can see objects or parts of the objects on this level.
The level is not visible, and you cannot see objects or parts of the objects on this level.

Note: in-between levels can't be hidden individually. When an in-between level is hidden, the rest of levels above will be hidden automatically as well.

Plan View On/Off

The Plan View displays the 2D representation of any level, in the CPlane (construction plane) of the active viewport. This 2D representation is not editable and only shows the sectioned objects in the selected level, according to the Cut Plane position in that level.

The selected level 2D representation is visible in the CPlane of the active viewport.
The selected level 2D representation is not visible in the CPlane of the active viewport.

Set Current Floor

To set a specific level as the current floor in the active viewport, double click on level name. This operation will automatically put the CPlane onto the selected level and will turn this level on in case it was hidden.

Elevation

The Elevation value tells the base height of each building or level and the position of a Construction plane relative to each Building or Level. It determines which part of the model corresponds to each level. Elevation values on levels are relative to building elevation height. The building elevation value is measured along the z-axis of the document.


The elevation mark of current level appears highlighted.

Cut Plane height

The cut plane determines the elevation of the horizontal section of each floor; i.e., only the objects in that elevation or right under it are included in the representation.

NOTE: When a viewport is turned into a VisualARQ Plan view mode, the display mode is changed into "hidden" mode and changes the viewport for the Top view. As long as the Plan view mode is enabled, the viewport will show any selected display mode although you switch the level view.

Top offset

The top offset determines the value by which a level's upper limit is increased or decreased when hiding entire levels. This is especially useful when you hide the upper part of a building and want to hide also the upper slab of the last shown level. This value is irrelevant for the top level of a building and only affects viewports that have the Cut Plane off.

Bottom offset

The bottom offset determines the value by which a level's lower limit is increased or decreased when hiding entire levels. This is especially useful when you hide the lower part of a building and want to hide also the lower slab of the last shown level. This value is irrelevant for the bottom level of a building and only affects viewports that have the Cut Plane off.
