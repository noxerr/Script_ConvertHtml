Dynamic Section

Keyboard:
vaDynamicSection
Icon:
left click
Documentation toolbar:
Menu:
VisualARQ > Section > Dynamic Section

The vaDynamicSection command shows the 3D model sectioned in real time, according to the Section line position and length.

Steps:

  1. Select the Section line.

When the dynamic section is enabled, all the VisualARQ objects and the closed meshes and polysurfaces are capped at their sectioned areas. Curves, hatches and annotations are not affected by the Dynamic section tool.

Turn off the dynamic section:

In order to disable the dynamic section view, repeat the process of creating a dynamic section and select the same section line.


Model in Dynamic Section view, Section line and Section drawing

NOTE: Only objects contained within the section area delimited by its insertion points and its section length points will be displayed in the dynamic section.
