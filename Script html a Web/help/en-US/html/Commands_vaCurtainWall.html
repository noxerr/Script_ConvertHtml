Curtain Wall

Keyboard:
vaCurtainWall
Icon:
left click [right click]
Objects Toolbar:
Menu:
VisualARQ > Curtain Wall > Curtain Wall

After running the vaCurtainWall command, the curtain wall insert dialog box will appear. This dialog box shows a list of all curtain wall styles available, as well as the basic options of the curtain wall object and its preview.

Steps:

  1. Select the curtain wall style you wish to use and define the basic insertion options.
  2. Select the curtain wall start point (insertion point) with a left-click. When you specify the start point, a dynamic dimension will display the length value of the curtain wall being inserted in real time.
  3. Left-click again to specify the curtain wall end point.

The length and direction of the curtain wall can be specified as follows:

  • Entering the coordinates (relative and absolute) in the command line.
  • Entering the value to specify the length and/or angle in the command line.
  • Manually, by left-clicking the desired point.

Each new curtain wall will be drawn next to the previous one until you press ENTER, ESC or right click. The command immediately prompts the start point to start drawing the next curtain wall, but it does not link it to the previous curtain wall by default but rather starts drawing it on the new start point selected.

Curtain Wall: From Curves

Curtain walls can be also created from previously drawn curves in the model.

When you run the vaCurtainWall command, the curtain wall insert dialog box will appear.

Steps:

  1. Select FromCurves in the command line.
  2. Pick a Curve or multiple Curves from the model that will be used to generate curtain walls. The curves can be any type or shape, as long as they are flat. If a polyline curve is selected, a curtain wall object will be generated for each polyline segment.
  3. Select the curtain wall insertion options from the curtain wall insert dialog box that will define the curtain wall created from that curve. This step can be done before or after selecting the curve or curves to generate curtain walls.

The curtain walls created with this operation behave as all other VisualARQ curtain walls.


Control Points

Curtain walls have three control points and three control arrows:

  • The start and end control points can be used to stretch and/or change its direction, while the center control point is used to bend it in the case of curved curtain walls.
  • A control point situated at the top start of the curtain wall object is used to move its top horizontal edge and lean the curtain wall.
  • The control arrows that appear in the Plan view (as well as in the Perspective view) are used to stretch the curtain wall horizontally from its ends, while the control arrow that appears in all other views (Perspective, Front and Right) is used to stretch the curtain wall vertically, as required to modify its height manually.


Control points on Curtain walls

  1. Start control point (insertion point).
  2. Control arrow (vertical), (not editable in Top viewport).
  3. End control point.
  4. Control arrows (horizontal).
  5. Curtain Wall from curve control points.
  6. Start top control point.

Insertion Options and Parameters

Height

Even though Height is a curtain wall style property, you can assign a value that is different to the default height value in the insert dialog box. Simply clear the Height by style checkbox and choose the desired value, using the numerical height control.

To modify a curtain wall height, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in Rhino Properties panel ).

Alignment

There are three options to align the curtain wall, depending on its position based on the direction of its base curve:

  • Center
  • Right
  • Left
Alignment offset

Inserts the curtain wall at a specific distance from the direction of its base curve.

To modify a curtain wall alignment or the alignment offset, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in Rhino Properties panel ).
