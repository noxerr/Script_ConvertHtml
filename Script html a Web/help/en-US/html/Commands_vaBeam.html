VisualARQ > Beam > Beam

After running the vaBeam command, the beam insert dialog box will appear. This dialog box shows a list of all beam styles available, as well as the basic options and the preview of the beam object.

Steps:

  1. Select the Beam style you wish to use and define the basic insertion options.
  2. Select the Beam start point with a left-click on the desired point.
  3. Specify the Beam end point with a left-click on the desired point.

You can enter as many beams as you wish and you can cancel the command by pressing ENTER or ESC.

Beam From Curve

Beams can also be created from a curve in the model:

Steps:

  1. Start the vaBeam command to create a new beam. The Beam insert dialog box will appear.
  2. Select the Beam style you wish to use and define the basic insertion options.
  3. Select FromCurves in the command line and then pick a Curve from the model.
  4. Press ENTER or right click to finish the command.

Control Points

Beams can have different control points depending on how the beam has been created. These control points can be moved in any direcction, and modify the beam position and dimensions.

  • Straight beam: the beam has two control points, at the ends of the object.
  • Beam from curve: the beam has as many control points as the curve from which it was created.

Insertion Options and Parameters

Alignment

The beam alignment determines the position of the beam insertion point according to its profile shape.
Alignment options:

  • Top Left, Top Center and Top Right
  • Middle Left, Center and Middle Right
  • Bottom Left, Bottom Center and Bottom Right

To modify the alignment of a beam, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).

Offset

Shows the distance (offset) from the beam X and Y points to the insertion point.

To modify a beam offset, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).

Elevation

A beam elevation is always measured from the baseline of the level it is inserted in. The elevation is not a style property. The value is specified for each beam object in the insert dialog box when a new beam is inserted.
The default elevation value for beams is 3 meters. If you modify this value, the Beam dialog box will show the last value used.

Roll

This value determines the angle by which the beam rotates around its axis, taking as the center of rotation its alignment position.

The beam roll can only be edited in the VisualARQ Properties section (in the Rhino Properties Panel ).

Mirrored

When enabled this option changes the beam orientation into its mirror image.

This option can only be edited in the VisualARQ Properties section (in the Rhino Properties Panel ).
