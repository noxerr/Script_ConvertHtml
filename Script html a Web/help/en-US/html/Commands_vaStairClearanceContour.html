Stair Clearance Contour

Keyboard:
vaStairClearanceContour
Menu:
VisualARQ > Stair > Clearance Contour

Creates a contour curve that can be used to calculate the profile to subtract on a slab to maintain the specified minimum stair clearance height.

When you run the vaStairClearanceContour command, follow the steps from the command line. Press the right mouse button or ENTER to accept and go to the next step.

Steps:

  1. Select the stair you wish to create the profile from.
  2. Select the stair clearance height.
  3. Select the thickness of the slab you wish to maintain the minimum stair clearance height from.

The curve profile will be created on top of the stair height.


h: Stair clearance height
