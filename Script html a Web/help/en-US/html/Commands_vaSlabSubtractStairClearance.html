Slab: Subtract Stair Clearance

Keyboard:
vaSlabSubtractStairClearance
Menu:
VisualARQ > Slab > Subtract Stair Clearance

Creates the required hole in a slab to maintain the specified minimum clearance height between the stair path and the slab bottom surface.

When you run the vaSlabSubtractStairClearance command, follow the steps from the command line. Press the right mouse button or ENTER to accept and go to the next step.

Steps:

  1. Select the slab you wish to subtract the hole from.
  2. Select the stair you wish to get the boundary from in order to maintain the specified minimum clearance height.
  3. Select the stair clearance height.

The slab subtracts the silhouette of the stair boundary until there is a minimum clearance height between the stair path and the slab bottom surface.


h: Stair clearance height
