Table

Keyboard:
vaTable
Icon:
left click [right click]
Documentation toolbar:
Menu:
VisualARQ > Table > Table

The table object displays information about objects of the same type.

When you run the vaTable command, the table insert dialog box will appear. This box shows the table styles available. The list includes the Openings, Spaces and Walls table styles by default, but new table styles can be created to quantify any VisualARQ object.

Steps:

  1. Select the table style you wish to use.
  2. Select all the objects you wish to quantify in the table. The command filters the selection automatically, detecting only the type of object corresponding to the table style selected.
  3. Choose an insertion point by clicking anywhere on the screen to insert the table.

The command ends when the table quantifying the selected objects is inserted.


Table insert dialog

Control Points

Tables have one control point in the top left corner.

Insertion Options and Parameters

The Table insert dialog box does NOT have insertion parameters. In this box, you can only select the Style of the table being inserted.

The parameters that define the features of a table correspond to a table style.

Export Table to Excel

Once they have been inserted in the model, tables can be exported to Excel and in .csv format with the vaTableExport command.
