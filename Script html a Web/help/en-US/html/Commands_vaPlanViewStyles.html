Plan View Styles

Keyboard:
vaPlanViewStyles
Icon:
right click [left click]
Documentation toolbar:
Menu:
VisualARQ > Plan View > Plan View Styles
Shortcut key:
Select a plan view object > (Alt, Shift or Ctrl) + double left click

The plan view style defines the attributes of plan views.

New Plan view style:

You can create new styles and duplicate existing styles from the Plan View Style Manager dialog box.

Delete a Plan view style:

You can delete Plan view styles from the Plan view Style Manager dialog box. You can only delete those styles that are not being used by any existing Plan view in the model.

Edit a Plan view style:

Select the plan view style at the top left panel, and edit their different parameters from the tabs that appear in the right panel.


Plan view styles dialog

General style editing
You can use the vaPlanViewStyles command or, alternatively,
  • Use the vaStyleProperties command, then select a plan view and press Enter, or
  • Right click on , then select a plan view and press Enter, or
  • Select a plan view and open the context menu by pressing the right mouse button for a while then select Plan View> Style Properties, or
  • Press Alt, Shift or Ctrl then double click on a plan view object.
The changes applied to a style are applied to all elements with that style.
Object-specific editing
  • Use the vaProperties command, then select a plan view and press Enter, or
  • Click on , then select a plan view and press Enter, or
  • Select a plan view and open the context menu by pressing the right mouse button for a while then select Plan View > Properties, or,
  • Double click on a plan view object.
The changes are applied only to the style of the selected element.

Attributes

Attributes define the features of each plan view style for display and print purposes. The valid attributes are:

  • Plan view: Layer and Visibility.
  • Plan View boundary curve: Display color, Linetype, Print width and Print color.
  • Plan View text: Text style.
