Railing

Keyboard:
vaRailing
Icon:
left click [right click]
Objects Toolbar:
Menu:
VisualARQ > Railing > Railing

After running the vaRailing command, the railing insert dialog box will appear. This dialog box shows a list of all railing styles available, as well as the basic options and the preview of the railing object.

Steps:

  1. Select the Railing style you wish to use and define the basic insertion options.
  2. Select any number of points throughout the total length of the railing object.
  3. Press ENTER or right click to finish the railing command.

You can enter as many railings as you wish and you can cancel the command by pressing ENTER or ESC.

Railing from Curve

Railings can also be created from a curve in the model:

Steps:

  1. Select the Railing style you wish to use and define the basic insertion options.
  2. Select FromCurves in the command line then pick one Curve or multiple Curves from the model. The curves can be any type or shape of curve, as long as they are open.
  3. Press enter or right click to finish the command.

Railing created from a curve and its control points.

Control Points

Railings can have different control points depending on how the railing has been created: these control points can be moved in any direction, and modify the railing position and dimensions.

  • Straight railing: the railing has two control points, at the edges of the object.
  • Railing from curve: the railing has as many control points as the curve from which it was created.

Insertion Options and Parameters

Alignment

Railings can be aligned according to three rules, depending on the direction of the base curve:

  • Center
  • Right
  • Left

To modify the alignment of a Railing, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).

Alignment offset

Inserts the railing at a specific distance from the direction of its base curve.

To modify a Railing offset, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).

Elevation

A railing elevation is always measured in relation to the baseline of the level it is inserted in. The elevation is not a style property. The value is specified for each railing object in the insert dialog box when a new railing is inserted.
The default elevation value for railings is 0. If you modify this value, the insert dialog box will show the last value used.

Height

Even though the height is a property of the railing style, you can assign a value that is different to the default height value in the insert dialog box. Simply clear the Height by style checkbox and choose the desired value, using the numerical height control.

This value can be edited also in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).
