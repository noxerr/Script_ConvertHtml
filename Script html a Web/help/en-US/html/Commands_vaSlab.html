Slabs are defined by a boundary curve. They can be created as rectangular shapes (by default) or from any existing curve in the model.

After running the vaSlab command, the slab insert dialog box will appear. This dialog box shows a list of all slab styles available, as well as the basic options and the preview of the slab object.

Follow the steps from the command line. Press the right mouse button out of the insert dialog box or hit ENTER to end the command.

Rectangular Slab

Steps:

  1. Select the Slab style you wish to use and define the basic insertion options.
  2. Pick the two opposite corners that will define the slab.

Slab: From Curves

Slabs can be also created from previously drawn curves in the model.

Steps:

  1. Select the Slab style you wish to use and define the basic insertion options from the Insert Dialog.
  2. Pick a curve or multiple curves that will be used as the slabs start boundary. These curves must be planar and closed.

Slabs can be also be created from curves after running the vaSlab command, and selecting the FromCurves option in the command line.

Control Points

Slabs have as many control points as the original curve they were created from. You can select any of these points and change the slab boundary position.

Insertion Options and Parameters

Alignment

The alignment defines the position of the slab relative to its boundary curve. There are three options available:

Top: the slab is aligned with the boundary curve from its top surface.
Center: the slab is aligned with the boundary curve from the middle point of the slab thickness.
Bottom: the slab is aligned with the boundary curve from its bottom surface.

To modify the slab alignment edit it in the Object Properties dialog box or in the VisualARQ Properties section (in Rhino Properties Panel ).

Thickness

The slab thickness is a property defined by the slab style, but it can be edited individually for each one of the slab layers.

To modify the slab thickness edit its values in the Object Properties dialog box or in the VisualARQ Properties section (in Rhino Properties Panel ).
