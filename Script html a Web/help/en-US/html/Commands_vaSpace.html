Space

Keyboard:
vaSpace
Icon:
left click [right click]
Documentation toolbar:
Menu:
VisualARQ > Space >
  • From Point
  • From Curve
  • From Planar Surface

    • The space object represents the surface entity that displays the information (name, area, perimeter, etc.) of a space or room with a hatch pattern and a label.

    A room is defined as any closed space delimited by walls and/or columns on all four sides. Walls and columns can be at different elevations, provided that they are part of the same floor.

    Steps:

    1. There are three options to create a space. These options display in the command line when you run the vaSpace command:
      • Space made up of walls and/or columns (room): Left click inside the room for which you wish to calculate the area.
      • Space defined by a curve: Click on the FromCurve command line option. Select a closed and planar curve.
      • Space defined by a surface: Click on the FromPlanarSurface command line option. Select a planar surface.
    2. The New Space dialog box will display. Enter the information for the selected space:
      • Name
      • Area type: Usable or Built
      • Style

    Space insert dialog box

    A hatch pattern is inserted in the room marking the occupied area, with a label containing the space name and its geometric features.

    You can create as many spaces as you wish and you can cancel the command by pressing ENTER or ESC.

    Modify a space

    Keyboard:
    vaSpaceSetContour

    A space can be modified with the vaSpaceSetContour command in three different ways:

    1. Select the space you wish to modify.
    2. Select a new point (inside a room) to insert the space, or select one of the command options displayed in the command line: FromCurve or FromPlanarSurface.

    The space is modified while maintaining its properties.

    NOTE: The spaces made up of walls and/or columns can be moved manually with the cursor inside other rooms. They can also be updated after making a change in their original walls and/or columns with the vaUpdate command .

    Control points

    Spaces have two control points:

    1. Insertion point for the hatch pattern that marks the space.
    2. Insertion point for the label.

    NOTE: All space control points are used to move it (they do not scale it).

    To move a space to a new room, select the space (hatch pattern and label) and drag it to the new room.

    To move the space label, enable the space control points and move the label control point to the desired point.

    Insertion Options and Parameters

    The Space insert dialog box includes the options to choose the space: Name, Area type and Style.

    The parameters that define the features of a space correspond to a Space style, although they can be modified regardless of the style in the Object Properties dialog box or in the VisualARQ Properties section (Rhino Properties Panel ).

    Name

    The name of a space is used to distinguish it from all others and state its functionality.

    To modify the name of a space, enter the object properties dialog box of the space and left click on the name.

    Area type

    There are two types of space areas:

    • Usable: Area occupied by a room (or building), excluding the thickness of walls.
    • Built: Area delimited by the outer limits of the plan construction.

    To modify the area type of a space, delete the space and create a new one, selecting the desired area.

    Height

    The space object has a height value which determines the volume of the room. This value can be changed in the VisualARQ Properties section (Rhino Properties Panel ).
