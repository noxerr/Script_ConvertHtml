Wall: Set Path

Keyboard:
vaWallSetPath
Icon:
right click [left click]
Walls toolbar:
Menu:
VisualARQ > Wall > Set Path

Option to edit a wall that has already been created, adjusting it to a selected curve path.

When you run the vaWallSetPath command, follow the steps from the command line. Press the right mouse button or ENTER to accept and go to the next step.

Steps:

  1. Select a wall path you wish to modify.
  2. Select the curve that will define the new wall path. It can be any type or shape of curve, as long as it is planar and open.

When you select the new curve, the wall will match its shape and position.

In the case of walls with doors and/or windows, these will be moved with the wall.
