Object Properties panel (in Rhino Properties panel)


VisualARQ Properties section for the Column object,
in Rhino Properties dialog box.

In addition to the Object Properties dialog box described above, you can edit the properties of a VisualARQ object from the VisualARQ Properties section. (also available from the Rhino menu Edit > Object Properties...
This section is integrated in the Rhino Properties Panel, but it is only displayed when you select a VisualARQ object.

The parameters available in this section are not the same for all VisualARQ objects. This example shows the parameters of the Column object that can be edited.

You can edit the parameters of the selected VisualARQ object that do not depend on a specific style from the VisualARQ Properties section.

The parameters available in this section depend on the type of object selected and are the same as those in the Object Properties dialog box.


General
  • Name: name of the selected object.
  • Description: description of the selected object.
  • Style: list of available object styles from the VisualARQ object libraries plus the new styles created by the user.
  • Serial Number: serial number of the selected object.
  • Tag: reference name that appears in tag objects.
Display
  • Isocurve Density: Defines the number of isoparametric curves that Rhino draws on the surface.
  • Plan Visibility: Hides or displays an object in a Plan view based on its location regarding the cutting plane inside a level.
    • Hidden: The object won't be displayed regardless of its size and position.
    • In Cut Plane: The object will be displayed only when it is located within the cutting plane
    • Below Cutting Plane: The object will be displayed when it is located within the cutting plane or below it.
    • Above Cutting Plane: The object will be displayed when it is located within the cutting plane or above it.
    • Inside Level: The object will be always displayed while it is located inside a level, regardless of its size and position.


    Plan Visibility scheme.


    Example of which windows from the above image would be displayed depending on the Plan visibility option selected:

    • Hidden: None.
    • In Cut Plane: Window 2.
    • Below Cutting Plane: Windows 2 and 3.
    • Above Cutting Plane: Windows 1 and 2.
    • Inside level: Windows 1, 2 and 3.

Location
  • Insert Point: Coordinates of an object insert point.
  • Insert Mark (only for columns): Enables the display of the insert mark and its type.
Geometry
  • Height: Specifies the object height. This value can be changed regardless of the object style. In walls and columns, this height can be also defined by style.
  • Profile: Description of the object profile.
  • Width/Height: The object dimensions.
  • Cut Depth (option only available for doors, windows and openings): Specifies the maximum depth distance an opening profile will cut a wall.
Opening

(Option available for doors and windows)

  • Aperture (%): The aperture percentage defines how open or closed doors and windows leaves are displayed in the Plan and Perspective views.
  • Opening Side: Direction in which the door or window leave will open.
Profile

(Option available for beams and columns)

  • Roll/Rotation: Rotation of the object profile.
  • Alignment: Insert position of the object according to its profile.
  • Alignment Offset: Distance from the alignment position.

The More... button

At the bottom of the dialog there is the More... button from which you can quickly access to some shortcuts related to the selected objects. These options are similar to those that can be found in the Context Menu after selecting an object.

  • Delete
  • Duplicate
  • Isolate Objects
  • Hide Objects
  • Lock Objects
  • Control Points On
  • Reverse (only for walls)
  • Select
  • Update
  • Style Properties
  • Properties
