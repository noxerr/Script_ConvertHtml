Table Export

Keyboard:
vaTableExport
Menu:
VisualARQ > Schedule Table > Export
Context Menu:
Schedule table > Export table

Once they have been inserted in the model, tables can be exported to Excel (.xls) and in .csv format.

Steps:

  1. Select the schedule table you wish to export.
  2. Select the folder and file format you wish to save your table as: .xls or .csv.

A new file is created in your selected destination folder. This file doesn't update automatically when applying changes to the Rhino-VisualARQ table. To have the latest updated version, just repeat the steps above and overwrite the former file.
