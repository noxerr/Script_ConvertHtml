Tag Styles

Keyboard:
vaTagStyles
Icon:
right click [left click]
Documentation toolbar:
Menu:
VisualARQ > Tag > Tag Styles
Shortcut key:
Select a tag object > (Alt, Shift or Ctrl) + double left click

The tag style defines the attributes of tags.

New tag style:

You can create new styles and duplicate existing styles from the tag Style Manager dialog box.

Delete a tag style:

You can delete tag styles from the tag Style Manager dialog box. You can only delete those styles that are not being used by any existing tag in the model.

Edit a tag style:

Select the tag style at the top left panel, and edit their different parameters from the tabs that appear in the right panel.



Tag styles dialog

General style editing
You can use the vaTagStyles command or, alternatively,
  • Use the vaStyleProperties command, then select a tag and press Enter, or
  • Right click on , then select a tag and press Enter, or
  • Select a tag and open the context menu by pressing the right mouse button for a while then select Tag > Style Properties, or
  • Press Alt, Shift or Ctrl then double click on a tag object.
The changes applied to a style are applied to all elements with that style.
Object-specific editing
  • Use the vaProperties command, then select a tag and press Enter, or
  • Click on , then select a tag and press Enter, or
  • Select a tag and open the context menu by pressing the right mouse button for a while then select Tag > Properties, or,
  • Double click on a tag object.
The changes are applied only to the style of the selected element.

Attributes

Attributes define the features of each tag style for display and print purposes. The valid attributes are:

  • Layer, Visibility, Text style, Dimension style, Color, Linetype, Line print width and Line print color.
