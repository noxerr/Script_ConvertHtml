Profile: From Curve

Keyboard:
vaProfileFromCurve
Icon:
left click
Tools toolbar:
Menu:
VisualARQ > Profile > From curve

The vaProfileFromCurve command can be used to create custom profiles from planar and closed curves. These profiles can be used to create custom column, beam, door, window and opening styles as well as railing components shapes.

Steps:

  1. Select a flat and closed curve to define the profile shape.
  2. Type the Profile name in the command line

Once you have created a custom profile, it will appear at the end of the list of the parametric profiles library for columns, beams, doors and windows.
The resulting profile is not parametric, so it can only be used with the dimension and proportions of the original drawing.
