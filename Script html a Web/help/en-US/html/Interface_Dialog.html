Insert Dialog Boxes

The insert dialog boxes show the options for objects that are being inserted. They are only visible while a command is running.

The option panels can be expanded (displayed) or collapsed (hidden) by clicking on the small arrows located at the corners.


Insert dialog box for the Wall object.

The insert dialog boxes are divided in three or more sections:

  • The top section shows a list with the object styles available.
  • The middle section(s) show the basic object insertion option.
  • The lower section shows a preview -in plan view and perspective view- of the object to insert. No preview is displayed when this section is collapsed so that the program does not have to calculate it.

NOTE: Slab and Roof objects don't use the insert dialog box.
