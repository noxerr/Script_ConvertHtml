Stair

Keyboard:
vaStair
Icon:
left click [right click]
Toolbar:
Menu:
VisualARQ > Stair > Stair

After running the vaStair command, the stair insert dialog box will appear. This dialog box shows a list of all stair styles available, as well as the basic options of the stair object and its preview.

Two minimum points (insertion and end point) are required to create a stair.

Steps:

  1. Select the Stair style you wish to use and define the basic insertion options.
  2. Insert the stair by selecting an insertion point.
  3. Select any number of points throughout the total length of the stair. Each point determines a new stair flight and its direction. The length value depends on the parameters previously defined (Height, Tread, Tread count).
  4. Press ENTER or right click button to finish the stair command.

The command creates a stair flight that can change its direction or remain straight on each point selected. These flights are joined with landings.


Stair insert dialog box

Spiral Stair

Select the Spiral stair option at the bottom part of the stair insert dialog box to display a special section, which will be used to define the parameters of a spiral stair:

  • Turning angle
  • Radius
  • Clockwise: determines the direction of the spiral stair

This option cannot be edited once the stair has been inserted. To change a stair with straight section(s) to a spiral stair, insert the stair again, selecting the desired option.

NOTE: spiral stairs can't have a turning angle superior to 360º. To create a spiral stair that turns over itself more than 360º, you need to create two or more spiral stairs.

Control Points

Stairs have as many control points as points inserted to define the stair:

  1. Insert point (stair start)
  2. Start of the second flight
  3. Start of the third flight
  4. End of the stair

All stair control points are used to move the stair. They do not scale it.

To define the direction of each stair flight, select and move the desired point on the screen or enter an angle in the command line.


Insertion Options and Parameters

Most parameters in a stair are specified in the insert dialog box when the stair is inserted. These parameters can be edited afterwards in the Object Properties dialog box or in the VisualARQ Properties section (Rhino Properties Panel ).

Alignment

The stair can be aligned with 3 methods:

  • Center
  • Right
  • Left
Height

The Height value corresponds to the distance between the stair starting point level and the stair ending point level.

Width

The Width value remains constant for all flights throughout the stair.

Tread count

Specifies the total number of treads throughout a stair.

Tread

The Tread value corresponds to the depth of each step. It remains constant for all steps throughout the stair.

Riser

The Riser value corresponds to the height of each step. It is calculated automatically with the Height and the Tread count and it remains constant for all steps throughout the stair.

Stair slab Top and Bottom thickness:

These values adjust the stair slab thickness at its start and ending point in order to fit with slabs thickness.

Stair flight and landing parameters:

Unlike other VisualARQ objects, the stair has some object properties which can only be assigned after the stair has been inserted.
These properties are the Stair flight and Stair landing parameters and can only be edited in the Object Properties dialog box :


Stair dialog box (Object Properties)

Flight Parameters

  • Type
    • Landing: Indicates that the flight consists of a flat slab or landing.
    • Steps: Indicates that the flight consists of steps.

Landing Parameters

  • Type
    • Landing: Indicates that the landing consists of a flat slab or landing.
    • Steps: Indicates that the landing consists of steps.
  • Boundary
    • Interior
      • Straight: Indicates that the interior boundary is straight (angular).
      • Curved: Indicates that the interior boundary is curved (arc shaped).
    • Exterior
      • Straight: Indicates that the exterior boundary is straight (angular).
      • Curved: Indicates that the exterior boundary is curved (arc shaped)
