IFC Tag

The Ifc Tag panel is used to assign properties to the objects when they are exported to IFC. This panel is located under the icon, in the Rhino Properties Panel , after selecting one or more objects that are not VisualARQ objects.


IFC properties panel

Type

VisualARQ objects:

VisualARQ objects already have an IFC type assigned by default, according to VisualARQ object type, when they are exported to IFC:

  • Wall > IfcWall
  • Curtain wall > IfcCurtainWall
  • Beam > IfcBeam
  • Column > IfcColumn
  • Door > IfcDoor
  • Window > IfcWindow
  • Stair > IfcStair
  • Railing > IfcRailing
  • Slab > IfcSlab
  • Roof > IfcRoof
  • Space > IfcSpace

    • Rhino 3D geometry:

    All Rhino geometry, (non-VisualARQ objects), has its IFC Type Property assigned "By Layer" by default.

    The IFC type properties can be assigned by layer from the IfcExportOptionsDialog command.

    • By Layer: the IFC Property is assigned by layer.
    • No export: the object won't be exported to IFC.
    • Ifc Proxy: the objects will be exported to IFC as Proxies or generic geometry.
