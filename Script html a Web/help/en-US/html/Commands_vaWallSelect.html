Wall: Select

Keyboard:
vaWallSelect
Menu:
VisualARQ > Wall > Select.
  • Connected walls
  • Contained doors
  • Contained windows

Command to select walls and its contained openings in different ways.

When you run the vaWallSelect command, a list of options will appear in the command line.

  • All: Every wall in the model will be selected.
  • Connected walls: Pick up a wall from the model and all the connected walls will be selected.
  • ContainedDoors: Pick up a wall from the model and all the doors contained in this wall will be selected.
  • ContainedWindows: Pick up a wall from the model and all the windows contained in this wall will be selected.
