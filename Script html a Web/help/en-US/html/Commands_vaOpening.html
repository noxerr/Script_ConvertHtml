Opening

Keyboard:
vaOpening
Menu:
VisualARQ > Opening > Opening

The operation and behavior of an opening is identical to the one of a window or a door.
Openings can be inserted in a wall or a curtain wall object or as floating objects. In the first case both elements are anchored and the opening is generated automatically in the wall/curtain wall where it is inserted. Openings can be detached from walls/curtain walls by moving them outside of their surface limits.

After running the vaOpening command, the opening insert dialog box will appear. This dialog box shows a list of all opening styles available, as well as the basic options and the preview of the opening object.

The steps required to insert an opening depend on the insertion options selected.

Steps:

  1. Select the Opening style you wish to use and define the basic insertion options.
  2. Select the Opening insertion point with a left-click on the desired point in the model. If that point is near to a wall/curtain wall path, the opening will be automatically inserted in that wall/curtain wall. In that case there are two options available:
    • Offset: insert the opening at a specific distance (offset), from the end nearest to the wall/curtain wall where the opening is inserted. Specify the opening side by moving the cursor; the command calculates it automatically with the cursor's position. Click to insert the opening.
    • Horizontal alignment: Insert the opening freely in any part of the wall/curtain wall. Click to insert the opening in the desired point.
  3. Specify the opening orientation manually with the cursor. This step is not required when the opening has detected a wall or a curtain wall.
  4. Specify the Opening side by clicking on the desired side of the wall/curtain wall or the opening position. This step is not required when you insert an opening with the offset option.

You can enter as many openings as you wish and you can cancel the command by pressing ENTER or ESC.

In the two insertion options, the opening is automatically generated in the wall/curtain wall as soon as it detects the wall/curtain wall path.

Note: the opening will not be invisible nor selectable in the model if its style has no component and it has been inserted without being attached to a wall/curtain wall.

Opening from curve

Openings can be created from curves, which determine their shape profile.

Steps:

  1. Run the vaOpening command.
  2. Select the Opening style you wish to use and define the basic insertion options.
  3. Select the FromCurves option in the command line.
  4. Select a curve (or several curves) in the model. This curve must be closed and vertical.

The opening will be generated in the selected curve and will only be inserted in a wall/curtain wall if it is close enough.

Note: The curves selected to create openings does not need to be oriented parallel to the wall/curtain wall where the openings will be placed. Openings are inserted and oriented automatically according to the closest wall/curtain wall position.

Control points

Openings have five control points:

  1. Control point for the opening side
  2. Left insert
  3. Center insert
  4. Right insert
  5. Reference mark control point

To move the opening (either in the wall/curtain wall or to a different wall/curtain wall), select it from any point and drag it to the desired location.

Openings can only be selected from the VisualARQ Top view.

Insertion options and parameters

Alignment

The opening can be aligned with three methods within the wall/curtain wall thickness:

  • Interior
  • Center
  • Exterior

To modify the alignment of an opening, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).

Vertical alignment

Openings can be inserted in a wall/curtain wall from two different vertical alignments in relation with the base line of the wall/curtain wall:

  • Bottom: reference distance between the base line of the wall/curtain wall and the bottom surface of the opening.
  • Top: reference distance between the base line of the wall/curtain wall and the top surface of the opening.

The vertical alignment is not a style property. The value is specified for each object in the insert dialog box when each object is inserted.


To modify an opening vertical alignment, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).

Elevation

The opening elevation in the insert dialog is measured in relation to the baseline of the wall/curtain wall it is inserted in, regardless of its level. After the opening has been inserted, its elevation value indicates the opening position in the world Z coordinates, regardless of the wall/curtain wall and the level where it is inserted. The elevation is not a style property. The value is specified for each object in the insert dialog box when each object is inserted.
The default elevation value for openings is 0. If you modify this value, the insert dialog box will show the last value used.

To modify an opening elevation, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).
