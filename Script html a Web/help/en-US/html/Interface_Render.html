Render

VisualARQ's material manager supports materials of all render engines that are compatible with Rhino. VisualARQ object material assignment works as in Rhino.

Material assignment to the different components of a VisualARQ object is also possible by editing the component Material from the Style properties dialog.

List of Animation and Rendering products related to Rhino:

  • Air Rendering Software: Photoreal + illustration rendering + IpR (plugin)
  • Bongo: Animation for Designers (plugin)
  • Brazil for Rhino: Rendering tools for professionals (plugin)
  • Flamingo nXt: Raytrace and radiosity rendering (plugin)
  • Hayabusa 2.0 for Rhino: Real Time High quality renderer for Rhinoceros (companion with 3DM Support)
  • KeyShot: Realtime Raytracing and Global Illumination Application (companion with 3DM Support)
  • Maxwell Render, the light simulator: Maxwell Render plugin for Rhino (plugin)
  • NuGraf for large CAD rendering: Multi-threaded ray tracing & 3D conversion for Rhino (plugin)
  • Penguin: Conceptual, non-photorealistic renderer (plugin)
  • Renderosity: Rhino online discussion forum (useful link)
  • RhinoMan: RenderMan compatible rendering plug-in (plugin)
  • V-Ray for Rhino: Advanced global illumination renderer (plugin)
  • VSR Realtime Renderer: Fully integrated Realtime Renderer, available as Rhino display mode (plugin)
