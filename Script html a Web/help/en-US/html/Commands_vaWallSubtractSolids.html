Wall: Subtract Solids

Keyboard:
vaWallSubtractSolids
Icon:
left click
Walls toolbar:
Menu:
VisualARQ > Wall > Subtract Solids

Option to subtract one or more solid objects from a wall.

When you run the vaWallSubtractSolids command, follow the steps from the command line. Press the right mouse button or ENTER to accept and go to the next step.

Steps:

  1. Select the wall from which you wish to subtract one or more solid objects.
  2. Select the solids you wish to subtract. These solids must be closed extrusions or polysurfaces and must intersect with one or more points on the wall where they will be subtracted.

The solid object volume is removed from the wall body. The result can be displayed in any view.

The wall features are not affected by this operation, so you can subtract doors and windows from the modified wall.
