Wall

Keyboard:
vaWall
Icon:
left click [right click, long click]
Toolbar:
Menu:
VisualARQ > Wall > Wall

Walls are fundamental elements in VisualARQ.

After running the vaWall command, the wall insert dialog box will appear. This dialog box shows a list of all wall styles available, as well as the basic options of the wall object and its preview.

Steps:

  1. Select the wall style you wish to use and define the basic insertion options.
  2. Select the wall start point (insertion point) with a left-click. When you specify the start point, a dynamic dimension will display the length value of the wall being inserted in real time.
  3. Left-click again to specify the wall end point.

The length and direction of the wall can be specified as follows:

  • Entering the coordinates (relative and absolute) in the command line.
  • Entering the value to specify the length and/or angle in the command line.
  • Manually, by left-clicking the desired point.

Each new wall will be drawn next to the previous one until you press ENTER, ESC or right click. The command immediately prompts the start point to start drawing the next wall, but it does not link it to the previous wall by default but rather starts drawing it on the new start point selected.

Wall: From Curves

Icon:
left click [right click]
Walls toolbar:
Menu:
VisualARQ > Wall > From Curves

Option to create walls from previously drawn curves, following its boundary.

When you run the vaWall > FromCurves command, the wall insert dialog box will appear. Follow the steps from the command line. Press the right mouse button out of the wall insert dialog box or hit ENTER to end the command.

Steps:

  1. Select a Curve or multiple Curves that will be used to generate walls. The curves can be any type or shape of curve, as long as they are flat.
  2. Select the wall insertion options from the wall insert dialog box that will define the wall created from that curve. This step can be done before or after selecting the curve or curves to generate walls.

The walls created with this operation behave as all other VisualARQ walls. They can be edited (by object or style) and you can insert doors and windows in them.

Walls created with curves are not only parametric but you can also change their path manually. To do so, VisualARQ offers two options:

  • Deform the wall path using its own control points.
  • Deform the wall using the vaWallSetPath.


Control Points

The walls have three control points and three control arrows:

  • The start and end control points can be used to stretch and/or change its direction, while the center control point is used to bend it in the case of curved walls.
  • The control arrows that appear in the Plan view are used to stretch the wall horizontally from its ends, while the control arrow that appears in all other views (Perspective, Front and Right) is used to stretch the wall vertically, as required to modify its height manually.


Control points on walls

  1. Start control point (insertion point).
  2. Control arrow (vertical), (not editable in Top viewport).
  3. End control point.
  4. Control arrows (horizontal).
  5. Wall from curve control points.

Insertion Options and Parameters

Height

Even though Height is a wall style property, you can assign a value that is different to the default height value in the insert dialog box. Simply clear the Height by style checkbox and choose the desired value, using the numerical height control.

To modify a wall height, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in Rhino Properties Panel ).

Alignment

There are three options to align the wall, depending on its position based on the direction of its base curve:

  • Center
  • Right
  • Left
Alignment offset

Inserts the wall at a specific distance from the direction of its base curve.

To modify a wall alignment or the alignment offset, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in Rhino Properties Panel ).

Wall intersections:

The Intersections between walls (and their layers) are resolved automatically. A wall is extended to a nearby wall by default when it is within its cleanup radius.

  • The cleanup radius specifies the maximum distance at which a wall baseline can be from another wall baseline when joining the two walls.
  • The maximum extension specifies the maximum distance by which one wall extends from its ends points to the point where it intersects with another wall.

To modify a wall cleanup radius and the maximum extension, edit their values in the Object Properties dialog box or in the VisualARQ Properties section (in Rhino Properties Panel ).
