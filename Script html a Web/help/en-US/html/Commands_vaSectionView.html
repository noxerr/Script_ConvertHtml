Section View

Keyboard:
vaSectionView
Icon:
left click [right click]
Documentation toolbar:
Menu:
VisualARQ > Section > Section View

When you run the vaSectionView command, you are creating a 2D drawing that represents the 3D objects projected on a plane. This section view is created according to the section mark position in the model. After running the command, the Section View insert dialog box will appear. Follow the steps from the command line. Press the right mouse button or ENTER to accept and go to the next step.


Section View insert dialog

Steps:

  1. Specify a section view style from the insert dialog box.
  2. Select the section mark.
  3. Specify an insertion point. Left click anywhere on the screen to insert the section drawing.

This section drawing keeps attached to the section mark it was created from, as long as you don't explode it. If the section mark is deleted, the section view is also deleted. When you move the section mark position, the section drawing updates automatically. The section drawing also mantains the layer properties of the objects it represents.

Section View parameters

The following parameters can be defined for each Section View object, after they have been created. They can be edited in the VisualARQ Properties section (in the Rhino Properties Panel ).
  • Style: the style defines the attributes of section views
  • Title: the section view title takes by default the corresponding section mark reference text and adds the same text with a " ' ". (i.e. Section reference text = A, Section View title = A-A'). However the section view title can be changed by any other text.
  • Section: option to change the section view by selecting a different section mark.
  • Auto Update: option to enable an automatic update of the Section View when its corresponding section mark object is moved or edited.
  • Tolerance factor: this value determines the level of detail the section view is generated with. The higher this value is, the less detailed the section view will be.

Model in dynamic section view, section mark and section view

Note 1: The section view does not show the hidden faces.

Note 2: The section view and its corresponding section mark are created in the layer that is on when you run the command. Deleting one automatically deletes the other.

Note 3: Only objects contained within the section area delimited by its insertion points and its section lenght points will be displayed in the section view.
