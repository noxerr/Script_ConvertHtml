Curtain Wall Styles

Keyboard:
vaCurtainWallStyles
Icon:
right click [left click]
Objects toolbar:
Menu:
VisualARQ > Curtain Wall > Curtain Wall Styles
Context Menu:
Curtain Wall > Styles Properties
Shortcut key:
Select a curtain wall object > (Alt, Shift or Ctrl) + double click

The curtain wall style defines the curtain wall features. The following parameters can be specified for each style: Name, Attributes, Panel and Location.

The curtain wall styles are defined by different components that can be added or deleted at any time: Cell, Frame, Mullions and Supports.


Different parts and components of a Curtain Wall:
  1. Cell
  2. Mullion
  3. Support (vertical)
  4. Frame

New curtain wall style:

You can create new styles and duplicate existing styles from the curtain wall Style Manager dialog box.

Delete a curtain wall style:

You can delete curtain wall styles from the curtain wall Style Manager dialog box. You can only delete those styles that are not being used by any existing curtain wall in the model.

Edit a curtain wall style:

Select the curtain wall style or the curtain wall component item at the top left panel, and edit their different parameters from the tabs that appear in the right panel.

Add a new curtain wall component:

  • Context menu: do right-click on the curtain wall style name to open the context menu, and select New > "Component item".
  • New icon : select the curtain wall style and click on the New icon, from the dialog toolbar. Select "Component item".

Delete components: use the Context menu (righ-click on the component to delete and select Delete) or the Delete button .


Style Manager dialog box for the curtain wall object

General style editing
You can use the vaCurtainWallStyles command or, alternatively,
  • Use the vaStyleProperties command, then select a curtain wall and press Enter, or
  • Right click on , then select a curtain wall and press Enter, or
  • Select a curtain wall and open the context menu by pressing the right mouse button for a while, then select Curtain Wall > Style Properties, or
  • Press Alt, Shift or Ctrl, then double click on a curtain wall object.
The changes applied to a style are applied to all elements with that style.
Object-specific editing
  • Use the vaProperties command, then select a curtain wall and press Enter, or
  • Click on , then select a curtain wall and press Enter, or
  • Select a curtain wall and open the context menu by pressing the right mouse button for a while, then select Curtain Wall > Properties, or,
  • Double click on a curtain wall object.
The changes are applied only to the style of the selected element.

Attributes

Attributes define the features of each curtain wall style and its components for display and print purposes. The valid attributes are:

  • General: Layer, Visibility and Material.
  • Projection: Color, Linetype, Line print width and Line print color.
  • Section: Color, Linetype, Line print width, Line print color, Pattern, Pattern scale, Pattern angle and Pattern color.

Panel

This option determines whether the curtain wall panels will be curved or flat, when the curtain wall is created from an arch or a non straight curve.

Location

Curtain wall panel position relative to Frame. There are three alignment options available: Interior, Center and Exterior.

Curtain wall components

Cell

The Cell component determines the curtain wall design and how the panels will be distributed along the curtain wall area. It is defined by the following parameters:

Division

The division of the Cell component determines how the curtain wall is divided. There are 5 different division types:

1. Fixed number of cells

Determines the number of curtain wall cells according to a fixed number of horizontal and vertical divisions, regardless of the curtain wall length and height in the model.

2. Fixed cell size

Fixes the cell size regardless of the curtain wall length and height in the model.

Since the curtain wall objects length and height may not fit with the cell size, there are some parameters that help to adjust the position of the grid division along a curtain wall object in the model:

  • Row fit: option to specify whether the last cell of each row of the curtain wall object extends to fit with the end or is trimmed.
  • Column fit: option to specify whether the last cell of each column of the curtain wall object extends to fit with the end or is trimmed.
  • Alignment: option to specify the position from which the grid division will be created from along the curtain wall object.

3. Horizontal division

By default, it generates two Cell components divided horizontally.

New Cell components can be added by doing right click on the Cell component > New > Cell. Each subdivision panel takes a "None" division type by default, but it can have any other type of division, and get subdivided again.

The Cell components that have an horizontal division can have their subdivision cells size defined as a percentage value (%) or a real size value.

Note: If a cell component is divided again as a vertical or horizontal division type, it can only have its width/hight value defined as a fixed size, in case its parent cell width/hight is set to fixed size too.

4. Vertical division

By default, it generates two Cell components divided vertically. It works in the same way as the Horizontal division type.

5. None

Generates a single panel with no subdivision.

Infill

The infills determine how each cell composing a Curtain Wall is filled. There are 3 different Infill types:

  • Single Panel: Each cell is filled with a single panel.
  • Empty: Each cell appears empty.
  • Curtain Wall: Each cell is filled with a curtain wall style.

Panel Thickness: When the infill panel thickness is 0, it is represented as a single surface.

Curtain Wall Style: this option shows the list of curtain wall styles available when the infill type is set to Curtain Wall.

Panel gap: space left among panels at their horizontal and vertical joints.

Frame

Curtain walls can have a frame. The frame component is defined by the following parameters:

Geometry
  • Depth and width: the sizes can be edited independently for the vertical and the horizontal parts of the frame.

Mullions

Curtain walls can have mullions. The mullion component represents the frames situated at cells divisions. They are defined by the following parameters:

Geometry
  • Depth and width: the sizes can be edited independently for the vertical and the horizontal mullions.

Supports

Curtain walls can have supports.

Supports are structural elements that support the curtain wall panels. Curtain walls can have more than one Support component. To add a new Support component, right click on the Curtain wall style > New > Curtain wall support.

Supports are defined by the following parameters:

Geometry
  • Support type: supports can be either vertical or horizontal. If you want to combine both, add a new Support component to the Curtain Wall style. You can also specify if supports are created at the start and at the end of the curtain wall.
  • Placement: You can specify the Longitudinal and Transversal offset of the support position from the panels.
  • Rotation: The rotation of the support profile around its own axis.
Profile
  • The support shape is defined by a profile and by its size. You can assign a custom profile to a Support component.

To add a custom profile, draw a closed and flat curve and convert it into a new profile using the vaProfileFromCurve command.
Once the custom profile has been created, it will appear at the end of the list of preset parametric profiles library.
