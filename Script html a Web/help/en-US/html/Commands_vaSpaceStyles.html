Space Styles

Keyboard:
vaSpaceStyles
Icon:
right click [left click]
Documentation toolbar:
Menu:
VisualARQ > Space > Space Styles
Shortcut key:
Select a space object > (Alt, Shift or Ctrl) + double left click

The space styles define the features of spaces/rooms. You can define the following components for each space style: Attributes, General, Name, Area and Perimeter.

New space style:

You can create new styles and duplicate existing styles from the space Style Manager dialog box.

Delete a space style

You can delete space styles from the space Style Manager dialog box. You can only delete those styles that are not being used by any existing space in the model.

Edit a space style:

Select the space style at the top left panel, and edit their different parameters from the tabs that appear in the right panel.


Space object Style Manager dialog box, General tab.

General style editing
You can use the vaSpaceStyles command or, alternatively,
  • Use the vaStyleProperties command, then select a space and press Enter, or
  • Right click on , then select a space and press Enter, or
  • Select a space and open the context menu by pressing the right mouse button for a while then select Space > Style Properties, or
  • Press Alt, Shift or Ctrl then double click on a space object.
The changes applied to a style are applied to all elements with that style.
Object-specific editing
  • Use the vaProperties command, then select a space and press Enter, or
  • Click on , then select a space and press Enter, or
  • Select a space and open the context menu by pressing the right mouse button for a while then select Space > Properties, or,
  • Double click on a space object.
The changes are applied only to the style of the selected element.

Attributes

Attributes define the features of each space style for display and print purposes. The valid attributes are:

  • Layer, Visibility, Text style, Dimension style, Color, Linetype, Line print width, Line print color, Pattern, Pattern scale, Pattern angle and Pattern color.

General

Options to define the way in which the information is displayed in the space legend:

  • Show border
  • Show name
  • Show area
  • Show perimeter

Name

Options to define the way in which the room name is displayed in the space legend:

  • Prefix: Text displayed before the name
  • Suffix: Text displayed after the name

Area

Options to define the way in which the room area is displayed in the space legend:

  • Prefix: Text displayed before the area
  • Suffix: Text displayed after the area

Perimeter

Options to define the way in which the room perimeter is displayed in the space legend:

  • Prefix: Text displayed before the perimeter
  • Suffix: Text displayed after the perimeter
