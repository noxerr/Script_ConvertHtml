Opening Elevation

Keyboard:
vaOpeningElevation
Icon:
left click
Documentation toolbar:
Menu:
VisualARQ > Opening Elevation > Opening Elevation

This command generates 2D representations from the elevations of the different model openings.

Steps:

  1. Select the doors and windows you wish to obtain an elevation from. The command automatically filters the selection, so that only the opening objects are displayed.
  2. Select the start and end point of the insertion.

The command ends when the 2D drawing of each selected opening is inserted, with their corresponding references. The same references will be displayed in the Plan view, identifying the location of each type of door and window in the model.


Selection of openings (filtering)

References of the model opening

Dimensioned opening elevations, with reference to each model opening

NOTE: In the case of openings (doors or windows) with the same features - sizes, elevation and opening side - a single drawing will be created with a single reference.

Opening elevations and their references are created by default in the Documentation layer.
To hide the references, simply turn off the layer where they have been created.
To delete the references, select the opening you wish to delete and open the VisualARQ Properties section (in the Rhino Properties Panel ). Delete the Reference text value, in the Reference section.
