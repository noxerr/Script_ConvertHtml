Welcome to VisualARQ

What is VisualARQ?

VisualARQ is a plug-in that has been designed for Rhino users working in the architecture sector (architects, interior designers, developers, etc). It helps to boost the workflow and improve the efficiency of the design process, allowing the user to carry out a greater number of tasks within a single production environment. Therefore, it reduces the loss of time and information, a common problem when using different software programs and formats throughout the project.

VisualARQ is fully integrated in the Rhino interface and it offers a powerful editor that can be easily and rapidly used to create and modify parametric architectural objects. Likewise, it offers a series of architectural tools that make the design and documentation process simpler and more dynamic.

Unlike other products in the market, VisualARQ integrates intuitively in the Rhino design workflow with the shortest learning curve.

Features

  • The plug-in is fully integrated in the Rhino interface.
  • Designed for Rhino users working in the architecture sector.
  • Boosts the workflow and improves the efficiency of the design process.
  • Allows you to carry out easily a greater number of tasks within a single production environment.
  • Offers a powerful function editor to easily and rapidly create and edit parametric architectural objects.
  • Offers a series of architectural tools.
  • Makes the project documentation and design process simpler and more dynamic.
  • It is very intuitive and easy to learn.
  • Imports and exports files to IFC to exchange Rhino models with other AEC software.
