The Slab styles define the slab features. You can define the following parameters in each slab style: Name and Attributes.

The slab styles are defined by different Slab Layers that can be added or deleted at any time.

New slab style:

You can create new styles and duplicate existing styles from the Slab Style Manager dialog box.

Delete a slab style:

You can delete slab styles from the Slab Style Manager dialog box. You can only delete those styles that are not being used by any existing slab in the model.

Edit a slab style:

Select the slab style or the slab component item at the top left panel, and edit their different parameters from the tabs that appear in the right panel.

Add a slab layer

  • Context menu: do right-click on the slab style name to open the context menu, and select New > Slab layer.
  • New icon : select the slab style and click on the New icon, from dialog toolbar. Select Slab layer.

Delete layers: use the Context menu (righ-click on the layer to delete and select Delete) or the Delete button .


Style Manager dialog box for the slab object

General style editing
You can use the vaSlabStyles command or, alternatively,
  • Use the vaStyleProperties command, then select a slab and press Enter, or
  • Right click on , then select a slab and press Enter, or
  • Select a slab and open the context menu by pressing the right mouse button for a while, then select Slab > Style Properties, or
  • Press Alt, Shift or Ctrl, then double click on a slab object.
The changes applied to a style are applied to all elements with that style.
Object-specific editing
  • Use the vaProperties command, then select a slab and press Enter, or
  • Click on , then select a slab and press Enter, or
  • Select a slab and open the context menu by pressing the right mouse button for a while, then select Slab > Properties, or,
  • Double click on a slab object.
The changes are applied only to the style of the selected element.

Attributes

Attributes define the features of each slab style and its layers for display and print purposes. The valid attributes are:

  • General: Layer, Visibility and Material.
  • Projection: Color, Linetype, Line print width and Line print color.
  • Section: Color, Linetype, Line print width, Line print color, Pattern, Pattern scale, Pattern angle, and Pattern color.

Slab Components

Layers

Slab styles can have different number of layers, but they must have at least one.

You can specify the following parameters for each layer: Name, Attributes and Geometry.

Geometry

Thickness: the thickness of the slab layer is a property of the slab style. However, it can be also edited for each slab object individually.
