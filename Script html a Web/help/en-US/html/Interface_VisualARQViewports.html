VisualARQ Viewports

VisualARQ viewports follow the same behavior and control pattern as Rhino viewports. The main difference lies in the representation of VisualARQ objects in the Top view (Plan view), called Floor 1 in VisualARQ.


Plan view (Level 02).

Rhino objects are displayed in 3-D from aboveù but VisualARQ objects are drawn in 2-D for this type of view. The 2-D drawing uses the standard display of floor elements in architectural drawings: walls show their components, doors show their swing, steps on stairs are numbered, etc.

VisualARQ Plan view is a horizontal section with a cutting plane at a default elevation of 1.40 meters of the elevation from the current floor. You can modify the height of floors and elevation of the cut plane of each floor from the Level Manager dialog box.

When working with VisualARQ templates, the title of the Plan view is displayed by default as Level 01 or Floor 1. When you create additional levels on the drawing using the Level Manager dialog box, VisualARQ allows you to filter the information displayed on the selected view, showing only the elements corresponding to the level defined as Current Level, and the projection of the visible objects underneath. The current level name is shown on the view tooltip.

You can launch the Level Manager dialog box in four different ways:

  • In VisualARQ menu, click Levels > Edit Levels...
  • Right-click on the view title and choose VisualARQ > Levels...
  • In the VisualARQ Tools toolbar, click on the Levels icon.
  • Run the vaLevels command from the command line.



To open the Levels dialog box, right-click on the view title and choose VisualARQ > Levels...

You can set any level as Current from the Level Manager dialog box.

There are two ways to change between the usual display of a Rhino Top view and the VisualARQ 2-D drawing representation.

  • In the Level Manager toolbar, click on the Cut Plane icon.
  • Run the vaSetView command from the command line. This command automatically changes the current viewport into Top and activates the Cut Plane of a selected Building and Level.

NOTE: The vaSetView > Level and vaSetView > Model options can be used in any view.

Scale Symbol

When VisualARQ is loaded and you are working with one of its templates, the viewports with parallel projection views (i.e. all default views except the Perspective view) show a scale symbol on the lower right corner.

The Scale symbol options can be edited in the VisualARQ Options from the Rhino Options dialog box.


Elevation References

The Front and Side views show the elevation references; their number, name and elevation depend on the levels created with the Level Manager. The level reference appears highlighted indicating the level where you are working from.

The Elevation references can be edited in the VisualARQ Options from the Rhino Options dialog box.
