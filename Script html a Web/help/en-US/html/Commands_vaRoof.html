VisualARQ > Roof > From Boundary

There are three types of roof you can create with VisualARQ when you run the vaRoof command.

  • Gable roof: roof with two flights.
  • Hip roof: roof with three or more flights.
  • Shed roof: one flight roof.

The command to create roofs does not use an insert box or dialog boxes. There is no wizard availabe to create styles, since there are no roof styles.

When you run the vaRoof command, follow the steps from the command line. Press the right mouse button or ENTER to accept and go to next step.

Steps:

  • Gable roof

  1. Pick a Curve that will be used as the roof boundary. This curve must be flat and closed.
  2. Select the roof type Gable from the command line.
  3. Enter the roof Slope angle.
  4. Enter the roof Thickness.
  5. Define the Axis direction of the gable roof ridge.

  • Hip roof

  1. Pick a Curve that will be used as the roof boundary. This curve must be flat and closed.
  2. Select the roof type Hip from the command line.
  3. Enter the roof Slope angle.
  4. Enter the roof Thickness.

  • Shed roof

  1. Pick a Curve that will be used as the roof boundary. This curve must be flat and closed.
  2. Select the roof type Shed from the command line.
  3. Enter the roof Slope angle.
  4. Enter the roof Thickness.
  5. Specify roof axis start and end points.

The command ends when the roof is created according to the curve that defines its boundary.

Roof Properties

Once a roof has been created, there are some values that can be edited to change the roof geometry. These options are available in the VisualARQ Properties section (in the Rhino Properties Panel ), after selecting the roof object.

Thickness:

Determines the roof thickness for all its flights.

Roof type:

Change the roof type from Hip to Gable and from Gable to Hip.

Edge cut:

There are three options available to define the roof edge cut:

  1. Flat
  2. Plumb
  3. Square
Slope:

The roof slope can be edited globally or individually for each roof flight. The roof slope is measured in angle degrees.

Gable height: (only in Gable roofs)

Set the value of the gable roof ridge position. When this value is changed, the roof slope angles may get modified.

Axis orientation: (only in Gable roofs)

Set the value in angle degrees of the gable roof ridge axis orientation in the XY plane.

Top cutting height: (only in Hip roofs)

Represents the height value in which a hip roof will be flatten.

Control Points

The roofs have as many control points as the original curve they were created from. You can select any of these points and change the roof boundary position. The roof is regenerated automatically keeping the slopes angle values and roof thickness.

The Gable roof has 3 additional control points to move the ridge position and change its axis orientation.


  1. Roof Boundary control points.
  2. Gable roof ridge position
  3. Gable roof ridge axis orientation
  4. Roof hole contour
  5. Height control arrow (only editable in front and side viewports)

Edit Roofs

To edit the roof, VisualARQ has a toolbar with some options than can be used to modify the roof slope angle and its shape.
