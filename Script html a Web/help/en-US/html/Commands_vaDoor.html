Door

Keyboard:
vaDoor
Icon:
left click [right click]
Objects Toolbar:
Menu:
VisualARQ > Door > Door

Doors can be inserted in a wall or a curtain wall object or as floating objects. In the first case both elements are anchored and the door opening is generated automatically in the wall/curtain wall where it is inserted. Doors can be detached from walls/curtain walls by moving them outside of their surface limits.

After running the vaDoor command, the door insert dialog box will appear. This dialog box shows a list of all door styles available, as well as the basic options and the preview of the door object.

The steps required to insert a door depend on the insertion options selected.

Steps:

  1. Select the Door style you wish to use and define the basic insertion options.
  2. Select the Door insertion point with a left-click on the desired point in the model. If that point is near to a wall/curtain wall path, the door will be automatically inserted in that wall/curtain wall. In that case there are two options available:
    • Offset: insert the door at a specific distance (offset), from the end nearest to the wall/curtain wall where the door is inserted. Specify the opening side by moving the cursor; the command calculates it automatically with the cursor's position. Click to insert the door.
    • Horizontal alignment: Insert the door freely in any part of the wall/curtain wall. Click to insert the door in the desired point.
  3. Specify the door orientation manually with the cursor. This step is not required when the door has detected a wall or a curtain wall.
  4. Specify the Opening side by clicking on the desired side of the wall/curtain wall or the door position. This step is not required when you insert a door with the offset option.

You can enter as many doors as you wish and you can cancel the command by pressing ENTER or ESC.

In the two insertion options, the opening corresponding to the door is automatically generated in the wall/curtain wall as soon as it detects the wall/curtain path.

Door from curve

Doors can be created from curves, which determine their shape profile.

Steps:

  1. Run the vaDoor command.
  2. Select the Door style you wish to use and define the basic insertion options.
  3. Select the FromCurves option in the command line.
  4. Select a curve (or several curves) in the model. This curve must be closed and vertical.

The door will be generated in the selected curve and will only be inserted in a wall/curtain wall if it is close enough.

Note: The curves selected to create doors don't need to be oriented parallel to the wall/curtain wall where the doors will be placed. Doors are inserted and oriented automatically according to the closest wall/curtain wall position.

Control points

Doors have five control points:

  1. Control point for the opening side
  2. Left insert
  3. Center insert
  4. Right insert
  5. Reference mark control point

To move the door (either in the wall/curtain wall or to a different wall/curtain wall), select it from any point and drag it to the desired location.

To change the opening side of the door, select the control point for the opening side (1) and drag it to the other side of the wall/curtain wall.

Insertion options and parameters

Alignment

The door can be aligned with three methods within the wall/curtain wall thickness:

  • Interior
  • Center
  • Exterior

To modify the alignment of a door, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).

Aperture percentage

The aperture percentage defines how open or closed the door leaves are displayed in the plan and perspective views. There are two Aperture percentage options:

  • Max aperture (%): This value defines the maximum aperture the door leaves can reach. The Maximum door aperture is only displayed in Top Viewport when the VisualARQ Plan view mode is enabled, and it is represented with a simple line and an arch.
  • Aperture (%): This value defines how open or closed the door leaves are displayed in perspective and plan views.

Door aperture example: Max aperture : 50%, Aperture: 25%

The aperture percentage is not a style property. The value is defined, in the insert dialog box of the Door object, for each object inserted .

The default aperture value for doors is 50% but you can use values from 0% to 100%. If you modify this value, the insert dialog box will show the last value used.

To modify a door aperture percentage, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).


Vertical alignment

Doors can be inserted in a wall/curtain with two different vertical alignments in relation with the base line of the wall/curtain wall:

  • Bottom: reference distance between the base line of the wall/curtain wall and the bottom surface of the door.
  • Top: reference distance between the base line of the wall/curtain wall and the top surface of the door.

The vertical alignment is not a style property. The value is specified for each object in the insert dialog box when each object is inserted.


To modify a door vertical alignment, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).

Elevation

The door elevation in the insert dialog is measured in relation to the baseline of the wall/curtain wall it is inserted in, regardless of its level. After the door has been inserted, its elevation value indicates the door position in the world Z coordinates, regardless of the wall/curtain wall and the level where it is inserted. The elevation is not a style property. The value is specified for each object in the insert dialog box when the object is inserted.
The default elevation value for doors is 0. If you modify this value, the insert dialog box will show the last value used.

To modify a door elevation, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).
