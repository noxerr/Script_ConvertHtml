Wall: Add Solids

Keyboard:
vaWallAddSolids
Icon:
left click
Walls toolbar:
Menu:
VisualARQ > Wall > Add Solids

Option to add one or more solid objects to a wall.

When you run the vaWallAddSolids command, follow the steps from the command line. Press the right mouse button or ENTER to accept and go to the next step.

Steps:

  1. Select the wall where you wish to add one or more solid objects.
  2. Select the solids you wish to add. These solids must be closed extrusions or polysurfaces and must intersect with one or more points on the wall where they will be added.

The volume of the solid object is added to the wall body and it becomes part of the wall. The result can be displayed in any view.

The wall features are not affected by this operation, so you can add doors and windows to the modified wall.
