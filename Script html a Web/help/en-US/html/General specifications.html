General Specifications

Software

VisualARQ requires Rhinoceros 4.0 SR7 or Rhinoceros 5 SR6 (32 or 64-bit) to run.

Versions and Updates

VisualARQ checks for updates automatically. If there is an update available, an icon appears in the Windows taskbar notification area.

An option to disable the automatic update checking can be found in the VisualARQ section inside the Rhino Options dialog.

Languages

VisualARQ's interface, installer, templates and toolbars are available in 6 languages: English, Spanish, French, Italian, German and Czech.

A dialog to choose VisualARQ's language can be found in the VisualARQ section inside the Rhino Options dialog.

Units

VisualARQ supports Metric and Imperial unit systems. When changing the drawing units, VisualARQ shows a message asking if you want to scale style and profile definitions. When importing and exporting object styles, they are scaled according to the document units.

  • VisualARQ - Meters.3dm
  • VisualARQ - Milimeters.3dm
  • VisualARQ - Feet.3dm
  • VisualARQ - Inches.3dm

Layers

VisualARQ allows you to set the default layer in creation commands. The Layers panel can be found in the VisualARQ's section inside the Rhino Options dialog.

When running the object creation commands, the current layer is switched to the target layer. When the commands end, the previously active layer becomes the current layer, unless the user changes it during the command.

Symbols

VisualARQ adds a scale icon symbol to specify the view scale in drawing units. An option to disable this scale icon can be found in the VisualARQ section inside the Rhino Options dialog.

VisualARQ adds elevation marks to the front and side views that make it easier to work with multi-level models. An option to disable these elevation marks can be found in the VisualARQ section inside the Rhino Options dialog.

Templates

When installing the software, VisualARQ templates are copied to the default Rhino template folder.

VisualARQ templates contain a predefined library of object styles, which can be modified, imported and exported to be used in other files.

VisualARQ templates have a tolerance of 1mm.

Help

VisualARQ includes some Help options, which can be found in the VisualARQ main menu.
Options include the Help file in six languages, links to Technical Support and VisualARQ website, and the About dialog with information about License, User Name, Organization and Serial Number.
