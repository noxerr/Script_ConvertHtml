Section Styles

Keyboard:
vaSectionStyles
Icon:
right click [left click]
Documentation toolbar:
Menu:
VisualARQ > Section > Section Styles
Shortcut key:
Select a section object > (Alt, Shift or Ctrl) + double left click

The section style defines the attributes of section marks (section cutting lines).

New section style:

You can create new styles and duplicate existing styles from the Section Style Manager dialog box.

Delete a section style:

You can delete section styles from the section Style Manager dialog box. You can only delete those styles that are not being used by any existing section in the model.

Edit a section style:

Select the section style at the top left panel, and edit their different parameters from the tabs that appear in the right panel.


Section styles dialog

General style editing
You can use the vaSectionStyles command or, alternatively,
  • Use the vaStyleProperties command, then select a section and press Enter, or
  • Right click on , then select a section and press Enter, or
  • Select a section and open the context menu by pressing the right mouse button for a while then select Section > Style Properties, or
  • Press Alt, Shift or Ctrl then double click on a section object.
The changes applied to a style are applied to all elements with that style.
Object-specific editing
  • Use the vaProperties command, then select a section and press Enter, or
  • Click on , then select a section and press Enter, or
  • Select a section and open the context menu by pressing the right mouse button for a while then select Section > Properties, or,
  • Double click on a section object.
The changes are applied only to the style of the selected element.

Attributes

Attributes define the features of each section style for display and print purposes. The valid attributes are:

  • Layer, Visibility, Text style, Dimension style, Color, Linetype, Line print width, Line print color, Pattern, Pattern scale, Pattern angle and Pattern color.
