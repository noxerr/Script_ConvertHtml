Roof: Add Boundary

Keyboard:
vaRoofAdd
Icon:
left click
Roofs toolbar:
Menu:
VisualARQ > Roof > Add Boundary

Option to add a boundary to a roof.

When you run the vaRoofAdd command, follow the steps from the command line. Press the right mouse button or ENTER to accept and go to next step.

Steps:

  1. Select the Roof you wish to add the boundary to.
  2. Select the Closed and planar curve you wish to add as a boundary.

The roof adds the silhouette of the new boundary to its original shape.

NOTE: Boundaries do not need to be in the roof plane, but they have to be planar and closed curves.
