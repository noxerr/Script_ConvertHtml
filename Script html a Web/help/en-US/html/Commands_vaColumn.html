Column

Keyboard:
vaColumn
Icon:
left click [right click]
Objects Toolbar:
Menu:
VisualARQ > Column > Column

Columns can be inserted inside or outside a wall.
When they are inserted in a wall, the column boundary automatically cuts out the wall line crossing it in order to define the intersection between both elements (in the Plan and Perspective views).

After running the vaColumn command, the column insert dialog box will appear. This dialog box shows a list of all column styles available, as well as the basic options and the preview of the column object.

Steps:

  1. Select the Column style you wish to use and define the basic insertion options.
  2. Select the Column insertion point with a left-click on the desired point.
  3. Specify the Column orientation manually with the cursor. Click again when the column is on the desired point.

You can enter as many columns as you wish and you can cancel the command by pressing ENTER or ESC.

Control Points

Columns have a single control point, which is always the same as its insertion point. This point corresponds to the alignment chosen for the column when it is inserted.

Insertion Options and Parameters

Alignment

The column alignment determines the position of the column insertion point according to its profile shape.

Alignment options:

  • Top Left, Top Center and Top Right
  • Middle Left, Center and Middle Right
  • Bottom Left, Bottom Center and Bottom Right

To modify the alignment of a column, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).

Insert mark

The mark displayed on the alignment point which indicates the fixed column face. The mark is only displayed when the Insert Mark option is enabled. You can select one of the following shapes:

  • Circular
  • Triangular

Columns with different types of insert marks

Insert marks are only displayed in the Plan view.
To modify the column insert mark, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).

Offset

Shows the distance (offset) from the column X and Y points to the insertion point.

To modify a column offset, edit its value in the Object Properties dialog box or in the VisualARQ Properties section (in the Rhino Properties Panel ).

Roll

This value determines the angle by which the column rotates around its axis, taking as the center of rotation its alignment position.

The column roll can only be edited in the VisualARQ Properties section (in the Rhino Properties Panel ).

Mirrored

When enabled this option changes the column orientation into its mirror image.

This option can only be edited in the VisualARQ Properties section (in the Rhino Properties Panel ).
